# Gin-Free & Tonic / 无酒精金汤力

## 故事 / Story
Gin & Tonic 的零酒精重构是现代无酒精酒吧最常见的经典改写之一。杜松子和汤力水本身就提供很高的辨识度。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精杜松子风味蒸馏液 50 ml |
| 150 ml | 汤力水 |
| 1 片 | 柠檬皮或葡萄柚皮 |
| 3 颗 | 杜松子 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
杜松子、柑橘皮和奎宁苦味清楚，结构干净，余味长。

## 情绪 / Mood
克制、社交、适合黄昏和餐前。

## 调酒师笔记 / Bartender Notes
- 汤力水甜度差异大，应选择偏干款。
- 柑橘皮比果汁更适合这杯酒。
- 不要过度搅拌。
