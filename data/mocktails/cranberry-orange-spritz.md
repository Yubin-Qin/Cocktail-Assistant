# Cranberry Orange Spritz / 蔓越莓橙香气泡

## 故事 / Story
蔓越莓与橙子是冬季饮品的经典组合。Spritz 形式使它比普通果汁潘趣更轻。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 蔓越莓汁 |
| 30 ml | 橙汁 |
|  | 0.0% 无酒精干型起泡酒 90 ml |
| 30 ml | 苏打水 |
|  | 橙皮装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
蔓越莓酸涩、橙子甜香和干型气泡，颜色节庆。

## 情绪 / Mood
冬季、聚会、举杯感。

## 调酒师笔记 / Bartender Notes
- 起泡酒必须 0.0%。
- 蔓越莓汁不要太甜。
- 无需额外糖。
