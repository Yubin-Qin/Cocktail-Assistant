# Zero Aviation / 无酒精航空

## 故事 / Story
Aviation 是 20 世纪初经典，以独特的紫罗兰香和天空色调闻名。零酒精版本需要非常克制花香。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精杜松子风味蒸馏液 45 ml |
| 20 ml | 新鲜柠檬汁 |
| 10 ml | 樱桃糖浆 |
| 5 ml | 紫罗兰糖浆 |
|  | 柠檬皮装饰 |

## 步骤 / Steps
1. 将所有液体材料加入摇酒壶。
2. 加满冰块，用力摇和 10-12 秒。
3. 双重过滤入预冷鸡尾酒杯，或过滤入装有新鲜冰块的杯中。
4. 完成装饰后立即饮用。

## 风味 / Flavor
柠檬、樱桃核果和轻微紫罗兰花香，色泽淡紫。

## 情绪 / Mood
梦幻、复古、克制。

## 调酒师笔记 / Bartender Notes
- 紫罗兰糖浆极强，只需少量。
- 樱桃糖浆选择偏干款。
- 成品不应像香水。
