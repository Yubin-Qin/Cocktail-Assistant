# Ayran / 艾兰

## 故事 / Story
Ayran 是土耳其及周边地区广泛饮用的酸奶饮，结构极简：酸奶、水和盐。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 120 g | 原味酸奶 |
| 120 ml | 冷水 |
| 1 小撮 | 盐 |
|  | 冰块适量 |

## 步骤 / Steps
1. 将所有材料充分冰镇。
2. 用搅拌机、奶泡器或密封容器快速搅打至产生细泡。
3. 倒入冰冷杯中。
4. 立即饮用。

## 风味 / Flavor
酸奶酸、轻咸、非常清爽，泡沫细腻。

## 情绪 / Mood
朴素、清凉、适合烤肉。

## 调酒师笔记 / Bartender Notes
- 充分打出泡沫。
- 不要做得太咸。
- 所有材料必须很冷。
