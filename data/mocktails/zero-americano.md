# Zero Americano / 无酒精美式开胃酒

## 故事 / Story
Americano 是 Negroni 的重要前身之一。无酒精版本只需保留苦味开胃酒、红味美思风格和苏打水即可。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精红色苦味开胃酒 45 ml |
|  | 0.0% 无酒精红味美思风格饮品 45 ml |
| 90 ml | 苏打水 |
|  | 橙片装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
苦甜草本、红果和气泡，酒体比 No-Groni 更轻。

## 情绪 / Mood
从容、餐前、适合慢喝。

## 调酒师笔记 / Bartender Notes
- 苏打水要足够冰冷。
- 苦味产品甜度高时可减少比例。
- 橙皮是配方的一部分。
