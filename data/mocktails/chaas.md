# Chaas / 印度咸味酸奶饮

## 故事 / Story
Chaas 是印度常见的稀释咸味酸奶饮，也常被称作 spiced buttermilk。它体现了无酒精饮品的咸鲜方向。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 100 g | 原味酸奶 |
| 180 ml | 冷水 |
| 1 小撮 | 黑盐 |
| 1 小撮 | 烤孜然粉 |
|  | 香菜或薄荷少许 |

## 步骤 / Steps
1. 将所有液体、果肉和冰块放入搅拌机。
2. 高速搅打至均匀顺滑。
3. 试味并调整酸甜。
4. 倒入冰冷杯中，完成装饰。

## 风味 / Flavor
酸奶清酸、咸味、孜然和草本，轻盈不甜。

## 情绪 / Mood
清凉、务实、适合重口味餐食。

## 调酒师笔记 / Bartender Notes
- 酸奶与水充分打匀。
- 盐量要克制。
- 可加姜或咖喱叶变化。
