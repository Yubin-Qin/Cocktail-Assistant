# Planter's Punchless / 无酒精种植园潘趣

## 故事 / Story
Planter’s Punchless 是对加勒比 Punch 传统的零酒精改写。它保留“酸、甜、香料、长饮”的思想，用黑糖和无酒精苦精补回深度。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 菠萝汁 |
| 45 ml | 橙汁 |
| 25 ml | 新鲜青柠汁 |
| 15 ml | 黑糖糖浆 |
|  | 0.0% 无酒精芳香苦精 2 dashes |
| 60 ml | 苏打水 |
|  | 肉豆蔻少许 |

## 步骤 / Steps
1. 将除气泡材料外的液体加入摇酒壶。
2. 加满冰块，用力摇和 8-12 秒。
3. 过滤入装有新鲜冰块的杯中。
4. 补入气泡材料，轻轻搅拌。
5. 完成装饰后立即饮用。

## 风味 / Flavor
热带水果、青柠、黑糖和香料构成比普通果汁更深的层次，尾段略带烘烤感。

## 情绪 / Mood
热带、宽松、但比甜型果汁饮更成熟。

## 调酒师笔记 / Bartender Notes
- 黑糖糖浆比白糖浆更适合热带香料。
- 肉豆蔻只需极少量。
- 无酒精苦精需确认 0.0%。
