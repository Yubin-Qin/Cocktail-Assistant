# Coffee Collins / 咖啡柯林斯

## 故事 / Story
Coffee Collins 借用经典 Collins 的酸甜气泡框架，用冷萃咖啡替代植物基酒作为苦味骨架。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 冷萃咖啡 |
| 20 ml | 新鲜柠檬汁 |
| 20 ml | 单糖浆 |
| 120 ml | 苏打水 |
|  | 柠檬皮装饰 |

## 步骤 / Steps
1. 将除气泡材料外的液体加入摇酒壶。
2. 加满冰块，用力摇和 8-12 秒。
3. 过滤入装有新鲜冰块的杯中。
4. 补入气泡材料，轻轻搅拌。
5. 完成装饰后立即饮用。

## 风味 / Flavor
咖啡、柠檬与气泡形成修长结构，比 Espresso Lemonade 更轻。

## 情绪 / Mood
都市、理性、适合白天长饮。

## 调酒师笔记 / Bartender Notes
- 冷萃浓度要足。
- 柠檬汁不能太多。
- 苏打最后补入。
