# Pink Lemonade / 粉红柠檬水

## 故事 / Story
Pink Lemonade 在北美有悠久的大众饮用传统，关于最初的粉红色来源有多种传说。现代版本通常使用莓果或石榴赋色，让它更适合酒吧式呈现。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 30 ml | 新鲜柠檬汁 |
| 15 ml | 覆盆子糖浆 |
| 10 ml | 单糖浆 |
| 90 ml | 冷水 |
| 60 ml | 苏打水 |
|  | 柠檬片和覆盆子装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
鲜明柠檬酸、莓果香和轻气泡，颜色柔和，口味比普通柠檬水多一层果香。

## 情绪 / Mood
明快、夏日、适合野餐和聚会。它是看起来轻松、喝起来也没有负担的一杯。

## 调酒师笔记 / Bartender Notes
- 莓果糖浆只需少量，重点仍然是柠檬。
- 使用苏打水可提升鸡尾酒感。
- 提前批量制作时不要提前加入气泡。
