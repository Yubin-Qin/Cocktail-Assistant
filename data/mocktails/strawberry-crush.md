# Strawberry Crush / 草莓撞击

## 故事 / Story
Strawberry Crush 是现代酒吧中常见的无酒精果香长饮结构。真正的新鲜水果和姜汁啤酒让它远离糖浆汽水感。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 4 颗 | 草莓 |
| 6 片 | 薄荷叶 |
| 25 ml | 新鲜青柠汁 |
| 15 ml | 单糖浆 |
| 120 ml | 姜汁啤酒 |

## 步骤 / Steps
1. 在杯中轻压水果或香草，只释放汁液和香气。
2. 加入其余非气泡材料并搅匀。
3. 加满冰块。
4. 补入气泡材料，轻轻上下搅拌。
5. 完成装饰。

## 风味 / Flavor
草莓鲜果、薄荷、青柠和姜辣，颜色明亮，口感活泼。

## 情绪 / Mood
夏日、社交、容易喜欢。

## 调酒师笔记 / Bartender Notes
- 草莓只轻压。
- 双重过滤可更精致。
- 姜汁啤酒不要太甜。
