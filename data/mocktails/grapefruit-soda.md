# Grapefruit Soda / 鲜葡萄柚苏打

## 故事 / Story
鲜制 Grapefruit Soda 是许多经典 Paloma 与现代无酒精长饮的基础。它比商业甜汽水更能保留葡萄柚的苦味和层次。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 新鲜葡萄柚汁 |
| 10 ml | 新鲜青柠汁 |
| 15 ml | 单糖浆 |
| 120 ml | 苏打水 |
| 1 小撮 | 盐 |
|  | 葡萄柚皮装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
葡萄柚微苦、青柠明亮、盐感轻微，整体干爽而带果皮香。

## 情绪 / Mood
清醒、成熟、非常适合餐前。

## 调酒师笔记 / Bartender Notes
- 一小撮盐能压低粗糙苦味。
- 葡萄柚品种会显著影响甜度。
- 现挤现喝最佳。
