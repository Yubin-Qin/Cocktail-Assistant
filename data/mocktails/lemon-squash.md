# Lemon Squash / 柠檬司夸什

## 故事 / Story
Squash 是英国和英联邦地区常见的果汁浓缩饮法，通常以果汁、糖和水或苏打调成。Lemon Squash 是其中最经典的形态之一，后来也进入许多酒店和咖啡厅的无酒精饮品菜单。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 30 ml | 新鲜柠檬汁 |
| 20 ml | 单糖浆 |
| 120 ml | 苏打水 |
| 1 片 | 柠檬片 |
|  | 冰块适量 |

## 步骤 / Steps
1. 高球杯中加入柠檬汁和糖浆。
2. 加入冰块，搅拌均匀。
3. 补入冰镇苏打水。
4. 用柠檬片装饰。

## 风味 / Flavor
明亮柠檬、清爽酸甜、气泡和干净收尾。它比柠檬汽水更鲜活，因为酸味来自现榨果汁。

## 情绪 / Mood
简单、清醒、利落。它像午后的一次重启，适合工作间隙、运动后或任何想要轻盈的时刻。

## 调酒师笔记 / Bartender Notes
- 糖浆和柠檬汁可先调成浓缩基底，再用苏打水拉长。
- 喜欢更英式的味道，可加少量柠檬皮糖浆。
- 一小撮盐能让柠檬香更立体。
