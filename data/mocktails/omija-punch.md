# Omija Punch / 五味子花菜

## 故事 / Story
Omija-cha 是韩国传统五味子饮，“五味”指其复杂的多重味感。加入梨片和松子后具有典型宴席感。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 12 g | 干五味子 |
| 250 ml | 冷水 |
| 20 ml | 蜂蜜 |
| 3 片 | 梨片 |
|  | 松子少许 |

## 步骤 / Steps
1. 将干燥植物材料加入冷水。
2. 冷藏浸泡数小时，直到颜色和香气充分释放。
3. 过滤后加入甜味材料并搅匀。
4. 加冰倒入杯中，完成装饰。

## 风味 / Flavor
酸、甜、微苦、微咸感和独特药果香交织，颜色红亮。

## 情绪 / Mood
精致、传统、安静。

## 调酒师笔记 / Bartender Notes
- 建议冷泡数小时。
- 热水容易带出过多苦涩。
- 蜂蜜最后调整。
