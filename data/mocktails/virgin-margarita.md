# Virgin Margarita / 无酒精玛格丽特

## 故事 / Story
这是 Margarita 的零酒精版本。保留龙舌兰风味、柑橘和橙味甜味元素后，最关键的是用盐和低温维持原版的抓力。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精龙舌兰风味蒸馏液 50 ml |
| 25 ml | 新鲜青柠汁 |
| 20 ml | 橙皮糖浆 |
|  | 盐边 |
|  | 青柠角装饰 |

## 步骤 / Steps
1. 将所有液体材料加入摇酒壶。
2. 加满冰块，用力摇和 10-12 秒。
3. 双重过滤入预冷鸡尾酒杯，或过滤入装有新鲜冰块的杯中。
4. 完成装饰后立即饮用。

## 风味 / Flavor
青柠、烤龙舌兰式草本、橙皮和盐感共同形成锐利结构。

## 情绪 / Mood
明快、外向、带派对气息。

## 调酒师笔记 / Bartender Notes
- 盐边做半圈更灵活。
- 橙皮糖浆应带微苦而非单纯甜。
- 无酒精基酒偏淡时可加一小撮盐增强存在感。
