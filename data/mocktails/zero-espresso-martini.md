# Zero Espresso Martini / 无酒精浓缩马天尼

## 故事 / Story
Espresso Martini 已成为现代经典。零酒精版本最容易保留的核心是新鲜咖啡和有力摇出的泡沫。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 30 ml | 浓缩咖啡 |
|  | 0.0% 无酒精咖啡利口酒风味饮品 30 ml |
|  | 0.0% 无酒精伏特加风味蒸馏液 30 ml |
| 5-10 ml | 单糖浆 |
| 3 颗 | 咖啡豆 |

## 步骤 / Steps
1. 将所有液体材料加入摇酒壶。
2. 加满冰块，用力摇和 10-12 秒。
3. 双重过滤入预冷鸡尾酒杯，或过滤入装有新鲜冰块的杯中。
4. 完成装饰后立即饮用。

## 风味 / Flavor
浓缩咖啡、可可、焦糖和细腻泡沫，苦甜清楚。

## 情绪 / Mood
夜晚、精神、都市。

## 调酒师笔记 / Bartender Notes
- 咖啡要新鲜。
- 咖啡风味基底通常有甜度。
- 用力摇和形成稳定泡沫。
