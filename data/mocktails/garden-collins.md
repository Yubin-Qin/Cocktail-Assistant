# Garden Collins / 花园柯林斯

## 故事 / Story
Garden Collins 代表现代无酒精调酒用蔬菜、茶和植物蒸馏液构建复杂度的方向。类似结构常见于专业零酒精菜单。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精植物蒸馏液 45 ml |
| 45 ml | 冷泡绿茶 |
| 25 ml | 新鲜柠檬汁 |
| 20 ml | 芹菜糖浆 |
| 60 ml | 姜汁汽水 |
|  | 香草束装饰 |

## 步骤 / Steps
1. 将除气泡材料外的液体加入摇酒壶。
2. 加满冰块，用力摇和 8-12 秒。
3. 过滤入装有新鲜冰块的杯中。
4. 补入气泡材料，轻轻搅拌。
5. 完成装饰后立即饮用。

## 风味 / Flavor
芹菜、绿茶、柠檬、姜和草本层次清晰，偏绿色、偏干。

## 情绪 / Mood
现代、清醒、像一座雨后花园。

## 调酒师笔记 / Bartender Notes
- 芹菜糖浆要有鲜味但不能像菜汁。
- 绿茶提供单宁。
- 姜汁汽水最后加入。
