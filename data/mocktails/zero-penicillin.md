# Zero Penicillin / 无酒精盘尼西林

## 故事 / Story
Penicillin 是 21 世纪最重要的现代经典之一。零酒精版本常用烟熏茶替代泥煤威士忌的烟感，用姜和蜂蜜保留核心个性。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精威士忌风味蒸馏液 50 ml |
| 25 ml | 新鲜柠檬汁 |
| 20 ml | 蜂蜜姜糖浆 |
| 10 ml | 烟熏茶浓缩液 |
|  | 姜片装饰 |

## 步骤 / Steps
1. 将除浮顶材料外的其余材料加入摇酒壶。
2. 加冰摇和 10-12 秒。
3. 过滤入杯中。
4. 将最后的风味材料缓慢浮在表面。
5. 完成装饰。

## 风味 / Flavor
蜂蜜、姜、柠檬和烟熏茶形成温暖、辛辣、带烟感的层次。

## 情绪 / Mood
治愈、夜晚、带壁炉感。

## 调酒师笔记 / Bartender Notes
- 烟熏茶要浓但不能涩。
- 蜂蜜姜糖浆应有明显鲜姜感。
- 烟熏液最后少量浮顶。
