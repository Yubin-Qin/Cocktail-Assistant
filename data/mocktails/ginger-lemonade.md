# Ginger Lemonade / 姜汁柠檬水

## 故事 / Story
姜与柑橘是跨地区常见的无酒精组合。Ginger Lemonade 既可追溯到家常柠檬水传统，也符合现代酒吧用辛辣感替代酒精刺激的思路。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 30 ml | 新鲜柠檬汁 |
| 20 ml | 姜糖浆 |
| 90 ml | 冷水 |
| 60 ml | 苏打水 |
|  | 姜片和柠檬片装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
柠檬酸与鲜姜辛辣互相推进，甜味把两者连接起来，尾段干净而醒神。

## 情绪 / Mood
活泼、提神、适合炎热天气或餐前。

## 调酒师笔记 / Bartender Notes
- 姜糖浆最好保留鲜姜辛辣而不是只剩糖味。
- 苏打水比例可按想要的长度调整。
- 少量盐能让姜味更圆。
