# Mango Lassi / 芒果拉西

## 故事 / Story
Lassi 是南亚重要酸奶饮品。芒果版本在全球最为知名，既是传统饮品也是现代无酒精菜单常客。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 120 g | 熟芒果肉 |
| 120 g | 原味酸奶 |
| 60 ml | 冷水或牛奶 |
| 15 ml | 蜂蜜 |
| 1 小撮 | 豆蔻粉 |

## 步骤 / Steps
1. 将所有液体、果肉和冰块放入搅拌机。
2. 高速搅打至均匀顺滑。
3. 试味并调整酸甜。
4. 倒入冰冷杯中，完成装饰。

## 风味 / Flavor
芒果浓甜、酸奶微酸、豆蔻温暖，质地厚实。

## 情绪 / Mood
舒适、柔软、适合辛辣餐食。

## 调酒师笔记 / Bartender Notes
- 使用成熟芒果。
- 酸奶越稠成品越像奶昔。
- 豆蔻只需一点。
