# Orange Blossom Cooler / 橙花清凉饮

## 故事 / Story
橙花水长期用于地中海、中东和北非甜点与饮品。现代 Cooler 把这种古老香气放入柑橘高球结构中。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 90 ml | 橙汁 |
| 15 ml | 新鲜柠檬汁 |
| 2-3 滴 | 橙花水 |
| 15 ml | 蜂蜜糖浆 |
| 90 ml | 苏打水 |
|  | 橙片装饰 |

## 步骤 / Steps
1. 将除气泡材料外的液体加入摇酒壶。
2. 加满冰块，用力摇和 8-12 秒。
3. 过滤入装有新鲜冰块的杯中。
4. 补入气泡材料，轻轻搅拌。
5. 完成装饰后立即饮用。

## 风味 / Flavor
橙子甜香、柠檬酸和极轻的白花香叠加，气泡让蜂蜜不显黏重。

## 情绪 / Mood
柔和、明亮、带一点地中海和中东气息。

## 调酒师笔记 / Bartender Notes
- 橙花水非常强，只需几滴。
- 蜂蜜糖浆应先与酸汁混匀。
- 过度使用花香会像香水。
