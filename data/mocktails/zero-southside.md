# Zero Southside / 无酒精南区

## 故事 / Story
Southside 是经典琴酒、青柠、糖和薄荷组合。零酒精版本仍然依靠植物基底与薄荷建立清晰识别。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精杜松子风味蒸馏液 50 ml |
| 25 ml | 新鲜青柠汁 |
| 20 ml | 单糖浆 |
| 8 片 | 薄荷叶 |
|  | 薄荷叶装饰 |

## 步骤 / Steps
1. 在摇酒壶中轻压水果、香草或香料。
2. 加入其余液体材料。
3. 加冰用力摇和 10-12 秒。
4. 双重过滤入预冷杯或装有新鲜冰块的杯中。
5. 完成装饰。

## 风味 / Flavor
薄荷清凉、青柠明亮、杜松草本干净，整体比 Mojito 更紧凑。

## 情绪 / Mood
利落、都市、适合晚餐前。

## 调酒师笔记 / Bartender Notes
- 薄荷只轻压。
- 双重过滤避免碎叶。
- 酒杯要充分预冷。
