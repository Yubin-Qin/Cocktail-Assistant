# Rose Sharbat / 玫瑰夏尔巴特

## 故事 / Story
Sharbat 是波斯及更广泛伊斯兰世界的重要饮品传统，可由水果、花、香草或香料糖浆加水制成。玫瑰版本是最具代表性的形态之一。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 20 ml | 玫瑰糖浆 |
| 15 ml | 新鲜柠檬汁 |
| 120 ml | 冷水 |
| 60 ml | 苏打水 |
| 1-2 滴 | 玫瑰水 |
|  | 可食用玫瑰花瓣装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
玫瑰花香、柠檬酸和轻气泡，甜美但可通过酸度保持轻盈。

## 情绪 / Mood
柔和、节庆、带中东花园感。

## 调酒师笔记 / Bartender Notes
- 玫瑰水极强。
- 柠檬汁用于防止香水感。
- 糖浆甜度高时减少用量。
