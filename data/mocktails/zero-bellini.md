# Zero Bellini / 无酒精贝里尼

## 故事 / Story
Bellini 源自威尼斯 Harry’s Bar，经典组合是白桃与起泡酒。无酒精版本结构几乎不变，只需换成 0.0% 起泡酒。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 白桃泥 |
|  | 0.0% 无酒精干型起泡酒 120 ml |
| 5 ml | 新鲜柠檬汁 |
|  | 白桃片装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
白桃柔甜、细腻果肉和干型气泡相互平衡。

## 情绪 / Mood
柔和、浪漫、适合早午餐。

## 调酒师笔记 / Bartender Notes
- 桃泥要细腻无大块纤维。
- 起泡酒偏甜时加入少量柠檬汁。
- 轻轻混合避免泡沫溢出。
