# Cola Nostra / 香料可乐酸饮

## 故事 / Story
Cola 与柑橘、苦味和茶的组合是现代无酒精调酒常见思路。名称借用意式语感，但配方核心是让熟悉的可乐变得更复杂。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 120 ml | 可乐 |
| 20 ml | 新鲜柠檬汁 |
| 10 ml | 黑糖糖浆 |
| 30 ml | 冷泡红茶 |
|  | 0.0% 无酒精芳香苦精 2 dashes |
|  | 柠檬皮装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
可乐香料、茶单宁、柠檬酸和轻微苦味叠加，像把普通可乐拉进酒吧语境。

## 情绪 / Mood
夜晚、复古、带一点神秘感。

## 调酒师笔记 / Bartender Notes
- 红茶不要泡得过涩。
- 可乐已经很甜，黑糖糖浆只作质地补充。
- 所有材料都应充分冰镇。
