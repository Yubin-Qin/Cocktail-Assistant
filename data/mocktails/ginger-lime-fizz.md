# Ginger Lime Fizz / 姜青柠菲士

## 故事 / Story
Fizz 是历史悠久的鸡尾酒类型，通常由酸、甜、气泡和烈酒构成。Ginger Lime Fizz 的无酒精版本用姜的辛辣替代酒精刺激，保留了 Fizz 最重要的轻盈泡沫和明快节奏。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 25 ml | 新鲜青柠汁 |
| 20 ml | 姜糖浆 |
| 120 ml | 苏打水 |
| 15 ml | 可选：蛋白或 aquafaba |
|  | 青柠皮或姜片装饰 |

## 步骤 / Steps
1. 若使用蛋白或 aquafaba，将青柠汁、姜糖浆和蛋白干摇 10 秒。
2. 加冰摇和 8 秒。
3. 过滤入高球杯。
4. 补入苏打水，让泡沫升起。
5. 用青柠皮或姜片装饰。

## 风味 / Flavor
姜辣、青柠酸、气泡、轻甜和明亮尾韵。口感有推进力，不会疲软。

## 情绪 / Mood
醒神、活泼、带一点火花。适合餐前、加班后或需要提振精神但不想摄入酒精的时候。

## 调酒师笔记 / Bartender Notes
- 姜糖浆可用鲜姜汁和糖浆混合，辛辣感会更鲜明。
- 有泡沫版本更像经典 Fizz；无泡沫版本更像清爽长饮。
- 青柠酸度要足，才能承托姜的热辣。
