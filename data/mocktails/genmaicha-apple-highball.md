# Genmaicha Apple Highball / 玄米茶苹果高球

## 故事 / Story
玄米茶的烘烤香与苹果天然兼容。现代茶高球通过气泡把这类组合变得更轻。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 90 ml | 浓冷泡玄米茶 |
| 45 ml | 浑浊苹果汁 |
| 10 ml | 新鲜柠檬汁 |
| 90 ml | 苏打水 |
|  | 苹果片装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
炒米、绿茶、苹果和轻酸结合，口感既熟悉又有层次。

## 情绪 / Mood
秋日、安静、适合餐食。

## 调酒师笔记 / Bartender Notes
- 苹果汁不要太甜。
- 玄米茶冷泡能减少苦涩。
- 柠檬只用于提亮。
