# Qamar al-Din / 杏干饮

## 故事 / Story
Qamar al-Din 是中东著名杏饮，常用杏皮或杏干泡软后制成，在斋月期间尤其常见。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 80 g | 杏干 |
| 250 ml | 热水 |
| 2 滴 | 橙花水 |
| 10 ml | 蜂蜜 |
|  | 冰块适量 |

## 步骤 / Steps
1. 按配方提前浸泡需要软化的谷物或果干。
2. 将浸泡物与其余材料放入搅拌机。
3. 搅打至顺滑，并按需要过滤。
4. 充分冰镇后倒入杯中，完成装饰。

## 风味 / Flavor
杏干浓郁果香、轻酸、黏稠质地和微弱花香。

## 情绪 / Mood
柔和、传统、带节庆记忆。

## 调酒师笔记 / Bartender Notes
- 杏干先浸泡数小时。
- 搅打后可过滤或保留果肉。
- 橙花水只需极少量。
