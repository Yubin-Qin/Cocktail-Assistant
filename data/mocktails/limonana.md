# Limonana / 中东薄荷柠檬冰饮

## 故事 / Story
Limonana 是中东地区广受欢迎的薄荷柠檬饮，名称由 lemon 与 nana（薄荷）组合而来。它常以搅拌机打成冰凉碎冰质地。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 35 ml | 新鲜柠檬汁 |
| 25 ml | 单糖浆 |
| 15 片 | 薄荷叶 |
| 120 ml | 冷水 |
| 1 杯 | 碎冰 |

## 步骤 / Steps
1. 将所有液体、果肉和冰块放入搅拌机。
2. 高速搅打至均匀顺滑。
3. 试味并调整酸甜。
4. 倒入冰冷杯中，完成装饰。

## 风味 / Flavor
柠檬酸、薄荷清凉和碎冰形成极强降温感。

## 情绪 / Mood
阳光、海边、清醒。

## 调酒师笔记 / Bartender Notes
- 薄荷要新鲜。
- 不要搅打过久以免叶片发苦。
- 糖量按柠檬酸度调整。
