# Shrub Soda / 果醋灌木苏打

## 故事 / Story
Shrub 是以水果、糖和醋制成的历史型饮料浓缩液。现代无酒精酒吧常用它提供类似葡萄酒的酸度和身体。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 30 ml | 水果 shrub 果醋糖浆 |
| 180 ml | 苏打水 |
| 10 ml | 新鲜柑橘汁 |
|  | 对应水果装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
果香、醋酸、甜味与气泡，酸度复杂而持久。

## 情绪 / Mood
成熟、清醒、适合餐前。

## 调酒师笔记 / Bartender Notes
- shrub 酸度差异很大。
- 先少加再调整。
- 可使用莓果、苹果、菠萝等不同版本。
