# Bora Bora Brew / 波拉波拉

## 故事 / Story
Bora Bora 名称常被用于热带无酒精饮品，灵感来自南太平洋度假想象。它属于 Tiki 美学在无酒精领域的直接延伸。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 菠萝汁 |
| 30 ml | 百香果汁或果泥 |
| 20 ml | 新鲜青柠汁 |
| 10 ml | 石榴糖浆 |
| 60 ml | 姜汁汽水 |
|  | 菠萝叶装饰 |

## 步骤 / Steps
1. 将除气泡材料外的液体加入摇酒壶。
2. 加满冰块，用力摇和 8-12 秒。
3. 过滤入装有新鲜冰块的杯中。
4. 补入气泡材料，轻轻搅拌。
5. 完成装饰后立即饮用。

## 风味 / Flavor
菠萝和百香果强烈热带香，青柠拉紧甜度，石榴与姜带来颜色和辛香收尾。

## 情绪 / Mood
度假、热烈、适合派对。

## 调酒师笔记 / Bartender Notes
- 百香果酸度高，先试味再决定青柠量。
- 石榴糖浆不要超过 10 ml。
- 碎冰会更有热带风格。
