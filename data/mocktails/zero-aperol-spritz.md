# Zero Aperol Spritz / 无酒精阿佩罗气泡

## 故事 / Story
Spritz 是意大利开胃酒文化的重要形式。无酒精橙味开胃酒的发展，让经典 3-2-1 式结构也能完全零酒精实现。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精橙味开胃酒 60 ml |
|  | 0.0% 无酒精干型起泡酒 90 ml |
| 30 ml | 苏打水 |
|  | 橙片装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
橙皮、轻苦、草本、气泡和微甜构成典型开胃酒风格。

## 情绪 / Mood
黄昏、社交、轻松。

## 调酒师笔记 / Bartender Notes
- 所有材料必须冰镇。
- 选择偏苦而不是糖浆感重的开胃酒。
- 轻搅一次即可。
