# Cucumber Lime Agua Fresca / 黄瓜青柠鲜果水

## 故事 / Story
Agua fresca 是墨西哥和拉丁美洲常见的水果水。黄瓜青柠版本尤其适合热天，也与现代 spa-style 饮品风格相连。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 120 g | 黄瓜 |
| 25 ml | 新鲜青柠汁 |
| 20 ml | 单糖浆 |
| 180 ml | 冷水 |
| 1 小撮 | 盐 |

## 步骤 / Steps
1. 将所有液体、果肉和冰块放入搅拌机。
2. 高速搅打至均匀顺滑。
3. 试味并调整酸甜。
4. 倒入冰冷杯中，完成装饰。

## 风味 / Flavor
黄瓜清香、青柠酸、轻甜和微盐，口感透明。

## 情绪 / Mood
清晨、夏日、极度清爽。

## 调酒师笔记 / Bartender Notes
- 黄瓜可留部分皮增加颜色。
- 过滤与否取决于想要的质地。
- 盐只需一点。
