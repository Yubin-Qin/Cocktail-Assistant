# Zero Hugo Spritz / 无酒精雨果气泡

## 故事 / Story
Hugo Spritz 兴起于意大利北部，后来成为欧洲夏季热门 Spritz。零酒精版本天然容易成立，因为核心识别来自接骨木花、薄荷和气泡。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 20 ml | 接骨木花糖浆 |
|  | 0.0% 无酒精干型起泡酒 100 ml |
| 40 ml | 苏打水 |
| 6 片 | 薄荷叶 |
| 2 片 | 青柠轮 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
接骨木花、薄荷、青柠和细气泡组成清新花草风格。

## 情绪 / Mood
夏日、花园、轻盈。

## 调酒师笔记 / Bartender Notes
- 接骨木花糖浆偏甜，用量要克制。
- 薄荷只拍香。
- 青柠片不要久泡以免发苦。
