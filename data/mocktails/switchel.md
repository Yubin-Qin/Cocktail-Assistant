# Switchel / 斯威彻姜醋饮

## 故事 / Story
Switchel 是北美历史悠久的醋饮，常与农场劳动和夏季解渴联系在一起。它是现代 shrub 风潮的重要历史亲戚。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 20 ml | 苹果醋 |
| 20 ml | 姜糖浆 |
| 15 ml | 枫糖浆 |
| 150 ml | 冷水 |
| 60 ml | 苏打水 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
苹果醋酸、姜辣和枫糖深甜，气泡使其轻快。

## 情绪 / Mood
乡村、劳动后、醒神。

## 调酒师笔记 / Bartender Notes
- 醋要有果香。
- 枫糖只做圆润。
- 可提前混合基底冷藏。
