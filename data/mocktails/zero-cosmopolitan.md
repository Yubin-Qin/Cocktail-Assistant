# Zero Cosmopolitan / 无酒精大都会

## 故事 / Story
Cosmopolitan 的经典辨识来自柑橘伏特加、蔓越莓、青柠和橙味元素。零酒精版本保留这四层关系。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精柑橘蒸馏液 40 ml |
| 45 ml | 蔓越莓汁 |
| 20 ml | 新鲜青柠汁 |
| 15 ml | 橙皮糖浆 |
|  | 橙皮装饰 |

## 步骤 / Steps
1. 将所有液体材料加入摇酒壶。
2. 加满冰块，用力摇和 10-12 秒。
3. 双重过滤入预冷鸡尾酒杯，或过滤入装有新鲜冰块的杯中。
4. 完成装饰后立即饮用。

## 风味 / Flavor
蔓越莓、青柠和橙皮清亮，颜色漂亮，甜酸紧致。

## 情绪 / Mood
都市、明快、适合夜晚。

## 调酒师笔记 / Bartender Notes
- 蔓越莓汁不要太甜。
- 杯子必须预冷。
- 橙皮精油能提升成熟感。
