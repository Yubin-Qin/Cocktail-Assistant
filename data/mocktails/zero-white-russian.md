# Zero White Russian / 无酒精白俄罗斯

## 故事 / Story
White Russian 因咖啡利口酒和奶油的组合而极具辨识度。零酒精版本几乎不依赖酒精本身，因此很容易重构。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精咖啡利口酒风味饮品 45 ml |
|  | 0.0% 无酒精伏特加风味蒸馏液 30 ml |
| 45 ml | 淡奶油或燕麦奶 |
|  | 大冰块适量 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
咖啡、奶油、焦糖和柔和甜味，质地厚实。

## 情绪 / Mood
慵懒、夜晚、像液体甜点。

## 调酒师笔记 / Bartender Notes
- 燕麦奶版本更轻。
- 咖啡基底甜时不必加糖。
- 可轻搅保持大理石纹。
