# Honey Limeade / 蜂蜜青柠水

## 故事 / Story
Honey Limeade 是 Limeade 的常见变体。蜂蜜不只是提供甜度，也为零酒精饮品增加了通常由酒精带来的口腔重量。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 30 ml | 新鲜青柠汁 |
| 25 ml | 蜂蜜糖浆 |
| 90 ml | 冷水 |
| 60 ml | 苏打水 |
|  | 青柠轮装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
青柠的尖酸被蜂蜜的花香和黏度柔化，气泡让口感重新变轻。

## 情绪 / Mood
温柔、自然、适合午后。

## 调酒师笔记 / Bartender Notes
- 蜂蜜应先按 1:1 与温水调成糖浆。
- 不同蜂蜜风味差异明显。
- 不要用过热的水以免香气变钝。
