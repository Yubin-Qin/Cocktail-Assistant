# Zero Whiskey Sour / 无酒精威士忌酸

## 故事 / Story
Whiskey Sour 的零酒精版本是现代酒吧最常见的经典挑战之一。泡沫和较深的糖浆能补充无酒精基酒常缺少的质地。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精威士忌风味蒸馏液 60 ml |
| 25 ml | 新鲜柠檬汁 |
| 20 ml | 黑糖糖浆 |
| 20 ml | aquafaba |
|  | 0.0% 无酒精芳香苦精 2 dashes |

## 步骤 / Steps
1. 将除装饰外的材料加入摇酒壶，先不加冰干摇 10 秒。
2. 加入冰块，再用力摇和 10-12 秒。
3. 双重过滤入预冷杯或装有新鲜冰块的杯中。
4. 静置数秒让泡沫稳定，完成装饰。

## 风味 / Flavor
柠檬酸、谷物木香、黑糖和柔滑泡沫结合，尾韵微苦。

## 情绪 / Mood
舒展、复古、亲切。

## 调酒师笔记 / Bartender Notes
- 使用 aquafaba 时先干摇。
- 黑糖糖浆不要过甜。
- 苦精要确认 0.0%。
