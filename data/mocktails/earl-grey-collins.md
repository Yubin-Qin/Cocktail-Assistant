# Earl Grey Collins / 伯爵茶柯林斯

## 故事 / Story
Earl Grey 的佛手柑香气与经典 Collins 中的柠檬非常契合，因此成为现代无酒精菜单常用茶基底。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 浓冷泡伯爵茶 |
| 25 ml | 新鲜柠檬汁 |
| 20 ml | 蜂蜜糖浆 |
| 120 ml | 苏打水 |
|  | 柠檬皮装饰 |

## 步骤 / Steps
1. 将除气泡材料外的液体加入摇酒壶。
2. 加满冰块，用力摇和 8-12 秒。
3. 过滤入装有新鲜冰块的杯中。
4. 补入气泡材料，轻轻搅拌。
5. 完成装饰后立即饮用。

## 风味 / Flavor
佛手柑、茶单宁、柠檬酸和蜂蜜形成英式花柑橘风格。

## 情绪 / Mood
优雅、书房、适合下午。

## 调酒师笔记 / Bartender Notes
- 茶要浓但不涩。
- 蜂蜜与佛手柑很搭。
- 苏打最后加入。
