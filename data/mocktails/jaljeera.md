# Jaljeera / 印度香料水

## 故事 / Story
Jaljeera 是印度传统香料饮，名称中的 jeera 指孜然。它常用于炎热天气和餐前，提供酸、盐、草本和香料感。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 30 ml | 罗望子水 |
| 20 ml | 新鲜青柠汁 |
| 8 片 | 薄荷叶 |
| 1/2 茶匙 | 烤孜然粉 |
| 1 小撮 | 黑盐 |
| 150 ml | 冷水 |

## 步骤 / Steps
1. 在杯中轻压水果或香草，只释放汁液和香气。
2. 加入其余非气泡材料并搅匀。
3. 加满冰块。
4. 补入气泡材料，轻轻上下搅拌。
5. 完成装饰。

## 风味 / Flavor
酸、咸、孜然、薄荷和罗望子层次鲜明，完全不走甜果汁路线。

## 情绪 / Mood
醒胃、复杂、适合餐前。

## 调酒师笔记 / Bartender Notes
- 孜然不宜过量。
- 黑盐带硫磺气息，少量即可。
- 可加苏打水变成 fizz。
