# Sujeonggwa Sparkler / 水正果气泡饮

## 故事 / Story
Sujeonggwa 是韩国传统肉桂姜饮，通常冷饮并搭配柿饼和松子。气泡版本是现代化改写。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 90 ml | 肉桂水 |
| 60 ml | 姜水 |
| 20 ml | 红糖糖浆 |
| 90 ml | 苏打水 |
| 1 小块 | 柿饼 |
|  | 松子少许 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
肉桂、姜、红糖和气泡温暖又清爽。

## 情绪 / Mood
冬季、传统、带韩国节庆感。

## 调酒师笔记 / Bartender Notes
- 肉桂与姜可分别煮后混合。
- 气泡最后加入。
- 甜度保持克制。
