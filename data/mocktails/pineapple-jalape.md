# Pineapple Jalape / ño Cooler 菠萝墨西哥辣椒清凉饮

## 故事 / Story
菠萝与辣椒是现代拉丁风格无酒精菜单中的经典搭配。辣味提供酒精缺失后的刺激。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 90 ml | 菠萝汁 |
| 25 ml | 新鲜青柠汁 |
| 2 片 | 墨西哥辣椒 |
| 10 ml | 龙舌兰糖浆 |
| 90 ml | 苏打水 |
| 1 小撮 | 盐 |

## 步骤 / Steps
1. 在摇酒壶中轻压水果或香草。
2. 加入除气泡材料外的其余材料和冰块。
3. 摇和 8-10 秒并双重过滤入装有新鲜冰块的杯中。
4. 补入气泡材料并轻搅。
5. 完成装饰。

## 风味 / Flavor
菠萝甜、青柠酸、辣椒热感和微盐，热带却有锋利收尾。

## 情绪 / Mood
热烈、派对、醒神。

## 调酒师笔记 / Bartender Notes
- 辣椒先少量。
- 菠萝汁甜时省略糖浆。
- 盐能让辣味更圆。
