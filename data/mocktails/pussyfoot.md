# Pussyfoot / 小猫脚

## 故事 / Story
Pussyfoot 的名字通常被认为与美国禁酒运动人士 William E. “Pussyfoot” Johnson 的绰号有关。它是早期 temperance drink 传统中最具辨识度的名字之一。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 橙汁 |
| 30 ml | 菠萝汁 |
| 15 ml | 新鲜青柠汁 |
| 10 ml | 石榴糖浆 |
| 1 个 | 蛋黄 |
|  | 橙皮装饰 |

## 步骤 / Steps
1. 将所有液体材料加入摇酒壶。
2. 加满冰块，用力摇和 10-12 秒。
3. 双重过滤入预冷鸡尾酒杯，或过滤入装有新鲜冰块的杯中。
4. 完成装饰后立即饮用。

## 风味 / Flavor
柑橘与菠萝明亮，石榴提供果甜；加入蛋黄时会变得浓郁、柔滑并带轻微甜点感。

## 情绪 / Mood
古典、柔软、略带戏剧性。像老酒店里一杯被认真对待的禁酒时代饮品。

## 调酒师笔记 / Bartender Notes
- 使用蛋黄时必须充分低温保存并注意食品安全。
- 不加蛋黄会更清爽，也更适合家庭制作。
- 双重过滤可以让口感更细腻。
