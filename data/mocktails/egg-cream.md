# Egg Cream / 纽约蛋奶汽水

## 故事 / Story
Egg Cream 是纽约经典汽水柜台饮品，名字虽然含 egg 和 cream，传统配方却通常不含鸡蛋和奶油。它依靠牛奶、巧克力糖浆和苏打水形成独特泡沫。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 全脂牛奶 |
| 25 ml | 巧克力糖浆 |
| 120 ml | 苏打水 |
|  | 冰块可选 |

## 步骤 / Steps
1. 将主要基底材料加入冰冷杯中。
2. 缓慢加入气泡或第二层材料，尽量保留泡沫和层次。
3. 视配方需要轻搅，不要完全破坏分层。
4. 完成装饰后立即饮用。

## 风味 / Flavor
巧克力、奶香和强气泡形成轻盈泡沫，口感既像甜点又比奶昔更清爽。

## 情绪 / Mood
怀旧、城市、带有老纽约熟食店气息。它适合想喝甜但不想太厚重的时候。

## 调酒师笔记 / Bartender Notes
- 苏打水要从高处缓慢冲入牛奶形成泡沫。
- 巧克力糖浆最后轻搅，保留层次。
- 杯子和牛奶都要冰冷。
