# Lychee Rose Collins / 荔枝玫瑰柯林斯

## 故事 / Story
荔枝与玫瑰在亚洲甜品和现代酒吧中都常被组合。Collins 结构用酸和气泡限制花果甜味。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 荔枝汁 |
| 25 ml | 新鲜柠檬汁 |
| 10 ml | 玫瑰糖浆 |
| 120 ml | 苏打水 |
|  | 荔枝和柠檬皮装饰 |

## 步骤 / Steps
1. 将除气泡材料外的液体加入摇酒壶。
2. 加满冰块，用力摇和 8-12 秒。
3. 过滤入装有新鲜冰块的杯中。
4. 补入气泡材料，轻轻搅拌。
5. 完成装饰后立即饮用。

## 风味 / Flavor
荔枝白花果香、玫瑰和柠檬结合，气泡保持轻盈。

## 情绪 / Mood
浪漫、精致、适合庆祝。

## 调酒师笔记 / Bartender Notes
- 玫瑰糖浆只能少量。
- 荔枝汁本身很甜。
- 柠檬汁要足。
