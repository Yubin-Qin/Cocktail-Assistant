# Zero Cuba Libre / 无酒精自由古巴

## 故事 / Story
Cuba Libre 的识别点是朗姆、可乐和青柠。零酒精版本虽然简单，但新鲜青柠是把它与普通可乐区分开的关键。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精朗姆风味蒸馏液 50 ml |
| 120 ml | 可乐 |
| 20 ml | 新鲜青柠汁 |
|  | 青柠角装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
可乐香料、青柠酸和甘蔗香结合，比普通可乐更有层次。

## 情绪 / Mood
轻松、熟悉、适合聚会。

## 调酒师笔记 / Bartender Notes
- 青柠汁不要省略。
- 可乐和杯子都要冰冷。
- 无酒精基酒偏甜时可减少可乐。
