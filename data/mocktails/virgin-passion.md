# Virgin Passion / 百香果无酒精特调

## 故事 / Story
百香果是现代 mocktail 中最常见的核心水果之一。Virgin Passion 并无唯一固定配方，但通常以百香果、热带果汁和气泡构成高接受度的长饮。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 35 ml | 百香果果泥 |
| 60 ml | 菠萝汁 |
| 20 ml | 新鲜青柠汁 |
| 15 ml | 香草糖浆 |
| 60 ml | 苏打水 |
|  | 半颗百香果装饰 |

## 步骤 / Steps
1. 将除气泡材料外的液体加入摇酒壶。
2. 加满冰块，用力摇和 8-12 秒。
3. 过滤入装有新鲜冰块的杯中。
4. 补入气泡材料，轻轻搅拌。
5. 完成装饰后立即饮用。

## 风味 / Flavor
百香果尖锐热带酸香、菠萝厚度和香草圆润度结合，最后由气泡减轻甜感。

## 情绪 / Mood
明亮、热情、派对感强。

## 调酒师笔记 / Bartender Notes
- 百香果果泥本身可能含糖，要相应减少糖浆。
- 香草只需做底色。
- 使用大量冰能防止口感黏重。
