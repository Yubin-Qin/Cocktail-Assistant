# Zero Mimosa / 无酒精含羞草

## 故事 / Story
Mimosa 是世界上最知名的起泡酒长饮之一。零酒精版本同样保持橙汁与起泡酒的极简结构。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 75 ml | 鲜榨橙汁 |
|  | 0.0% 无酒精干型起泡酒 75 ml |
|  | 橙皮装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
橙汁的甜酸与干型气泡结合，简单、明亮。

## 情绪 / Mood
早午餐、庆祝、毫无压力。

## 调酒师笔记 / Bartender Notes
- 橙汁现榨更好。
- 比例可按甜度在 1:1 到 1:2 调整。
- 不要加冰，以免稀释。
