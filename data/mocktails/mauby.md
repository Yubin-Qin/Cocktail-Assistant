# Mauby / 莫比树皮饮

## 故事 / Story
Mauby 是加勒比地区传统树皮饮，以特有苦味和香料著称。它是无酒精饮品中少见的真正苦型经典。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 20 ml | mauby 树皮或商业无酒精 mauby 浓缩液 |
| 180 ml | 水 |
| 15 ml | 黑糖糖浆 |
| 1 小段 | 肉桂 |
| 1 片 | 橙皮 |

## 步骤 / Steps
1. 按配方用热水或冷泡方式萃取花草、茶或香料。
2. 过滤并完全冷却。
3. 加入甜味、酸味和冰块，搅拌均匀。
4. 按需要补入水或气泡材料，完成装饰。

## 风味 / Flavor
木质苦味、黑糖、肉桂和柑橘皮，风格独特且成人。

## 情绪 / Mood
加勒比、传统、适合喜欢苦味的人。

## 调酒师笔记 / Bartender Notes
- 商业浓缩液强度差异大。
- 苦味应明显但可饮。
- 黑糖用于圆润而非掩盖。
