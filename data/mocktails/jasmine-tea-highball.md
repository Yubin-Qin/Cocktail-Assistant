# Jasmine Tea Highball / 茉莉茶高球

## 故事 / Story
茶高球是现代无酒精酒吧的重要方向。茉莉茶具有足够香气，白葡萄汁则提供少量身体和果糖。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 90 ml | 浓冷泡茉莉绿茶 |
| 30 ml | 白葡萄汁 |
| 10 ml | 新鲜柠檬汁 |
| 90 ml | 苏打水 |
|  | 柠檬皮装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
茉莉花香、白葡萄、茶涩和清脆气泡，风格偏干。

## 情绪 / Mood
安静、透明、适合餐食搭配。

## 调酒师笔记 / Bartender Notes
- 茉莉茶过浓会苦。
- 白葡萄汁只做底味。
- 使用长条大冰。
