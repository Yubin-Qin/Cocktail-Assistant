# Zero Manhattan / 无酒精曼哈顿

## 故事 / Story
Manhattan 是威士忌与甜味美思结构的典范。零酒精版本依靠无酒精威士忌和味美思风格饮品重建层次。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精威士忌风味蒸馏液 60 ml |
|  | 0.0% 无酒精红味美思风格饮品 30 ml |
|  | 0.0% 无酒精芳香苦精 2 dashes |
|  | 0.0% 酒渍风味樱桃 1 颗 |

## 步骤 / Steps
1. 将所有液体加入调酒杯。
2. 加满冰块，用吧勺搅拌约 20-30 秒。
3. 过滤入预冷杯或装有大冰块的古典杯。
4. 完成装饰。

## 风味 / Flavor
木香、香草、红果、草本与香料苦味结合，尾段温暖。

## 情绪 / Mood
都市、成熟、晚餐后。

## 调酒师笔记 / Bartender Notes
- 两种基底都要偏干。
- 充分搅拌以获得丝滑质地。
- 装饰樱桃避免使用酒渍产品。
