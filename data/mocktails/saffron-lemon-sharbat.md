# Saffron Lemon Sharbat / 藏红花柠檬夏尔巴特

## 故事 / Story
藏红花和 Sharbat 都与波斯饮食传统密切相关。现代版本常将藏红花、柠檬和甜味结合，形成明亮金黄色饮品。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 30 ml | 藏红花浸泡液 |
| 25 ml | 新鲜柠檬汁 |
| 20 ml | 蜂蜜糖浆 |
| 150 ml | 冷水 |
| 1 粒轻压 | 豆蔻 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
藏红花的干草、蜂蜜和花香与柠檬酸结合，豆蔻带来温暖尾韵。

## 情绪 / Mood
精致、安静、适合慢饮。

## 调酒师笔记 / Bartender Notes
- 藏红花需提前用温水浸泡。
- 不要过量，否则会发药味。
- 豆蔻只轻压一粒。
