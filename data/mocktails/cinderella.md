# Cinderella / 灰姑娘

## 故事 / Story
Cinderella 是 20 世纪无酒精酒吧菜单中最常见的经典之一。它以多种果汁、酸味和气泡构成完整结构，名字则延续了早期无酒精饮品偏爱童话与明星意象的传统。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 橙汁 |
| 45 ml | 菠萝汁 |
| 20 ml | 新鲜柠檬汁 |
| 10 ml | 石榴糖浆 |
| 60 ml | 苏打水 |
|  | 橙片和樱桃装饰 |

## 步骤 / Steps
1. 将除气泡材料外的液体加入摇酒壶。
2. 加满冰块，用力摇和 8-12 秒。
3. 过滤入装有新鲜冰块的杯中。
4. 补入气泡材料，轻轻搅拌。
5. 完成装饰后立即饮用。

## 风味 / Flavor
橙子和菠萝的热带甜香先出现，柠檬把结构拉紧，石榴带来红果色泽，最后以气泡收束。

## 情绪 / Mood
明亮、友好、带一点复古酒店感。它适合聚会开场，也适合第一次认真尝试 mocktail 的人。

## 调酒师笔记 / Bartender Notes
- 菠萝汁会天然起泡，摇和时间不宜过长。
- 苏打水最后补入，避免提前跑气。
- 石榴糖浆只负责颜色和底味，不要让它压过柑橘。
