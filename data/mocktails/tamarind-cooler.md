# Tamarind Cooler / 罗望子清凉饮

## 故事 / Story
罗望子饮广泛存在于南亚、东南亚、拉丁美洲和加勒比地区。Cooler 形式利用气泡和姜把传统酸甜浆饮转为现代长饮。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 30 ml | 罗望子浆 |
| 20 ml | 黑糖糖浆 |
| 15 ml | 新鲜青柠汁 |
| 120 ml | 冷水 |
| 60 ml | 姜汁汽水 |
| 1 小撮 | 盐 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
罗望子独特酸甜、黑糖、青柠和姜辣结合，尾段略咸。

## 情绪 / Mood
热带、街头、醒胃。

## 调酒师笔记 / Bartender Notes
- 罗望子浓缩浆酸度差异大。
- 一小撮盐非常重要。
- 姜汁汽水最后加入。
