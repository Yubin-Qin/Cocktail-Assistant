# Rooibos Citrus Cooler / 路易波士柑橘清凉饮

## 故事 / Story
Rooibos 不含咖啡因且带自然木甜香，适合做夜间无酒精长饮基底。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 90 ml | 浓冷泡路易波士茶 |
| 45 ml | 葡萄柚汁 |
| 15 ml | 蜂蜜糖浆 |
| 90 ml | 苏打水 |
|  | 葡萄柚皮装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
路易波士木甜、葡萄柚微苦、蜂蜜和气泡柔和。

## 情绪 / Mood
黄昏、温暖、低刺激。

## 调酒师笔记 / Bartender Notes
- 茶要冷却后使用。
- 葡萄柚甜度决定蜂蜜量。
- 可加少量盐。
