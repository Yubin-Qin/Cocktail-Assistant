# Zero Amaretto Sour / 无酒精杏仁酸

## 故事 / Story
Amaretto Sour 因杏仁甜香而广受欢迎。无酒精杏仁风味产品能较自然地保留原版识别。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精杏仁风味利口酒 50 ml |
| 30 ml | 新鲜柠檬汁 |
| 5-10 ml | 单糖浆 |
| 20 ml | aquafaba |
|  | 柠檬皮装饰 |

## 步骤 / Steps
1. 将除装饰外的材料加入摇酒壶，先不加冰干摇 10 秒。
2. 加入冰块，再用力摇和 10-12 秒。
3. 双重过滤入预冷杯或装有新鲜冰块的杯中。
4. 静置数秒让泡沫稳定，完成装饰。

## 风味 / Flavor
杏仁、杏核、柠檬酸和细腻泡沫，甜感明显但有足够酸度支撑。

## 情绪 / Mood
柔软、甜点感、带怀旧气息。

## 调酒师笔记 / Bartender Notes
- 无酒精杏仁基底通常已经很甜。
- 柠檬汁要足。
- 干摇后再加冰摇。
