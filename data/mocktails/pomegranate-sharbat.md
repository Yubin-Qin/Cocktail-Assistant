# Pomegranate Sharbat / 石榴夏尔巴特

## 故事 / Story
石榴在波斯与中东饮食文化中具有悠久地位。Pomegranate Sharbat 用石榴的酸涩和颜色形成比普通果汁更有仪式感的饮品。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 90 ml | 石榴汁 |
| 10 ml | 石榴糖浆 |
| 15 ml | 新鲜青柠汁 |
| 90 ml | 冷水或苏打水 |
| 4 片 | 薄荷叶 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
石榴红果酸涩、青柠明亮、轻甜和薄荷清香。

## 情绪 / Mood
华丽、节庆、适合餐前。

## 调酒师笔记 / Bartender Notes
- 优先使用纯石榴汁。
- 糖浆只用于修正酸度。
- 苏打版本更现代。
