# Zero Moscow Mule / 无酒精莫斯科骡子

## 故事 / Story
Moscow Mule 本就依赖姜汁啤酒和青柠形成辨识度，因此非常适合零酒精改写。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精伏特加风味蒸馏液 50 ml |
| 15 ml | 新鲜青柠汁 |
| 120 ml | 姜汁啤酒 |
|  | 青柠角装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
姜辣、青柠酸、强气泡和干净基底，直接而醒神。

## 情绪 / Mood
社交、轻松、第一口就有存在感。

## 调酒师笔记 / Bartender Notes
- 姜汁啤酒应有真实辛辣感。
- 不要过度搅拌。
- 铜杯并非必须。
