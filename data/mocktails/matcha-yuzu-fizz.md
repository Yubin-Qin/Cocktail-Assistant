# Matcha Yuzu Fizz / 抹茶柚子菲士

## 故事 / Story
抹茶与柚子都是东亚风味中辨识度极高的元素。Fizz 结构把两者变成更接近酒吧长饮的形式。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 2 g | 抹茶 |
| 40 ml | 冷水 |
| 20 ml | 柚子汁 |
| 20 ml | 蜂蜜糖浆 |
| 120 ml | 苏打水 |

## 步骤 / Steps
1. 将除气泡材料外的液体加入摇酒壶。
2. 加满冰块，用力摇和 8-12 秒。
3. 过滤入装有新鲜冰块的杯中。
4. 补入气泡材料，轻轻搅拌。
5. 完成装饰后立即饮用。

## 风味 / Flavor
抹茶青草、柚子高香、蜂蜜和气泡，酸甜苦三者平衡。

## 情绪 / Mood
清新、精致、适合春夏。

## 调酒师笔记 / Bartender Notes
- 柚子汁酸度强。
- 抹茶先筛粉。
- 蜂蜜不要掩盖柚子香。
