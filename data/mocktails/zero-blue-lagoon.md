# Zero Blue Lagoon / 无酒精蓝色珊瑚礁

## 故事 / Story
Blue Lagoon 是 20 世纪后期流行的蓝色鸡尾酒。零酒精版本保留蓝柑橘与柠檬汽水的视觉和口味记忆。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精柑橘蒸馏液 45 ml |
| 20 ml | 蓝柑橘糖浆 |
| 20 ml | 新鲜柠檬汁 |
| 90 ml | 柠檬青柠汽水 |
|  | 柠檬轮装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
柑橘甜酸、明亮蓝色和气泡，风格直白、视觉先行。

## 情绪 / Mood
派对、童趣、复古夜店感。

## 调酒师笔记 / Bartender Notes
- 蓝柑橘糖浆通常很甜。
- 增加柠檬汁可改善平衡。
- 透明冰块更漂亮。
