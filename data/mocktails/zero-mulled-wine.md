# Zero Mulled Wine / 无酒精热红酒

## 故事 / Story
Mulled Wine 是欧洲冬季传统。零酒精版本用 0.0% 红葡萄酒替代饮品保留葡萄和香料结构。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精红葡萄酒替代饮品 500 ml |
| 4 片 | 橙片 |
| 2 根 | 肉桂棒 |
| 4 粒 | 丁香 |
| 1 颗 | 八角 |
| 30 ml | 蜂蜜 |

## 步骤 / Steps
1. 将除新鲜柑橘汁外的材料放入小锅。
2. 小火加热至冒热气，不要沸腾。
3. 离火后加入新鲜柑橘汁并试味。
4. 倒入耐热杯，完成装饰。

## 风味 / Flavor
温热红果、橙皮、肉桂和丁香，香气浓郁。

## 情绪 / Mood
冬夜、节日、壁炉感。

## 调酒师笔记 / Bartender Notes
- 不要煮沸。
- 香料浸泡过久会发苦。
- 甜度最后调整。
