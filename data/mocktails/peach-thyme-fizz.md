# Peach Thyme Fizz / 蜜桃百里香菲士

## 故事 / Story
桃与百里香是现代季节性调饮中的成熟组合。Fizz 结构能避免桃泥过于甜厚。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 白桃泥 |
| 15 ml | 百里香糖浆 |
| 20 ml | 新鲜柠檬汁 |
| 120 ml | 苏打水 |
|  | 百里香枝装饰 |

## 步骤 / Steps
1. 将除气泡材料外的液体加入摇酒壶。
2. 加满冰块，用力摇和 8-12 秒。
3. 过滤入装有新鲜冰块的杯中。
4. 补入气泡材料，轻轻搅拌。
5. 完成装饰后立即饮用。

## 风味 / Flavor
桃香柔软，百里香带干燥草本感，柠檬和气泡提升线条。

## 情绪 / Mood
夏末、温柔、适合黄昏。

## 调酒师笔记 / Bartender Notes
- 百里香糖浆只需少量。
- 桃泥过稠时增加少量水。
- 苏打最后加入。
