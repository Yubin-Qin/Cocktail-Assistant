# Cucumber Cooler / 黄瓜清凉饮

## 故事 / Story
Cucumber Cooler 并没有唯一固定起源，而是现代酒吧无酒精菜单中的常见经典风格。它继承了高球饮的清爽结构，也借鉴了英式琴酒长饮中常见的黄瓜和柑橘香气。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 5-6 片 | 黄瓜片 |
| 20 ml | 新鲜青柠汁 |
| 15 ml | 单糖浆 |
| 120 ml | 苏打水 |
| 4-6 片 | 薄荷叶 |
|  | 黄瓜长片装饰 |

## 步骤 / Steps
1. 摇酒壶中加入黄瓜片、青柠汁和糖浆，轻轻捣压。
2. 加冰摇和 8-10 秒。
3. 双重过滤入加冰高球杯。
4. 补入苏打水并轻搅。
5. 用黄瓜长片装饰。

## 风味 / Flavor
黄瓜清香、青柠酸、轻甜、薄荷凉感和气泡。整体干净、透明、偏清爽。

## 情绪 / Mood
克制、清透、像清晨或水疗空间。适合不想喝甜饮、又希望有酒吧质感的人。

## 调酒师笔记 / Bartender Notes
- 黄瓜只需压出汁和香气，不要捣得过碎。
- 双重过滤能去掉小碎渣，让口感更清爽。
- 可加入少量盐水或白葡萄 verjus，让风味更像成人饮品。
