# Zero Mai Tai / 无酒精迈泰

## 故事 / Story
Mai Tai 是 Tiki 经典，核心并不是混合果汁，而是朗姆、青柠、杏仁和橙味。零酒精版本保留这一更正统的结构。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精朗姆风味蒸馏液 60 ml |
| 25 ml | 新鲜青柠汁 |
| 15 ml | 杏仁糖浆 |
| 10 ml | 橙皮糖浆 |
|  | 薄荷枝和青柠壳装饰 |

## 步骤 / Steps
1. 将所有液体材料加入摇酒壶。
2. 加满冰块，用力摇和 10-12 秒。
3. 双重过滤入预冷鸡尾酒杯，或过滤入装有新鲜冰块的杯中。
4. 完成装饰后立即饮用。

## 风味 / Flavor
青柠、杏仁、橙皮和甘蔗香层次丰富，甜度集中但有酸度支撑。

## 情绪 / Mood
热带、复古、适合认真慢喝。

## 调酒师笔记 / Bartender Notes
- 杏仁糖浆很强，不能过量。
- 不要加入大量菠萝汁，以免变成普通热带饮。
- 碎冰最合适。
