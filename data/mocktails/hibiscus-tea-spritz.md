# Hibiscus Tea Spritz / 洛神花茶气泡

## 故事 / Story
洛神花天然提供类似红色开胃酒的酸涩与颜色，因此非常适合无酒精 Spritz。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 75 ml | 浓冷泡洛神花茶 |
|  | 0.0% 无酒精干型起泡酒 90 ml |
| 20 ml | 石榴汁 |
|  | 橙皮装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
洛神花酸涩、石榴红果和干型气泡，颜色深红。

## 情绪 / Mood
庆祝、成熟、适合餐前。

## 调酒师笔记 / Bartender Notes
- 起泡酒必须 0.0%。
- 不必额外加糖。
- 橙皮能提升开胃酒感。
