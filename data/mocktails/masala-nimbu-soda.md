# Masala Nimbu Soda / 印度香料柠檬苏打

## 故事 / Story
Masala Nimbu Soda 是印度常见的香料柠檬汽水，把 Nimbu Pani 的逻辑与强气泡结合。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 25 ml | 新鲜青柠汁 |
| 15 ml | 单糖浆 |
| 1 小撮 | 黑盐 |
| 1 小撮 | 烤孜然粉 |
| 150 ml | 苏打水 |
|  | 薄荷装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
青柠酸、气泡、咸味和孜然香，非常醒神。

## 情绪 / Mood
街头、热天、带一点刺激。

## 调酒师笔记 / Bartender Notes
- 苏打水越强越好。
- 香料先与青柠和糖浆混匀。
- 不要过度搅拌。
