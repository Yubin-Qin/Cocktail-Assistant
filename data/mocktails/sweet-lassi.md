# Sweet Lassi / 甜拉西

## 故事 / Story
Sweet Lassi 是 Lassi 家族的经典甜型版本。它比芒果拉西更直接突出酸奶本身。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 150 g | 原味酸奶 |
| 90 ml | 冷水 |
| 20 ml | 糖或蜂蜜 |
| 1 小撮 | 豆蔻粉 |
| 1 滴 | 玫瑰水 |

## 步骤 / Steps
1. 将所有液体、果肉和冰块放入搅拌机。
2. 高速搅打至均匀顺滑。
3. 试味并调整酸甜。
4. 倒入冰冷杯中，完成装饰。

## 风味 / Flavor
酸奶酸香、甜味和豆蔻柔和，质地顺滑。

## 情绪 / Mood
平静、家常、适合午餐。

## 调酒师笔记 / Bartender Notes
- 不要加入太多玫瑰水。
- 可用冰水调整稠度。
- 现打现喝。
