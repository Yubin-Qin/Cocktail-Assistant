# Night at the Spritz / 夜色气泡

## 故事 / Story
现代零酒精 Spritz 常直接以无酒精苦味开胃酒加苏打水构成，不必依赖复杂配方。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精苦味开胃酒 60 ml |
| 30 ml | 葡萄柚汁 |
| 120 ml | 苏打水 |
|  | 橙皮装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
苦橙、葡萄柚和气泡，极简、偏干。

## 情绪 / Mood
黄昏、餐前、城市露台。

## 调酒师笔记 / Bartender Notes
- 开胃酒甜时提高苏打比例。
- 杯中加足大冰。
- 橙皮必须挤油。
