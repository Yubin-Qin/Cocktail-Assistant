# Zero Dark 'n' Stormy / 无酒精黑暗风暴

## 故事 / Story
Dark ’n’ Stormy 的视觉来自深色朗姆浮在姜汁啤酒上。零酒精版本可用深色朗姆风味蒸馏液保留这一效果。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精深色朗姆风味蒸馏液 50 ml |
| 120 ml | 姜汁啤酒 |
| 15 ml | 新鲜青柠汁 |
| 5 ml | 黑糖糖浆 |
|  | 青柠角装饰 |

## 步骤 / Steps
1. 先在杯中加入气泡或较轻的基底材料和冰块。
2. 将较深色或较浓的材料沿吧勺背缓慢倒入。
3. 保留分层，或饮用前轻搅。
4. 完成装饰。

## 风味 / Flavor
深色香料、姜辣、青柠与黑糖形成明暗分层。

## 情绪 / Mood
戏剧性、海风、适合黄昏。

## 调酒师笔记 / Bartender Notes
- 先倒姜汁啤酒，再缓慢浮顶基底。
- 黑糖糖浆只在基底太薄时使用。
- 轻搅后再饮可观察层次变化。
