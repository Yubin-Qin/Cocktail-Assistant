# Fruit Punch / 水果潘趣

## 故事 / Story
Punch 的历史可追溯到早期殖民贸易时代，传统上包含酒、糖、柑橘、水和香料等元素。无酒精 Fruit Punch 保留了“多人分享”和“酸甜香料平衡”的精神，是派对和节庆场景中最经典的大份饮品。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 橙汁 |
| 60 ml | 菠萝汁 |
| 45 ml | 蔓越莓汁或石榴汁 |
| 15 ml | 新鲜柠檬汁 |
| 10 ml | 单糖浆 |
| 60 ml | 苏打水或姜汁汽水 |
|  | 橙片、莓果、薄荷装饰 |

## 步骤 / Steps
1. 大杯或潘趣碗中加入果汁、柠檬汁和糖浆。
2. 加入大量冰块并搅拌。
3. 饮用前补入苏打水或姜汁汽水。
4. 加入水果片和薄荷装饰。

## 风味 / Flavor
多层水果甜香、柑橘酸、菠萝热带感和气泡。风味亲切、颜色热闹。

## 情绪 / Mood
分享、热闹、欢迎感强。它适合聚会，因为不需要每杯单独调制，却仍然有仪式感。

## 调酒师笔记 / Bartender Notes
- 至少要有一种酸味果汁支撑，否则会过甜。
- 红色果汁提供颜色和微涩感，菠萝汁提供热带厚度。
- 批量制作时，气泡饮料最后加入，避免提前跑气。
