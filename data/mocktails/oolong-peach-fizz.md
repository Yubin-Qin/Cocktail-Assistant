# Oolong Peach Fizz / 乌龙蜜桃菲士

## 故事 / Story
桃与乌龙是东亚饮品中非常成熟的风味搭配。放入 Fizz 结构后，茶的涩感能有效限制桃子的甜。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 75 ml | 浓冷泡乌龙茶 |
| 30 ml | 白桃泥 |
| 20 ml | 新鲜柠檬汁 |
| 10 ml | 单糖浆 |
| 90 ml | 苏打水 |

## 步骤 / Steps
1. 将除气泡材料外的液体加入摇酒壶。
2. 加满冰块，用力摇和 8-12 秒。
3. 过滤入装有新鲜冰块的杯中。
4. 补入气泡材料，轻轻搅拌。
5. 完成装饰后立即饮用。

## 风味 / Flavor
乌龙焙火、桃香、柠檬酸和气泡，果香柔和而不腻。

## 情绪 / Mood
夏日、茶园、温柔。

## 调酒师笔记 / Bartender Notes
- 乌龙选择中度烘焙。
- 桃泥不要太多。
- 双重过滤更细腻。
