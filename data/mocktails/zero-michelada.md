# Zero Michelada / 无酒精米切拉达

## 故事 / Story
Michelada 是墨西哥啤酒调饮的大类。零酒精版本必须使用 0.0% 啤酒，并保留青柠、盐和辣味。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精拉格啤酒 180 ml |
| 25 ml | 新鲜青柠汁 |
| 45 ml | 番茄汁 |
| 3 dashes | 伍斯特风味无酒精调味汁 |
| 2 dashes | 辣椒酱 |
|  | 辣盐杯边 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
麦芽、青柠、咸鲜和辣味结合，清爽又开胃。

## 情绪 / Mood
热天、街头、早午餐。

## 调酒师笔记 / Bartender Notes
- 确认啤酒为 0.0%。
- 不同地区 Michelada 是否加番茄差异很大。
- 轻搅避免泡沫过度。
