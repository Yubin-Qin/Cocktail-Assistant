# Zero Spicy Margarita / 无酒精辣味玛格丽特

## 故事 / Story
辣味 Margarita 是现代经典分支，无酒精版本尤其适合，因为辣椒能提供原本由酒精带来的灼热与长度。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精龙舌兰风味蒸馏液 45 ml |
| 25 ml | 新鲜青柠汁 |
| 15 ml | 龙舌兰糖浆 |
| 2-3 片 | 墨西哥辣椒 |
| 2 片 | 香菜叶 |
|  | 辣盐半边杯口 |

## 步骤 / Steps
1. 在摇酒壶中轻压水果、香草或香料。
2. 加入其余液体材料。
3. 加冰用力摇和 10-12 秒。
4. 双重过滤入预冷杯或装有新鲜冰块的杯中。
5. 完成装饰。

## 风味 / Flavor
青柠酸、辣椒热感、草本和轻甜叠加，刺激感承担了部分酒精的角色。

## 情绪 / Mood
活跃、直接、适合热闹场景。

## 调酒师笔记 / Bartender Notes
- 辣椒辣度差异很大，先少量。
- 不要把辣椒籽大量捣碎。
- 盐边能让辣味更圆。
