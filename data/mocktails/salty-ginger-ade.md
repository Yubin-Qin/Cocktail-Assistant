# Salty Ginger-Ade / 咸姜汁汽水

## 故事 / Story
这类配方受 Jaljeera 启发，用椰枣、孜然、盐和姜汁啤酒构建现代酒吧风格。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 1 颗去核 | 椰枣 |
| 25 ml | 新鲜青柠汁 |
| 120 ml | 姜汁啤酒 |
| 1 小撮 | 烤孜然粉 |
| 1 小撮 | 黑盐 |
| 4 片 | 薄荷叶 |

## 步骤 / Steps
1. 在杯中轻压水果或香草，只释放汁液和香气。
2. 加入其余非气泡材料并搅匀。
3. 加满冰块。
4. 补入气泡材料，轻轻上下搅拌。
5. 完成装饰。

## 风味 / Flavor
椰枣甜、青柠酸、姜辣、孜然和黑盐形成复杂咸甜结构。

## 情绪 / Mood
醒神、热天、带南亚香料感。

## 调酒师笔记 / Bartender Notes
- 椰枣捣成泥后要充分混匀。
- 黑盐少量。
- 姜汁啤酒最后加入。
