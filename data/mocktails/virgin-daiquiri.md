# Virgin Daiquiri / 无酒精戴基里

## 故事 / Story
它是经典 Daiquiri 的零酒精重构。原版依赖朗姆、青柠和糖的三角平衡，无酒精版本的重点同样不是复杂，而是温度、酸甜与适度质地。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精甘蔗风味蒸馏液 50 ml |
| 25 ml | 新鲜青柠汁 |
| 15-20 ml | 单糖浆 |

## 步骤 / Steps
1. 将所有液体材料加入摇酒壶。
2. 加满冰块，用力摇和 10-12 秒。
3. 双重过滤入预冷鸡尾酒杯，或过滤入装有新鲜冰块的杯中。
4. 完成装饰后立即饮用。

## 风味 / Flavor
青柠酸度紧致，甘蔗与轻微香草感提供骨架，整体清亮、直接。

## 情绪 / Mood
轻快、坦率、像热天里的一条干净线条。

## 调酒师笔记 / Bartender Notes
- 无酒精甘蔗风味基酒最好选择 0.0% 产品。
- 青柠酸度每天不同，糖浆应微调。
- 摇好后立刻饮用。
