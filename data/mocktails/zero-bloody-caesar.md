# Zero Bloody Caesar / 无酒精凯撒

## 故事 / Story
Caesar 是加拿大标志性鸡尾酒，和 Bloody Mary 最大区别是使用蛤蜊番茄风味汁。无酒精版本保留其核心咸鲜。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 120 ml | 蛤蜊番茄风味汁 |
| 15 ml | 新鲜青柠汁 |
| 3 dashes | 伍斯特风味无酒精调味汁 |
| 2 dashes | 辣椒酱 |
|  | 芹菜盐和黑胡椒适量 |
|  | 芹菜杆装饰 |

## 步骤 / Steps
1. 将所有液体和调味料加入一个装有冰块的容器。
2. 在两个容器之间来回缓慢倒 4-6 次。
3. 倒入装有新鲜冰块的高球杯。
4. 完成装饰。

## 风味 / Flavor
番茄、海鲜鲜味、酸、盐和辣味层次浓郁。

## 情绪 / Mood
醒神、咸鲜、适合早午餐。

## 调酒师笔记 / Bartender Notes
- 检查调味汁不含酒精。
- 用滚动法避免起泡。
- 咸度要谨慎。
