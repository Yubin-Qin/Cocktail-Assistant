# Chapman / 查普曼

## 故事 / Story
Chapman 是尼日利亚最具代表性的无酒精混合饮之一。传统版本常加入 aromatic bitters；这里改用 0.0% 无酒精苦精。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 75 ml | 橙味汽水 |
| 75 ml | 柠檬青柠汽水 |
| 15 ml | 石榴糖浆 |
| 15 ml | 新鲜青柠汁 |
|  | 0.0% 无酒精芳香苦精 2 dashes |
|  | 黄瓜片和橙片装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
柑橘汽水、石榴、青柠、香料苦味和黄瓜清香层次丰富。

## 情绪 / Mood
庆祝、社交、尼日利亚派对感。

## 调酒师笔记 / Bartender Notes
- 务必确认苦精 0.0%。
- 黄瓜是重要香气元素。
- 汽水已经很甜，石榴糖浆适量。
