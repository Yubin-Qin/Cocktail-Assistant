# Virgin Mary / 无酒精血腥玛丽

## 故事 / Story
Virgin Mary 是 Bloody Mary 的无酒精版本。Bloody Mary 本身与 20 世纪巴黎和纽约酒吧文化相关，而无酒精版本保留番茄、香料和咸鲜结构，成为早午餐和酒店菜单的常客。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 120 ml | 番茄汁 |
| 15 ml | 新鲜柠檬汁 |
| 2-3 dashes | 伍斯特辣酱油 |
| 1-2 dashes | 辣椒酱 |
| 1 小撮 | 芹菜盐 |
| 1 小撮 | 黑胡椒 |
| 1/2 茶匙 | 可选：辣根 |
|  | 芹菜杆、橄榄或柠檬角装饰 |

## 步骤 / Steps
1. 摇酒壶或调酒杯中加入所有液体和调味料。
2. 加冰后用“滚动法”在两个容器间来回倒 4-6 次，或轻摇数秒。
3. 倒入加冰高球杯。
4. 用芹菜杆、橄榄或柠檬角装饰。

## 风味 / Flavor
番茄鲜味、柠檬酸、咸鲜、香料和辣感。它更接近开胃冷汤，而非传统甜饮。

## 情绪 / Mood
醒神、务实、带一点早午餐的慵懒。它适合想喝得复杂但不想喝甜的时候。

## 调酒师笔记 / Bartender Notes
- 不要猛烈摇太久，否则番茄汁会起泡并显得粗糙。
- 调味应分层：酸、咸、辣、鲜，而不是单纯辣。
- 可用烟熏盐、泡菜汁或烤椒泥增加成人感。
