# Zero Clover Club / 无酒精三叶草俱乐部

## 故事 / Story
Clover Club 是费城经典，原版以琴酒、覆盆子、柠檬和蛋白著称。零酒精版本保留全部主要风味线索。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精杜松子风味蒸馏液 45 ml |
| 25 ml | 新鲜柠檬汁 |
| 20 ml | 覆盆子糖浆 |
| 20 ml | aquafaba |
|  | 覆盆子装饰 |

## 步骤 / Steps
1. 将除装饰外的材料加入摇酒壶，先不加冰干摇 10 秒。
2. 加入冰块，再用力摇和 10-12 秒。
3. 双重过滤入预冷杯或装有新鲜冰块的杯中。
4. 静置数秒让泡沫稳定，完成装饰。

## 风味 / Flavor
覆盆子、柠檬、杜松与细腻泡沫结合，酸甜优雅。

## 情绪 / Mood
复古、精致、带老俱乐部气息。

## 调酒师笔记 / Bartender Notes
- 覆盆子糖浆应有真实果酸。
- 先干摇后冰摇。
- 双重过滤更细。
