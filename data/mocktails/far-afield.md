# Far Afield / 远方

## 故事 / Story
Far Afield 一类现代 stirred zero-proof 饮品强调 verjus、植物蒸馏液和茶糖浆，用非果汁方式获得结构。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精花草开胃酒 30 ml |
|  | 0.0% 无酒精植物蒸馏液 30 ml |
| 30 ml | 白 verjus |
| 15 ml | 姜柑橘茶糖浆 |
|  | 葡萄柚皮装饰 |

## 步骤 / Steps
1. 将所有液体加入调酒杯。
2. 加满冰块，用吧勺搅拌约 20-30 秒。
3. 过滤入预冷杯或装有大冰块的古典杯。
4. 完成装饰。

## 风味 / Flavor
白葡萄酸、姜、柑橘、花草与轻微苦味，像无酒精 White Negroni。

## 情绪 / Mood
安静、复杂、适合慢慢分析。

## 调酒师笔记 / Bartender Notes
- verjus 应偏酸而不甜。
- 充分搅拌。
- 成品适合小杯无冰。
