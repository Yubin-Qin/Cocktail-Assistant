# Zero Hurricane / 无酒精飓风

## 故事 / Story
Hurricane 与新奥尔良 Pat O’Brien’s 酒吧密切相关。无酒精版本保留百香果、柑橘和夸张杯型。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 百香果汁 |
| 45 ml | 橙汁 |
| 25 ml | 新鲜柠檬汁 |
|  | 0.0% 无酒精朗姆风味蒸馏液 45 ml |
| 10 ml | 石榴糖浆 |
|  | 橙片装饰 |

## 步骤 / Steps
1. 将所有液体材料加入摇酒壶。
2. 加满冰块，用力摇和 10-12 秒。
3. 双重过滤入预冷鸡尾酒杯，或过滤入装有新鲜冰块的杯中。
4. 完成装饰后立即饮用。

## 风味 / Flavor
百香果与橙子热烈，柠檬提供酸度，甘蔗香和石榴拉长尾段。

## 情绪 / Mood
新奥尔良式热闹、明亮。

## 调酒师笔记 / Bartender Notes
- 百香果的酸度会影响糖浆需求。
- 石榴糖浆不要太多。
- 大杯加足冰。
