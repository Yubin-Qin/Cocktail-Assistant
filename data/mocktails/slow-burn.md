# Slow Burn / 慢燃

## 故事 / Story
Slow Burn 代表现代无酒精调酒常用辣椒、烟熏和酸味模拟酒精抓力的思路。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精龙舌兰风味蒸馏液 45 ml |
| 25 ml | 新鲜青柠汁 |
| 15 ml | 枫糖浆 |
| 10 ml | 石榴糖蜜 |
|  | 烟熏辣盐半边杯口 |

## 步骤 / Steps
1. 将所有液体材料加入摇酒壶。
2. 加满冰块，用力摇和 10-12 秒。
3. 双重过滤入预冷鸡尾酒杯，或过滤入装有新鲜冰块的杯中。
4. 完成装饰后立即饮用。

## 风味 / Flavor
青柠、石榴酸甜、枫糖和烟熏辣味逐渐展开。

## 情绪 / Mood
大胆、夜晚、后劲来自辣味而不是酒精。

## 调酒师笔记 / Bartender Notes
- 石榴糖蜜非常浓。
- 辣盐做半圈。
- 枫糖只用于圆润。
