# Virgin Pina Colada / 无酒精椰林飘香

## 故事 / Story
Pina Colada 与波多黎各密切相关，原版以朗姆、椰子和菠萝构成热带标志。无酒精版本保留了最具辨识度的椰子和菠萝结构，因此在酒店、家庭聚会和海边餐厅中非常常见。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 90 ml | 菠萝汁 |
| 45 ml | 椰浆或椰奶 |
| 15 ml | 椰子糖浆 |
| 10 ml | 青柠汁 |
| 1 杯 | 碎冰 |
|  | 菠萝片和樱桃装饰 |

## 步骤 / Steps
1. 搅拌机中加入菠萝汁、椰浆、青柠汁、糖浆和碎冰。
2. 搅打至顺滑冰沙状。
3. 倒入飓风杯或大杯。
4. 用菠萝片和樱桃装饰。

## 风味 / Flavor
菠萝、椰子、奶油质地、热带水果甜香和轻微青柠酸。柔滑、饱满、像甜点但更清爽。

## 情绪 / Mood
度假、柔软、外向。它几乎自带阳伞和海边想象，是最有逃离感的无酒精经典之一。

## 调酒师笔记 / Bartender Notes
- 椰浆较厚重，椰奶更轻；可按想要的质地选择。
- 青柠汁非常重要，用来抵消椰子和菠萝的甜腻。
- 不要加太多冰，否则香气会被稀释。
