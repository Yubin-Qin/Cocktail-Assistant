# Lemon Lime & Zero Bitters / 柠檬青柠无酒精苦精

## 故事 / Story
Lemon, Lime & Bitters 在澳大利亚和新西兰极具代表性。传统做法常使用含酒精的芳香苦精；这里明确改用 0.0% 产品，以保持真正无酒精。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 15 ml | 新鲜柠檬汁 |
| 15 ml | 新鲜青柠汁 |
| 15 ml | 单糖浆 |
| 120 ml | 苏打水 |
|  | 0.0% 无酒精芳香苦精 3 dashes |
|  | 柠檬片装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
柑橘酸、香料苦和气泡清脆结合，尾段有类似传统 bitters 的肉桂、丁香与草本感。

## 情绪 / Mood
成熟、社交、非常适合作为餐前长饮。

## 调酒师笔记 / Bartender Notes
- 购买苦精时务必确认 0.0% ABV。
- 苦精只需少量，过多会掩盖柑橘。
- 杯子预冷会显著改善口感。
