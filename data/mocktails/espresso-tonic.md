# Espresso Tonic / 浓缩咖啡汤力

## 故事 / Story
Espresso Tonic 在现代咖啡馆和酒吧文化中广泛流行。它把浓缩咖啡的厚重与汤力水的气泡和奎宁苦味放在同一杯中。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 30 ml | 新鲜浓缩咖啡 |
| 120 ml | 汤力水 |
| 1 片 | 橙皮 |
|  | 大冰块适量 |

## 步骤 / Steps
1. 先在杯中加入气泡或较轻的基底材料和冰块。
2. 将较深色或较浓的材料沿吧勺背缓慢倒入。
3. 保留分层，或饮用前轻搅。
4. 完成装饰。

## 风味 / Flavor
咖啡烘焙苦、奎宁苦和橙皮香叠加，苦味复杂但气泡轻盈。

## 情绪 / Mood
清醒、现代、适合午后。

## 调酒师笔记 / Bartender Notes
- 先倒汤力水再缓慢加入咖啡。
- 咖啡过深烘会让苦味粗糙。
- 橙皮能连接两种苦味。
