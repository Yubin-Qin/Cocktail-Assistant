# Lime Rickey / 青柠里奇

## 故事 / Story
Rickey 原本是 19 世纪美国的高球家族，后来发展出不含烈酒的 Lime Rickey。它在汽水喷泉文化中长期流行，核心始终是青柠与苏打水。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 30 ml | 新鲜青柠汁 |
| 15 ml | 单糖浆 |
| 150 ml | 苏打水 |
|  | 青柠半颗挤压后的果壳 |
|  | 冰块适量 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
青柠酸香直接、气泡清脆、甜度很低，尾段带一点青柠皮的微苦。

## 情绪 / Mood
清醒、利落、旧式汽水柜台感。适合炎热天气和需要快速刷新味觉的时候。

## 调酒师笔记 / Bartender Notes
- 青柠要现挤，瓶装汁会失去精油香气。
- 糖浆只需少量，Rickey 不应像柠檬水那样甜。
- 可把挤过的青柠壳投入杯中增加皮香。
