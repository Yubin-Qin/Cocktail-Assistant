# Chai Fizz / 印度香料茶菲士

## 故事 / Story
Chai Fizz 把南亚香料茶的香气移入西式气泡长饮结构，是现代无酒精调酒常见的跨文化做法。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 浓缩香料茶 |
| 20 ml | 新鲜柠檬汁 |
| 15 ml | 黑糖糖浆 |
| 120 ml | 苏打水 |
|  | 姜片装饰 |

## 步骤 / Steps
1. 将除气泡材料外的液体加入摇酒壶。
2. 加满冰块，用力摇和 8-12 秒。
3. 过滤入装有新鲜冰块的杯中。
4. 补入气泡材料，轻轻搅拌。
5. 完成装饰后立即饮用。

## 风味 / Flavor
肉桂、豆蔻、姜、红茶、柠檬和气泡层次丰富。

## 情绪 / Mood
温暖又清爽、适合秋季。

## 调酒师笔记 / Bartender Notes
- 茶要浓。
- 香料糖浆与茶本身不要重复过重。
- 柠檬量保持克制。
