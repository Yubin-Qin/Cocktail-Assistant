# Butterfly Pea Lemon Fizz / 蝶豆花柠檬菲士

## 故事 / Story
蝶豆花饮在东南亚有传统基础，现代酒吧利用其天然酸碱变色特性制作视觉型无酒精饮品。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 75 ml | 蝶豆花茶 |
| 25 ml | 新鲜柠檬汁 |
| 20 ml | 单糖浆 |
| 90 ml | 苏打水 |
|  | 柠檬片装饰 |

## 步骤 / Steps
1. 先将有色茶基底和冰块加入杯中。
2. 缓慢加入苏打水。
3. 将酸味材料单独保留，在饮用前或桌边加入。
4. 观察颜色变化后轻轻搅拌，完成装饰。

## 风味 / Flavor
味道以柠檬酸甜为主，蝶豆花带轻微豆香；最大特点是酸加入后颜色由蓝转紫。

## 情绪 / Mood
梦幻、视觉感、适合聚会。

## 调酒师笔记 / Bartender Notes
- 先倒蓝色茶，再在桌边加入柠檬可观察变色。
- 蝶豆花本身味道很轻。
- 不要依赖颜色而忽视酸甜平衡。
