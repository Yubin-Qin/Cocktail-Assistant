# Vitamin Parade / 维生素巡游

## 故事 / Story
现代无酒精菜单常把冷压蔬果汁与 Sour、Fizz 技法结合。Vitamin Parade 是这种“果蔬鸡尾酒”方向的代表风格。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 胡萝卜汁 |
| 45 ml | 菠萝汁 |
| 20 ml | 新鲜柠檬汁 |
| 15 ml | 肉桂糖浆 |
| 20 ml | aquafaba |
| 60 ml | 姜汁啤酒 |

## 步骤 / Steps
1. 将除气泡材料外的材料加入摇酒壶，先不加冰干摇 10 秒。
2. 加冰再摇和 8-10 秒。
3. 过滤入装有新鲜冰块的杯中。
4. 补入气泡材料，让泡沫自然升起。
5. 完成装饰。

## 风味 / Flavor
胡萝卜甜、菠萝热带感、肉桂、柠檬和姜形成饱满复杂度。

## 情绪 / Mood
活力、明亮、适合白天。

## 调酒师笔记 / Bartender Notes
- 胡萝卜汁必须新鲜。
- 先干摇形成泡沫。
- 姜汁啤酒最后加入。
