# Agua de Jamaica / 洛神花水

## 故事 / Story
Agua de Jamaica 是墨西哥及拉丁美洲常见的 agua fresca，以干洛神花浸泡制成。它天然具有酸度和单宁感，非常适合无酒精调饮。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 10 g | 干洛神花 |
| 250 ml | 热水 |
| 20 ml | 单糖浆 |
| 10 ml | 新鲜青柠汁 |
|  | 冰块适量 |

## 步骤 / Steps
1. 按配方用热水或冷泡方式萃取花草、茶或香料。
2. 过滤并完全冷却。
3. 加入甜味、酸味和冰块，搅拌均匀。
4. 按需要补入水或气泡材料，完成装饰。

## 风味 / Flavor
洛神花酸涩、红果感和轻花香，颜色深红，收尾干净。

## 情绪 / Mood
明亮、热天、适合餐食搭配。

## 调酒师笔记 / Bartender Notes
- 不要浸泡过久以免过涩。
- 可冷泡获得更柔和口感。
- 青柠只需少量。
