# Zero Tom Collins / 无酒精汤姆柯林斯

## 故事 / Story
Tom Collins 是 19 世纪形成的经典长饮。无酒精版本保留酸、甜、植物基底和苏打的四段式结构。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精杜松子风味蒸馏液 45 ml |
| 25 ml | 新鲜柠檬汁 |
| 20 ml | 单糖浆 |
| 90 ml | 苏打水 |
|  | 柠檬轮装饰 |

## 步骤 / Steps
1. 将除气泡材料外的液体加入摇酒壶。
2. 加满冰块，用力摇和 8-12 秒。
3. 过滤入装有新鲜冰块的杯中。
4. 补入气泡材料，轻轻搅拌。
5. 完成装饰后立即饮用。

## 风味 / Flavor
柠檬酸、杜松草本和气泡构成修长而明亮的经典高球。

## 情绪 / Mood
清爽、整洁、适合白天。

## 调酒师笔记 / Bartender Notes
- 基底偏淡时可加入少量冷泡绿茶增加身体。
- 柠檬汁必须新鲜。
- 苏打水最后加入。
