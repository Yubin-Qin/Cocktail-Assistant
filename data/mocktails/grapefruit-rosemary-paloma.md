# Grapefruit Rosemary Paloma / 葡萄柚迷迭香帕洛玛

## 故事 / Story
这是 Paloma 的季节性草本改写。迷迭香与葡萄柚皮的苦香方向一致。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 75 ml | 葡萄柚汁 |
| 15 ml | 新鲜青柠汁 |
| 15 ml | 迷迭香糖浆 |
|  | 0.0% 无酒精龙舌兰风味蒸馏液 45 ml |
| 90 ml | 苏打水 |

## 步骤 / Steps
1. 将除气泡材料外的液体加入摇酒壶。
2. 加满冰块，用力摇和 8-12 秒。
3. 过滤入装有新鲜冰块的杯中。
4. 补入气泡材料，轻轻搅拌。
5. 完成装饰后立即饮用。

## 风味 / Flavor
葡萄柚苦甜、迷迭香松针感、青柠和草本基底层次清楚。

## 情绪 / Mood
黄昏、成熟、适合餐前。

## 调酒师笔记 / Bartender Notes
- 迷迭香不要过重。
- 葡萄柚甜度决定糖浆量。
- 大冰块更适合。
