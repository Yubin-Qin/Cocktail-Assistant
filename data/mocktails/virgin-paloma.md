# Virgin Paloma / 无酒精帕洛玛

## 故事 / Story
Paloma 是墨西哥最具代表性的长饮之一。零酒精版本保留葡萄柚、青柠、盐和龙舌兰式风味，因此辨识度依然很高。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精龙舌兰风味蒸馏液 45 ml |
| 60 ml | 新鲜葡萄柚汁 |
| 15 ml | 新鲜青柠汁 |
| 10 ml | 龙舌兰糖浆 |
| 90 ml | 苏打水 |
| 1 小撮 | 盐 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
葡萄柚苦甜、青柠酸和盐感构成清爽长饮，草本基底增加深度。

## 情绪 / Mood
阳光、海边、毫不费力。

## 调酒师笔记 / Bartender Notes
- 葡萄柚甜时减少糖浆。
- 盐只需一小撮。
- 苏打水最后加入。
