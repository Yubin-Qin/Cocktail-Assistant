# Limeade / 青柠水

## 故事 / Story
Limeade 是柠檬水家族的经典分支，在热带与美洲地区尤其常见。它的配方简单，却能直接检验鲜榨果汁和酸甜平衡。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 35 ml | 新鲜青柠汁 |
| 25 ml | 单糖浆 |
| 120 ml | 冷水 |
|  | 冰块适量 |
|  | 青柠片装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
青柠比柠檬更偏绿色、锐利和带皮香，甜度柔化酸感但保留干净收尾。

## 情绪 / Mood
直白、清爽、没有负担。它适合作为其他无酒精配方的基础训练。

## 调酒师笔记 / Bartender Notes
- 青柠酸度波动大，糖浆要按当日水果微调。
- 可加入一小撮盐提升香气。
- 不要久放，青柠汁氧化后会发苦。
