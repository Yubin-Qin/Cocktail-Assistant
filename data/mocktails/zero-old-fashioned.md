# Zero Old Fashioned / 无酒精古典鸡尾酒

## 故事 / Story
Old Fashioned 的核心本来就是烈酒、糖、水和苦精。零酒精版本最难复制的是身体感，因此常用枫糖等较浓甜味剂补充质地。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精威士忌风味蒸馏液 60 ml |
| 7.5 ml | 枫糖浆 |
|  | 0.0% 无酒精芳香苦精 2 dashes |
| 1 片 | 橙皮 |
| 1 块 | 大冰块 |

## 步骤 / Steps
1. 将所有液体加入调酒杯。
2. 加满冰块，用吧勺搅拌约 20-30 秒。
3. 过滤入预冷杯或装有大冰块的古典杯。
4. 完成装饰。

## 风味 / Flavor
烘烤木香、香草、枫糖、橙油和香料苦味层层展开。

## 情绪 / Mood
沉稳、安静、适合慢饮。

## 调酒师笔记 / Bartender Notes
- 甜味只需少量。
- 必须使用 0.0% 苦精。
- 大冰块能减慢稀释。
