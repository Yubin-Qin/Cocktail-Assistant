# Mint Limeade / 薄荷青柠水

## 故事 / Story
Mint Limeade 是中东、南亚与现代酒吧都常见的组合。它与 Limonana 等地区性薄荷柠檬饮共享相似逻辑：柑橘、甜味、薄荷和低温。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 30 ml | 新鲜青柠汁 |
| 20 ml | 单糖浆 |
| 10 片 | 薄荷叶 |
| 150 ml | 冷水或苏打水 |
|  | 碎冰适量 |
|  | 薄荷枝装饰 |

## 步骤 / Steps
1. 在杯中轻压水果或香草，只释放汁液和香气。
2. 加入其余非气泡材料并搅匀。
3. 加满冰块。
4. 补入气泡材料，轻轻上下搅拌。
5. 完成装饰。

## 风味 / Flavor
青柠明亮、薄荷清凉、甜度轻，碎冰让香气持续释放。

## 情绪 / Mood
轻快、降温、适合白天。它像 Mojito 的更简洁亲戚，但没有任何基酒依赖。

## 调酒师笔记 / Bartender Notes
- 薄荷只轻压，不要捣烂。
- 碎冰应干净无异味。
- 使用苏打水会更像长饮，冷水版本更柔和。
