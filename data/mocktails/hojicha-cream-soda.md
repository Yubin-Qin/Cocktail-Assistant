# Hojicha Cream Soda / 焙茶奶油苏打

## 故事 / Story
Hojicha 的烘烤香天然适合与奶油和香草结合。Cream Soda 结构让它兼具咖啡馆与酒吧的视觉感。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 75 ml | 浓焙茶 |
| 15 ml | 香草糖浆 |
| 90 ml | 苏打水 |
| 20 ml | 淡奶油 |
|  | 冰块适量 |

## 步骤 / Steps
1. 将主要基底材料加入冰冷杯中。
2. 缓慢加入气泡或第二层材料，尽量保留泡沫和层次。
3. 视配方需要轻搅，不要完全破坏分层。
4. 完成装饰后立即饮用。

## 风味 / Flavor
焙茶烘烤、香草、轻奶油和气泡，像轻盈版甜点。

## 情绪 / Mood
舒适、日系咖啡馆、适合傍晚。

## 调酒师笔记 / Bartender Notes
- 焙茶要足够浓。
- 奶油只需轻轻浮顶。
- 香草糖浆不要过多。
