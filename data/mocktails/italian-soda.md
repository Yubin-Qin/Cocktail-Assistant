# Italian Soda / 意式苏打

## 故事 / Story
Italian Soda 实际上更多与美国的意式糖浆和咖啡馆文化相关。它的核心不是固定口味，而是风味糖浆加苏打水的模块化结构。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 25 ml | 水果糖浆 |
| 180 ml | 苏打水 |
| 10 ml | 柠檬汁 |
|  | 鲜果或香草装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
风味取决于糖浆，可从柑橘、莓果到香草变化；气泡让甜味变得更轻。

## 情绪 / Mood
轻松、可定制、适合家庭吧台。它像一张空白配方纸，可以随季节变化。

## 调酒师笔记 / Bartender Notes
- 糖浆浓度差异很大，先少加再调整。
- 酸味水果可省略柠檬，花香糖浆则适合加少量酸。
- 气泡最后加入。
