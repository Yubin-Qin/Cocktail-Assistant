# Zero Last Word / 无酒精最后一句

## 故事 / Story
Last Word 以四种材料等量组成，是经典鸡尾酒中最著名的对称结构之一。零酒精版本同样可以利用等量公式重建节奏。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精杜松子风味蒸馏液 30 ml |
| 30 ml | 草本糖浆 |
| 30 ml | 青樱桃糖浆 |
| 30 ml | 新鲜青柠汁 |

## 步骤 / Steps
1. 将所有液体材料加入摇酒壶。
2. 加满冰块，用力摇和 10-12 秒。
3. 双重过滤入预冷鸡尾酒杯，或过滤入装有新鲜冰块的杯中。
4. 完成装饰后立即饮用。

## 风味 / Flavor
草本、樱桃、青柠和杜松子等量碰撞，味道集中而复杂。

## 情绪 / Mood
聪明、戏剧性、适合喜欢复杂风味的人。

## 调酒师笔记 / Bartender Notes
- 草本糖浆应偏苦而非纯甜。
- 青樱桃糖浆控制糖度。
- 等量是起点，可按甜度微调。
