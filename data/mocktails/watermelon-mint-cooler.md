# Watermelon Mint Cooler / 西瓜薄荷清凉饮

## 故事 / Story
西瓜、薄荷和青柠是全球夏季无酒精饮品的高频组合。少量盐能让它更像认真调制的长饮。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 120 ml | 西瓜汁 |
| 20 ml | 新鲜青柠汁 |
| 8 片 | 薄荷叶 |
| 1 小撮 | 盐 |
| 60 ml | 苏打水 |

## 步骤 / Steps
1. 在杯中轻压水果或香草，只释放汁液和香气。
2. 加入其余非气泡材料并搅匀。
3. 加满冰块。
4. 补入气泡材料，轻轻上下搅拌。
5. 完成装饰。

## 风味 / Flavor
西瓜清甜、青柠、薄荷和微盐，极度多汁。

## 情绪 / Mood
盛夏、泳池、放松。

## 调酒师笔记 / Bartender Notes
- 西瓜汁现榨。
- 薄荷只轻压。
- 甜西瓜无需糖浆。
