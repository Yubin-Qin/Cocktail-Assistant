# Florida Cocktail / 佛罗里达

## 故事 / Story
Florida Cocktail 是早期无酒精果汁混合饮的代表名称之一。其配方在不同资料中会变化，但核心通常围绕佛罗里达州标志性的柑橘。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 橙汁 |
| 45 ml | 葡萄柚汁 |
| 15 ml | 新鲜柠檬汁 |
| 10 ml | 单糖浆 |
| 60 ml | 苏打水 |
|  | 葡萄柚皮装饰 |

## 步骤 / Steps
1. 将除气泡材料外的液体加入摇酒壶。
2. 加满冰块，用力摇和 8-12 秒。
3. 过滤入装有新鲜冰块的杯中。
4. 补入气泡材料，轻轻搅拌。
5. 完成装饰后立即饮用。

## 风味 / Flavor
橙子的甜与葡萄柚的微苦结合，柠檬提升酸度，整体有清楚的柑橘层次。

## 情绪 / Mood
阳光、海岸、早午餐。它像一杯更成熟的混合果汁。

## 调酒师笔记 / Bartender Notes
- 葡萄柚偏甜时应减少糖浆。
- 双重过滤能去除果肉。
- 葡萄柚皮装饰比橙片更能强调微苦。
