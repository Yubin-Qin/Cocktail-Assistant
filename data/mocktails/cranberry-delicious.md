# Cranberry Delicious / 蔓越莓佳饮

## 故事 / Story
Cranberry 类无酒精混合饮长期出现在酒店和节庆菜单中。这个结构利用苹果补足蔓越莓的尖锐，让它更像完整鸡尾酒。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 75 ml | 蔓越莓汁 |
| 45 ml | 苹果汁 |
| 20 ml | 新鲜青柠汁 |
| 15 ml | 蜂蜜糖浆 |
| 60 ml | 姜汁汽水 |
|  | 苹果片装饰 |

## 步骤 / Steps
1. 将除气泡材料外的液体加入摇酒壶。
2. 加满冰块，用力摇和 8-12 秒。
3. 过滤入装有新鲜冰块的杯中。
4. 补入气泡材料，轻轻搅拌。
5. 完成装饰后立即饮用。

## 风味 / Flavor
蔓越莓酸涩、苹果圆润、蜂蜜柔和，姜汁汽水在尾段提供辛香。

## 情绪 / Mood
节日、明亮、适合秋冬聚会。

## 调酒师笔记 / Bartender Notes
- 蔓越莓汁有加糖与无糖之分，糖浆量要随之调整。
- 苹果汁最好选浑浊型，身体更厚。
- 姜汁汽水最后加入。
