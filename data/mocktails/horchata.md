# Horchata / 欧洽塔

## 故事 / Story
Horchata 在西班牙和拉丁美洲有多种版本。墨西哥常见版本以大米、肉桂和甜味制成，是最具代表性的无酒精传统饮品之一。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 g | 大米 |
| 300 ml | 水 |
| 1 根 | 肉桂棒 |
| 90 ml | 牛奶或燕麦奶 |
| 20 g | 糖 |
| 1-2 滴 | 香草精 |

## 步骤 / Steps
1. 按配方提前浸泡需要软化的谷物或果干。
2. 将浸泡物与其余材料放入搅拌机。
3. 搅打至顺滑，并按需要过滤。
4. 充分冰镇后倒入杯中，完成装饰。

## 风味 / Flavor
米香、肉桂、奶感和香草柔和，质地顺滑。

## 情绪 / Mood
舒适、甜点感、适合辛辣食物。

## 调酒师笔记 / Bartender Notes
- 大米需提前浸泡。
- 搅打后过滤。
- 甜度不要过高。
