# Aam Panna / 生芒果香料饮

## 故事 / Story
Aam Panna 是印度传统生芒果饮，常加入孜然、盐和糖。它以青芒果的酸度和果胶带来天然身体感。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 熟制青芒果泥 |
| 150 ml | 冷水 |
| 15 ml | 黑糖糖浆 |
| 1 小撮 | 烤孜然粉 |
| 1 小撮 | 黑盐 |
| 4 片 | 薄荷叶 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
青芒果酸、黑糖、孜然和盐形成浓郁南亚风味。

## 情绪 / Mood
解暑、朴实、带家常感。

## 调酒师笔记 / Bartender Notes
- 青芒果需煮软或烤熟。
- 糖量按芒果酸度调整。
- 冰镇后风味更平衡。
