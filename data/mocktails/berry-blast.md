# Berry Blast / 莓果风暴

## 故事 / Story
Berry Blast 是现代无酒精菜单中常见的命名和结构。关键并非“堆水果”，而是用茶单宁和柠檬把莓果从果汁感拉回鸡尾酒结构。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 4 颗 | 覆盆子 |
| 2 颗 | 草莓 |
| 60 ml | 蔓越莓汁 |
| 45 ml | 冷泡红茶 |
| 20 ml | 新鲜柠檬汁 |
| 15 ml | 百香果糖浆 |

## 步骤 / Steps
1. 在摇酒壶中轻压水果、香草或香料。
2. 加入其余液体材料。
3. 加冰用力摇和 10-12 秒。
4. 双重过滤入预冷杯或装有新鲜冰块的杯中。
5. 完成装饰。

## 风味 / Flavor
多种莓果的酸甜、茶单宁和百香果热带香叠加，颜色浓但不只有糖味。

## 情绪 / Mood
热闹、青春、适合聚会。

## 调酒师笔记 / Bartender Notes
- 莓果只需轻压。
- 红茶提供结构，不能省略后只剩甜果汁。
- 双重过滤更细腻。
