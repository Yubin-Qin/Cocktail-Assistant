# Zero Painkiller / 无酒精止痛药

## 故事 / Story
Painkiller 是加勒比风格经典，以朗姆、菠萝、橙子和椰子著称。零酒精版本几乎可以完整保留其热带识别。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 90 ml | 菠萝汁 |
| 30 ml | 橙汁 |
| 30 ml | 椰浆 |
|  | 0.0% 无酒精深色朗姆风味蒸馏液 45 ml |
|  | 肉豆蔻少许 |

## 步骤 / Steps
1. 将所有液体材料加入摇酒壶。
2. 加满冰块，用力摇和 10-12 秒。
3. 双重过滤入预冷鸡尾酒杯，或过滤入装有新鲜冰块的杯中。
4. 完成装饰后立即饮用。

## 风味 / Flavor
菠萝、橙子、椰子和深色香料柔滑厚实，肉豆蔻带来温暖尾韵。

## 情绪 / Mood
度假、柔软、适合慢下来。

## 调酒师笔记 / Bartender Notes
- 椰浆不要过多。
- 肉豆蔻现磨更香。
- 充分摇和以融合脂肪。
