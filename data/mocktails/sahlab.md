# Sahlab / 萨赫兰热饮

## 故事 / Story
Sahlab 是东地中海和中东地区传统热饮，现代家庭版本常用淀粉增稠，并以橙花水、肉桂和坚果调味。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 200 ml | 牛奶 |
| 8 g | 玉米淀粉 |
| 15 g | 糖 |
| 2 滴 | 橙花水 |
|  | 肉桂粉少许 |
|  | 碎开心果少许 |

## 步骤 / Steps
1. 将主要材料在小锅中混匀。
2. 小火加热并持续搅拌，直到略微浓稠。
3. 离火后加入花水或易挥发香气材料。
4. 倒入耐热杯，撒上装饰。

## 风味 / Flavor
热牛奶、花香、肉桂和坚果，质地浓稠。

## 情绪 / Mood
冬夜、街头、温暖。

## 调酒师笔记 / Bartender Notes
- 小火持续搅拌。
- 不要煮得过稠。
- 花水最后加入。
