# Zero Zombie / 无酒精僵尸

## 故事 / Story
Zombie 是 Tiki 文化最著名的强烈鸡尾酒之一。零酒精版本无法复制酒精力度，因此转而强调香料、酸度和层次。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精深色朗姆风味蒸馏液 45 ml |
| 45 ml | 菠萝汁 |
| 30 ml | 葡萄柚汁 |
| 20 ml | 新鲜青柠汁 |
| 15 ml | 肉桂糖浆 |
| 5 ml | 石榴糖浆 |

## 步骤 / Steps
1. 将所有液体材料加入摇酒壶。
2. 加满冰块，用力摇和 10-12 秒。
3. 双重过滤入预冷鸡尾酒杯，或过滤入装有新鲜冰块的杯中。
4. 完成装饰后立即饮用。

## 风味 / Flavor
多层热带果汁、肉桂、深色香料与酸度叠加，浓郁但不应只甜。

## 情绪 / Mood
夸张、热带、派对感强。

## 调酒师笔记 / Bartender Notes
- 肉桂糖浆只做背景。
- 必须有葡萄柚和青柠来限制甜度。
- 用碎冰和高装饰呈现 Tiki 感。
