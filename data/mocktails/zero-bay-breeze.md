# Zero Bay Breeze / 无酒精湾风

## 故事 / Story
Bay Breeze 与 Sea Breeze 属于同一高球家族，以菠萝替代葡萄柚。无酒精版本也保持这种简单可识别的结构。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 90 ml | 蔓越莓汁 |
| 60 ml | 菠萝汁 |
|  | 0.0% 无酒精伏特加风味蒸馏液 45 ml |
|  | 青柠角装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
蔓越莓酸涩与菠萝甜香结合，果味明显但仍有清楚对比。

## 情绪 / Mood
休闲、热带、容易接受。

## 调酒师笔记 / Bartender Notes
- 菠萝汁会增加甜度。
- 可加少量青柠提升酸度。
- 不要过度搅拌。
