# Zero Appletini / 无酒精苹果马天尼

## 故事 / Story
Appletini 是 1990 年代末到 2000 年代流行文化的重要鸡尾酒。零酒精版本保留鲜绿色苹果风格，但可用真实苹果汁降低糖果感。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 青苹果汁 |
|  | 0.0% 无酒精伏特加风味蒸馏液 45 ml |
| 20 ml | 新鲜柠檬汁 |
| 10 ml | 青苹果糖浆 |
|  | 苹果片装饰 |

## 步骤 / Steps
1. 将所有液体材料加入摇酒壶。
2. 加满冰块，用力摇和 10-12 秒。
3. 双重过滤入预冷鸡尾酒杯，或过滤入装有新鲜冰块的杯中。
4. 完成装饰后立即饮用。

## 风味 / Flavor
青苹果酸甜、干净基底和鲜明果香，口感脆爽。

## 情绪 / Mood
俏皮、复古、带 2000 年代夜生活感。

## 调酒师笔记 / Bartender Notes
- 糖浆只需少量。
- 苹果片可泡柠檬水防氧化。
- 成品应很冷。
