# Zero Negroni Sbagliato / 无酒精错误尼格罗尼

## 故事 / Story
Negroni Sbagliato 的原意是“做错的 Negroni”，以起泡酒代替琴酒。零酒精版本也成为近年来经典改写的重要方向。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精红色苦味开胃酒 30 ml |
|  | 0.0% 无酒精红味美思风格饮品 30 ml |
|  | 0.0% 无酒精干型起泡酒 60 ml |
|  | 橙片装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
苦橙、红果、草本和气泡结合，比 No-Groni 更轻、更活跃。

## 情绪 / Mood
成熟、闪亮、适合餐前。

## 调酒师笔记 / Bartender Notes
- 三种产品都要确认 0.0%。
- 使用偏干起泡酒。
- 大冰块能保持结构。
