# Jallab / 贾拉布

## 故事 / Story
Jallab 是黎凡特地区常见传统饮品，常以枣、葡萄糖浆、玫瑰水和坚果制成，在斋月期间尤其常见。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 20 ml | 枣糖浆 |
| 20 ml | 葡萄糖浆或葡萄汁浓缩液 |
| 2 滴 | 玫瑰水 |
| 150 ml | 冷水 |
| 1 茶匙 | 松子 |
|  | 葡萄干少许 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
枣、葡萄、玫瑰和坚果形成深色、甜润、带中东甜点气息的层次。

## 情绪 / Mood
夜晚、节庆、适合慢慢啜饮。

## 调酒师笔记 / Bartender Notes
- 糖浆浓度很高，先少量。
- 水要足够冰冷。
- 松子和葡萄干既是装饰也是口感元素。
