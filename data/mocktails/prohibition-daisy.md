# Prohibition Daisy / 禁酒时代雏菊

## 故事 / Story
Daisy 是经典鸡尾酒家族，通常由基酒、柑橘和甜味元素组成。无酒精版本保留这一结构，以植物蒸馏液和橙皮糖浆建立骨架。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精植物蒸馏液 45 ml |
| 25 ml | 新鲜柠檬汁 |
| 15 ml | 橙皮糖浆 |
| 5 ml | 石榴糖浆 |
| 60 ml | 苏打水 |
|  | 柠檬皮装饰 |

## 步骤 / Steps
1. 将除气泡材料外的液体加入摇酒壶。
2. 加满冰块，用力摇和 8-12 秒。
3. 过滤入装有新鲜冰块的杯中。
4. 补入气泡材料，轻轻搅拌。
5. 完成装饰后立即饮用。

## 风味 / Flavor
柠檬酸、橙皮甜香、草本和轻微红果构成明亮而完整的 Sour-Daisy 结构。

## 情绪 / Mood
复古、乐观、像禁酒时代海报里的一杯现代重构。

## 调酒师笔记 / Bartender Notes
- 橙皮糖浆比普通糖浆更有层次。
- 若无无酒精蒸馏液，可用浓冷泡白茶替代。
- 苏打水只需少量。
