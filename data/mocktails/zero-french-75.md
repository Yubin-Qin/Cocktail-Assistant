# Zero French 75 / 无酒精法式 75

## 故事 / Story
French 75 把酸酒基底和起泡酒结合。零酒精版本只要使用真正 0.0% 的干型起泡酒，就能保留原版的节庆节奏。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精杜松子风味蒸馏液 30 ml |
| 15 ml | 新鲜柠檬汁 |
| 15 ml | 单糖浆 |
|  | 0.0% 无酒精干型起泡酒 90 ml |
|  | 柠檬皮装饰 |

## 步骤 / Steps
1. 将除气泡材料外的液体加入摇酒壶。
2. 加满冰块，用力摇和 8-12 秒。
3. 过滤入装有新鲜冰块的杯中。
4. 补入气泡材料，轻轻搅拌。
5. 完成装饰后立即饮用。

## 风味 / Flavor
柠檬、杜松、细腻气泡和轻微烘焙感结合，轻盈而有庆祝感。

## 情绪 / Mood
优雅、明亮、适合举杯。

## 调酒师笔记 / Bartender Notes
- 起泡酒必须充分冰镇。
- 糖浆不要过多。
- 只摇基底，不要摇起泡酒。
