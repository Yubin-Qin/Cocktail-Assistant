# Arnold Palmer / 阿诺德帕尔默

## 故事 / Story
Arnold Palmer 以美国高尔夫球手 Arnold Palmer 命名。据说他常点冰茶和柠檬水混合的饮品，后来这个组合以他的名字流行起来。它是无酒精饮品中少见的“名人习惯变成经典配方”的例子。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 90 ml | 冰红茶 |
| 90 ml | 柠檬水 |
| 1 片 | 柠檬片 |
|  | 冰块适量 |

## 步骤 / Steps
1. 高球杯加满冰块。
2. 倒入冰红茶和柠檬水。
3. 轻轻搅拌。
4. 用柠檬片装饰。

## 风味 / Flavor
茶单宁、柠檬酸、轻甜和冰爽感。比纯柠檬水更成熟，比纯冰茶更活泼。

## 情绪 / Mood
阳光、运动、午后。它适合不紧不慢的白天，像坐在阴影里看草地。

## 调酒师笔记 / Bartender Notes
- 红茶要有足够茶感，过淡会像普通柠檬饮料。
- 柠檬水建议用新鲜柠檬汁、糖浆和水调制。
- 经典比例约为 1:1，也可提高茶比例，让成品更干爽。
