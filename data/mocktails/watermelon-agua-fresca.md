# Watermelon Agua Fresca / 西瓜鲜果水

## 故事 / Story
西瓜 Agua Fresca 是墨西哥夏季最经典的水果水之一。它依靠高含水量水果和少量酸度形成自然清爽感。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 180 g | 西瓜 |
| 20 ml | 新鲜青柠汁 |
| 90 ml | 冷水 |
| 10 ml | 单糖浆 |
|  | 薄荷叶装饰 |

## 步骤 / Steps
1. 将所有液体、果肉和冰块放入搅拌机。
2. 高速搅打至均匀顺滑。
3. 试味并调整酸甜。
4. 倒入冰冷杯中，完成装饰。

## 风味 / Flavor
西瓜清甜、青柠明亮，整体多汁轻盈。

## 情绪 / Mood
盛夏、轻松、适合大量制作。

## 调酒师笔记 / Bartender Notes
- 西瓜很甜时不必加糖。
- 现打现喝。
- 可过滤去除籽和粗纤维。
