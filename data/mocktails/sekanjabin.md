# Sekanjabin / 塞坎贾宾蜂蜜醋饮

## 故事 / Story
Sekanjabin 是波斯古老饮品，核心是蜂蜜与醋制成的糖浆，常配薄荷、黄瓜和水饮用。它展示了无酒精调饮远早于现代 mocktail 的历史。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 25 ml | 蜂蜜 |
| 15 ml | 苹果醋 |
| 150 ml | 冷水 |
| 8 片 | 薄荷叶 |
| 4 片 | 黄瓜片 |
|  | 碎冰适量 |

## 步骤 / Steps
1. 在杯中轻压水果或香草，只释放汁液和香气。
2. 加入其余非气泡材料并搅匀。
3. 加满冰块。
4. 补入气泡材料，轻轻上下搅拌。
5. 完成装饰。

## 风味 / Flavor
蜂蜜柔甜、醋酸明亮、薄荷和黄瓜清凉，酸甜中带古老草本感。

## 情绪 / Mood
清凉、历史感、适合炎热午后。

## 调酒师笔记 / Bartender Notes
- 醋应有果香而非刺鼻。
- 蜂蜜先与少量温水混匀。
- 黄瓜只轻压。
