# Pear Rosemary Collins / 梨与迷迭香柯林斯

## 故事 / Story
梨与迷迭香是现代季节性无酒精菜单的经典搭配，Collins 结构让两者不显厚重。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 梨汁 |
| 15 ml | 迷迭香糖浆 |
| 25 ml | 新鲜柠檬汁 |
| 120 ml | 苏打水 |
|  | 迷迭香枝装饰 |

## 步骤 / Steps
1. 将除气泡材料外的液体加入摇酒壶。
2. 加满冰块，用力摇和 8-12 秒。
3. 过滤入装有新鲜冰块的杯中。
4. 补入气泡材料，轻轻搅拌。
5. 完成装饰后立即饮用。

## 风味 / Flavor
梨子柔甜、迷迭香松针感和柠檬酸形成优雅长饮。

## 情绪 / Mood
秋天、安静、适合晚餐前。

## 调酒师笔记 / Bartender Notes
- 梨汁最好偏浑浊。
- 迷迭香糖浆少量。
- 苏打最后加入。
