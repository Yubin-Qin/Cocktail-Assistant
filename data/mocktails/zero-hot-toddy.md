# Zero Hot Toddy / 无酒精热托迪

## 故事 / Story
Hot Toddy 是历史悠久的热饮结构。零酒精版本保留热水、甜味、柑橘和香料，只去掉酒精。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精威士忌风味蒸馏液 45 ml |
| 120 ml | 热水 |
| 20 ml | 蜂蜜 |
| 15 ml | 新鲜柠檬汁 |
| 1 根 | 肉桂棒 |

## 步骤 / Steps
1. 将除新鲜柑橘汁外的材料放入小锅。
2. 小火加热至冒热气，不要沸腾。
3. 离火后加入新鲜柑橘汁并试味。
4. 倒入耐热杯，完成装饰。

## 风味 / Flavor
蜂蜜、柠檬、温热香料和木香柔和展开。

## 情绪 / Mood
安静、温暖、适合寒冷夜晚。

## 调酒师笔记 / Bartender Notes
- 水温不必沸腾。
- 柠檬汁最后加入更鲜明。
- 蜂蜜量按口味微调。
