# Zero New York Sour / 无酒精纽约酸

## 故事 / Story
New York Sour 的标志是 Whiskey Sour 上浮一层红酒。零酒精版本使用 0.0% 葡萄酒替代饮品保留这一视觉与单宁结构。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精威士忌风味蒸馏液 50 ml |
| 25 ml | 新鲜柠檬汁 |
| 20 ml | 黑糖糖浆 |
| 15 ml | aquafaba |
|  | 0.0% 红葡萄酒替代饮品 20 ml |

## 步骤 / Steps
1. 将除浮顶材料外的其余材料加入摇酒壶，先不加冰干摇 10 秒。
2. 加入冰块，再用力摇和 10-12 秒。
3. 过滤入杯中。
4. 沿吧勺背缓慢加入浮顶材料，形成分层。
5. 完成装饰。

## 风味 / Flavor
下层酸甜木香，上层红葡萄与单宁形成视觉和味觉分层。

## 情绪 / Mood
戏剧性、都市、适合夜晚。

## 调酒师笔记 / Bartender Notes
- 上层液体要沿吧勺背缓慢倒入。
- 红葡萄替代饮品应偏干。
- 泡沫不要过厚。
