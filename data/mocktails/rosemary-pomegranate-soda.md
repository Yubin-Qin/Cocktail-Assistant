# Rosemary Pomegranate Soda / 迷迭香石榴苏打

## 故事 / Story
石榴与迷迭香是现代无酒精菜单常见搭配，前者提供颜色和单宁，后者提供明显植物香。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 75 ml | 石榴汁 |
| 20 ml | 迷迭香糖浆 |
| 15 ml | 新鲜柠檬汁 |
| 120 ml | 苏打水 |
|  | 迷迭香枝装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
石榴酸涩、松针般迷迭香和柠檬结合，气泡干净。

## 情绪 / Mood
冬季、优雅、适合节庆。

## 调酒师笔记 / Bartender Notes
- 迷迭香糖浆不要煮太久。
- 石榴汁最好无额外糖。
- 轻拍迷迭香释放香气。
