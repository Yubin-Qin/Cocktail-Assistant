# Zero Naked and Famous / 无酒精赤裸与名声

## 故事 / Story
Naked and Famous 是受 Last Word 启发的现代经典。零酒精版本保留四等分结构，并用烟熏植物元素替代 mezcal 的烟感。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精烟熏植物蒸馏液 30 ml |
|  | 0.0% 无酒精橙味开胃酒 30 ml |
| 30 ml | 草本糖浆 |
| 30 ml | 新鲜青柠汁 |

## 步骤 / Steps
1. 将所有液体材料加入摇酒壶。
2. 加满冰块，用力摇和 10-12 秒。
3. 双重过滤入预冷鸡尾酒杯，或过滤入装有新鲜冰块的杯中。
4. 完成装饰后立即饮用。

## 风味 / Flavor
烟熏、橙皮、草本与青柠形成鲜明现代感。

## 情绪 / Mood
前卫、明亮、带一点挑衅。

## 调酒师笔记 / Bartender Notes
- 烟熏基底不要像液体烟过重。
- 草本糖浆应有苦味。
- 严格控制甜度。
