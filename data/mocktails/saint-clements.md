# Saint Clements / 圣克莱门特

## 故事 / Story
Saint Clements 是英国常见的无酒精组合，名称常被解释为来自童谣中的“oranges and lemons”。它用橙和柠檬两种柑橘建立极简结构。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 90 ml | 橙汁 |
| 90 ml | 苦柠檬汽水 |
| 10 ml | 新鲜柠檬汁 |
|  | 橙片装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
橙汁的甜熟果香与苦柠檬的柑橘皮苦味相遇，整体比普通橙汁汽水更成人、更干爽。

## 情绪 / Mood
英式、轻松、适合白天。它像午餐后散步时的一杯简单长饮。

## 调酒师笔记 / Bartender Notes
- 苦柠檬汽水要充分冰镇。
- 若汽水本身很甜，可提高新鲜柠檬汁比例。
- 使用大块冰能减慢稀释。
