# Zero Ranch Water / 无酒精牧场水

## 故事 / Story
Ranch Water 与美国得州饮酒文化关系密切，经典结构极简。无酒精版本同样依赖高质量气泡水和足够新鲜的青柠。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精龙舌兰风味蒸馏液 50 ml |
| 20 ml | 新鲜青柠汁 |
| 150 ml | 高气泡矿泉水 |
| 1 小撮 | 盐 |
|  | 青柠角装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
极干爽，青柠与矿物气泡突出，草本底味轻而清楚。

## 情绪 / Mood
简洁、户外、适合炎热天气。

## 调酒师笔记 / Bartender Notes
- 矿泉水气泡越强越好。
- 不要加糖，否则会偏离 Ranch Water 的干爽个性。
- 杯子和材料都应充分冰镇。
