# Zero Bramble / 无酒精荆棘

## 故事 / Story
Bramble 是 1980 年代伦敦现代经典。零酒精版本保留杜松、柠檬和黑莓在碎冰上的标志性结构。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精杜松子风味蒸馏液 45 ml |
| 25 ml | 新鲜柠檬汁 |
| 15 ml | 单糖浆 |
| 20 ml | 黑莓糖浆 |
|  | 碎冰适量 |
|  | 黑莓装饰 |

## 步骤 / Steps
1. 将除浮顶材料外的其余材料加入摇酒壶。
2. 加冰摇和 10-12 秒。
3. 过滤入杯中。
4. 将最后的风味材料缓慢浮在表面。
5. 完成装饰。

## 风味 / Flavor
柠檬、杜松和黑莓酸甜形成层次，碎冰让红色糖浆缓慢渗透。

## 情绪 / Mood
英式、果园、带一点浪漫。

## 调酒师笔记 / Bartender Notes
- 黑莓糖浆最后淋入。
- 碎冰要装满。
- 糖浆不宜过甜。
