# Strawberry Basil Smash / 草莓罗勒碎冰饮

## 故事 / Story
Smash 是以鲜果和香草为核心的自由型结构。无酒精版本尤其适合展示季节水果。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 4 颗 | 草莓 |
| 6 片 | 罗勒叶 |
| 25 ml | 新鲜柠檬汁 |
| 15 ml | 蜂蜜糖浆 |
| 90 ml | 苏打水 |

## 步骤 / Steps
1. 在摇酒壶中轻压水果或香草。
2. 加入除气泡材料外的其余材料和冰块。
3. 摇和 8-10 秒并双重过滤入装有新鲜冰块的杯中。
4. 补入气泡材料并轻搅。
5. 完成装饰。

## 风味 / Flavor
草莓甜酸、罗勒辛香、蜂蜜和柠檬，果香但不幼稚。

## 情绪 / Mood
春夏、花园、活泼。

## 调酒师笔记 / Bartender Notes
- 罗勒不要捣烂。
- 草莓成熟时减少蜂蜜。
- 双重过滤口感更细。
