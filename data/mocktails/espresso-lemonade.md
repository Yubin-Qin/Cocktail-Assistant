# Espresso Lemonade / 浓缩咖啡柠檬水

## 故事 / Story
咖啡与柠檬的组合在多地咖啡馆文化中不断出现。Espresso Lemonade 用酸甜结构把咖啡从热饮语境中拉出。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 30 ml | 新鲜浓缩咖啡 |
| 25 ml | 新鲜柠檬汁 |
| 20 ml | 单糖浆 |
| 90 ml | 冷水 |
| 60 ml | 苏打水 |

## 步骤 / Steps
1. 将除气泡材料外的液体加入摇酒壶。
2. 加满冰块，用力摇和 8-12 秒。
3. 过滤入装有新鲜冰块的杯中。
4. 补入气泡材料，轻轻搅拌。
5. 完成装饰后立即饮用。

## 风味 / Flavor
柠檬酸、咖啡烘焙和糖形成大胆对比，气泡让收尾更干。

## 情绪 / Mood
醒神、实验感、适合炎热天气。

## 调酒师笔记 / Bartender Notes
- 咖啡要明亮果酸型更合适。
- 先摇基底再补气泡。
- 糖浆量必须跟着咖啡烘焙度调整。
