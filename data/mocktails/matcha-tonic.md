# Matcha Tonic / 抹茶汤力

## 故事 / Story
Matcha Tonic 是现代咖啡馆和无酒精菜单中常见的跨界组合。抹茶天然提供苦味、涩感和颜色。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 2 g | 抹茶 |
| 40 ml | 冷水 |
| 120 ml | 汤力水 |
| 10 ml | 柚子汁 |
|  | 柠檬皮装饰 |

## 步骤 / Steps
1. 先在杯中加入气泡或较轻的基底材料和冰块。
2. 将较深色或较浓的材料沿吧勺背缓慢倒入。
3. 保留分层，或饮用前轻搅。
4. 完成装饰。

## 风味 / Flavor
抹茶青草、轻涩、奎宁苦和柑橘香结合。

## 情绪 / Mood
清醒、日系、极简。

## 调酒师笔记 / Bartender Notes
- 抹茶要先完全打匀。
- 汤力水偏甜时加入少量柚子汁。
- 缓慢分层可保留视觉。
