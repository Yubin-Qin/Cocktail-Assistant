# Cherry Lime Rickey / 樱桃青柠里奇

## 故事 / Story
Cherry Lime Rickey 是美式汽水柜台对 Lime Rickey 的经典延伸。加入樱桃糖浆后，它成为许多地区夏季菜单上的标志性红色长饮。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 25 ml | 新鲜青柠汁 |
| 20 ml | 樱桃糖浆 |
| 150 ml | 苏打水 |
| 2 颗 | 樱桃 |
|  | 青柠轮装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
樱桃红果甜香与青柠酸度形成鲜明对比，气泡让颜色热闹但口感不至于厚重。

## 情绪 / Mood
复古、活泼、像夏季游乐场和老式餐厅。它有明显的童年感，但酸度让它不幼稚。

## 调酒师笔记 / Bartender Notes
- 樱桃糖浆最好选择真实果味而非纯香精。
- 提高青柠比例可以防止整体过甜。
- 最后轻搅即可，保留气泡。
