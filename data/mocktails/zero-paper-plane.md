# Zero Paper Plane / 无酒精纸飞机

## 故事 / Story
Paper Plane 是 2000 年代的重要现代经典，同样采用四等分结构。零酒精版本依靠无酒精开胃酒产品重建其酸苦平衡。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精威士忌风味蒸馏液 30 ml |
|  | 0.0% 无酒精苦橙开胃酒 30 ml |
|  | 0.0% 无酒精草本开胃酒 30 ml |
| 30 ml | 新鲜柠檬汁 |

## 步骤 / Steps
1. 将所有液体材料加入摇酒壶。
2. 加满冰块，用力摇和 10-12 秒。
3. 双重过滤入预冷鸡尾酒杯，或过滤入装有新鲜冰块的杯中。
4. 完成装饰后立即饮用。

## 风味 / Flavor
苦橙、草本、柠檬和木香平衡，酸苦甜同时存在。

## 情绪 / Mood
现代、利落、适合夜晚。

## 调酒师笔记 / Bartender Notes
- 各产品甜度差异大。
- 柠檬必须新鲜。
- 成品应偏干。
