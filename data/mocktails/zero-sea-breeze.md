# Zero Sea Breeze / 无酒精海风

## 故事 / Story
Sea Breeze 以伏特加、蔓越莓和葡萄柚构成。零酒精版本的主要个性仍来自两种果汁。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 90 ml | 蔓越莓汁 |
| 60 ml | 葡萄柚汁 |
|  | 0.0% 无酒精伏特加风味蒸馏液 45 ml |
|  | 青柠角装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
蔓越莓酸涩、葡萄柚微苦，基底干净。

## 情绪 / Mood
海岸、轻松、90 年代复古。

## 调酒师笔记 / Bartender Notes
- 选择不太甜的蔓越莓汁。
- 葡萄柚现榨更有层次。
- 大量冰块能保持清爽。
