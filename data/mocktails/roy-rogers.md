# Roy Rogers / 罗伊罗杰斯

## 故事 / Story
Roy Rogers 以美国西部片明星和歌手 Roy Rogers 命名，常被看作 Shirley Temple 的可乐版本。它延续了以明星命名无酒精长饮的传统，也让简单汽水拥有了酒吧式仪式感。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 150 ml | 可乐 |
| 15 ml | 石榴糖浆 |
| 5 ml | 青柠汁 |
| 1 颗 | 樱桃 |

## 步骤 / Steps
1. 高球杯加冰。
2. 倒入石榴糖浆和青柠汁。
3. 加入冰镇可乐，轻搅。
4. 以樱桃装饰。

## 风味 / Flavor
可乐香料、香草、红果糖浆、气泡和轻微酸味。比 Shirley Temple 更深色、更像晚间饮品。

## 情绪 / Mood
复古、轻松、适合家庭餐厅和电影夜。它有一种不装严肃的快乐。

## 调酒师笔记 / Bartender Notes
- 可乐本身很甜，石榴糖浆只需少量。
- 青柠汁能切开甜味，让尾段更清爽。
- 可加入一小撮盐提升可乐香料感。
