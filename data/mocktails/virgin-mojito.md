# Virgin Mojito / 无酒精莫吉托

## 故事 / Story
Virgin Mojito 来自经典 Mojito 的无酒精改写。它保留了古巴长饮的关键元素：薄荷、青柠、糖和气泡，只去掉朗姆。由于结构清楚、接受度高，它常是现代 mocktail 菜单里的基础款。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 25 ml | 新鲜青柠汁 |
| 20 ml | 单糖浆 |
| 10-12 片 | 新鲜薄荷叶 |
| 120 ml | 苏打水 |
|  | 碎冰适量 |
|  | 青柠角和薄荷枝装饰 |

## 步骤 / Steps
1. 高球杯中放入薄荷叶、青柠汁和糖浆。
2. 轻压薄荷释放香气。
3. 加入碎冰，倒入苏打水。
4. 轻轻上下搅拌，让薄荷分布均匀。
5. 加满碎冰，用薄荷枝和青柠角装饰。

## 风味 / Flavor
薄荷清凉、青柠明亮、轻甜、气泡干净。无酒精版本更突出植物香和清爽感。

## 情绪 / Mood
清新、松弛、适合白天。它的气质是“刚洗过的空气”，能把炎热或疲惫感迅速降下来。

## 调酒师笔记 / Bartender Notes
- 薄荷不要捣烂，否则会有草腥和苦味。
- 可以加入几片黄瓜或少量苹果汁增加厚度。
- 苏打水必须冰镇，气泡越足越有鸡尾酒感。
