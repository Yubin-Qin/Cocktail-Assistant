# Mint-Basil Limeade / 薄荷罗勒青柠水

## 故事 / Story
Mint-Basil Limeade 是现代酒吧常见的 herb-forward 无酒精结构，用两种香草建立比普通柠檬水更成熟的香气。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 30 ml | 新鲜青柠汁 |
| 6 片 | 薄荷叶 |
| 6 片 | 罗勒叶 |
| 20 ml | 单糖浆 |
| 150 ml | 苏打水 |

## 步骤 / Steps
1. 在杯中轻压水果或香草，只释放汁液和香气。
2. 加入其余非气泡材料并搅匀。
3. 加满冰块。
4. 补入气泡材料，轻轻上下搅拌。
5. 完成装饰。

## 风味 / Flavor
薄荷清凉、罗勒辛香、青柠明亮，草本层次比普通 Mojito 更复杂。

## 情绪 / Mood
花园、夏日、醒神。

## 调酒师笔记 / Bartender Notes
- 两种香草都只轻压。
- 罗勒容易氧化。
- 现做现喝。
