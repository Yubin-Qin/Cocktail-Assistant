# Zero Sangria / 无酒精桑格利亚

## 故事 / Story
Sangria 是西班牙和伊比利亚饮食文化中最知名的分享型饮品之一。零酒精版本用 0.0% 葡萄酒替代饮品保留果皮与单宁感。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精红葡萄酒替代饮品 500 ml |
| 120 ml | 橙汁 |
| 90 ml | 石榴汁 |
|  | 橙片、苹果块和莓果适量 |
| 1 根 | 肉桂棒 |
| 250 ml | 苏打水 |

## 步骤 / Steps
1. 将除气泡材料外的所有液体和水果加入大壶或潘趣碗。
2. 冷藏静置，让水果与香料短暂融合。
3. 饮用前加入大量冰块。
4. 最后补入气泡材料并轻轻搅拌。
5. 分杯饮用。

## 风味 / Flavor
红葡萄、柑橘、苹果、莓果和肉桂层层融合，气泡让大份饮品更轻。

## 情绪 / Mood
分享、夏日、聚会。

## 调酒师笔记 / Bartender Notes
- 水果浸泡 1-2 小时即可。
- 气泡饮用前加入。
- 确认葡萄酒替代饮品为 0.0%。
