# Zero Irish Coffee / 无酒精爱尔兰咖啡

## 故事 / Story
Irish Coffee 的标志是热咖啡、威士忌、糖与浮顶奶油。零酒精版本保留最重要的温度分层。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 120 ml | 热咖啡 |
|  | 0.0% 无酒精威士忌风味蒸馏液 30 ml |
| 15 ml | 黑糖糖浆 |
| 30 ml | 轻打发奶油 |

## 步骤 / Steps
1. 将热咖啡、甜味和无酒精基底加入耐热杯并搅匀。
2. 将奶油轻轻打至可流动的浓稠状态。
3. 沿吧勺背缓慢将奶油浮在表面。
4. 不再搅拌，直接从奶油下方饮用。

## 风味 / Flavor
热咖啡、黑糖、木香与冷奶油形成温度和质地对比。

## 情绪 / Mood
夜晚、舒适、餐后。

## 调酒师笔记 / Bartender Notes
- 奶油只打到可流动。
- 从奶油下方饮用。
- 咖啡不要过度烘焙发苦。
