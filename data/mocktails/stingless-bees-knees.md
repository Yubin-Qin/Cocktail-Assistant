# Stingless Bee's Knees / 无刺蜜蜂之膝

## 故事 / Story
Bee’s Knees 是禁酒时代著名琴酒酸酒。无酒精版本用植物蒸馏液保留原有香气逻辑，名称中的 Stingless 则暗示它没有酒精“蜂刺”。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精杜松子风味蒸馏液 50 ml |
| 25 ml | 新鲜柠檬汁 |
| 20 ml | 蜂蜜糖浆 |
|  | 柠檬皮装饰 |

## 步骤 / Steps
1. 将所有液体材料加入摇酒壶。
2. 加满冰块，用力摇和 10-12 秒。
3. 双重过滤入预冷鸡尾酒杯，或过滤入装有新鲜冰块的杯中。
4. 完成装饰后立即饮用。

## 风味 / Flavor
蜂蜜花香、柠檬酸和杜松草本形成柔和但有张力的 Sour。

## 情绪 / Mood
温暖、友好、略带复古。

## 调酒师笔记 / Bartender Notes
- 蜂蜜需先稀释成糖浆。
- 花香较强的蜂蜜用量要更少。
- 柠檬皮能把香气提亮。
