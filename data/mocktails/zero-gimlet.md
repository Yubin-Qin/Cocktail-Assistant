# Zero Gimlet / 无酒精吉姆雷特

## 故事 / Story
Gimlet 的经典识别来自琴酒和青柠 cordial。零酒精版本保留这种高度集中的青柠风格，比普通 Limeade 更像一杯短饮。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
|  | 0.0% 无酒精杜松子风味蒸馏液 60 ml |
| 20 ml | 新鲜青柠汁 |
| 15 ml | 青柠 cordial |
|  | 青柠皮装饰 |

## 步骤 / Steps
1. 将所有液体材料加入摇酒壶。
2. 加满冰块，用力摇和 10-12 秒。
3. 双重过滤入预冷鸡尾酒杯，或过滤入装有新鲜冰块的杯中。
4. 完成装饰后立即饮用。

## 风味 / Flavor
青柠皮、酸度和杜松子非常集中，风味短促而清晰。

## 情绪 / Mood
敏捷、干净、极简。

## 调酒师笔记 / Bartender Notes
- cordial 不宜过甜。
- 无酒精基酒要有明显植物感。
- 成品应冰冷且少水感。
