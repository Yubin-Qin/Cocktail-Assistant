# Shirley Temple / 雪莉登波

## 故事 / Story
Shirley Temple 以美国童星 Shirley Temple 命名，通常被认为诞生于 20 世纪上半叶的好莱坞餐厅或酒店文化。它的意义在于让儿童或不饮酒者也能拥有一杯“像鸡尾酒一样被认真装饰”的饮品。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 150 ml | 姜汁汽水或柠檬青柠汽水 |
| 15-20 ml | 石榴糖浆 |
| 5-10 ml | 新鲜青柠汁 |
| 1 颗 | 酒渍樱桃或糖渍樱桃 |
| 1 片 | 橙片或柠檬片 |

## 步骤 / Steps
1. 高球杯中加满冰块。
2. 倒入石榴糖浆和少量青柠汁。
3. 加入冰镇汽水，轻轻搅拌。
4. 用樱桃和柑橘片装饰。

## 风味 / Flavor
甜美、红果、气泡、轻微柑橘酸。若用姜汁汽水，会有温和姜辣收尾。

## 情绪 / Mood
明亮、无忧、庆祝感强。它像聚会里给不饮酒者的一杯正式邀请，而不是退而求其次。

## 调酒师笔记 / Bartender Notes
- 加少量青柠汁能让甜味更立体，避免像普通糖水。
- 石榴糖浆用量不要太多，否则颜色漂亮但味道厚重。
- 姜汁汽水版本更有辛香，柠檬青柠汽水版本更童趣。
