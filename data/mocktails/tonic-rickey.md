# Tonic Rickey / 奎宁里奇

## 故事 / Story
Tonic Rickey 是 Rickey 结构的现代无酒精改写。它用汤力水的奎宁苦味替代烈酒带来的抓力，是当代零酒精菜单中常见的极简做法。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 25 ml | 新鲜青柠汁 |
| 10 ml | 单糖浆 |
| 90 ml | 汤力水 |
| 60 ml | 苏打水 |
| 2 dashes | 无酒精芳香苦精 |
|  | 青柠轮装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
青柠酸、奎宁苦、轻甜和强气泡构成干爽骨架，尾韵比普通 Lime Rickey 更成熟。

## 情绪 / Mood
克制、清凉、适合餐前。它有鸡尾酒的苦味节奏，却保持很轻的身体。

## 调酒师笔记 / Bartender Notes
- 无酒精苦精要确认标签为 0.0% ABV。
- 汤力水与苏打水混合能避免奎宁过甜。
- 不要过度搅拌。
