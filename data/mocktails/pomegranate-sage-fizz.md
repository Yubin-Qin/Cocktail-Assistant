# Pomegranate Sage Fizz / 石榴鼠尾草菲士

## 故事 / Story
鼠尾草比薄荷更温暖，和石榴的深色果味非常合适。Fizz 技法能把季节性材料做得更轻。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 75 ml | 石榴汁 |
| 15 ml | 鼠尾草糖浆 |
| 20 ml | 新鲜柠檬汁 |
| 15 ml | aquafaba |
| 90 ml | 苏打水 |

## 步骤 / Steps
1. 将除气泡材料外的材料加入摇酒壶，先不加冰干摇 10 秒。
2. 加冰再摇和 8-10 秒。
3. 过滤入装有新鲜冰块的杯中。
4. 补入气泡材料，让泡沫自然升起。
5. 完成装饰。

## 风味 / Flavor
石榴酸涩、鼠尾草草本和柠檬明亮，泡沫增加质地。

## 情绪 / Mood
秋冬、成熟、略带神秘。

## 调酒师笔记 / Bartender Notes
- 鼠尾草糖浆浸泡时间不要太长。
- 使用 aquafaba 时先干摇。
- 苏打最后补入。
