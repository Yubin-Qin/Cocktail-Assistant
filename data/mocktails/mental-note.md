# Mental Note / 心中一记

## 故事 / Story
现代无酒精调酒常使用根茎蔬菜和果酱增加质地。Mental Note 体现了脱离经典鸡尾酒模板的自由创作。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 甜菜汁 |
| 45 ml | 葡萄柚汁 |
| 15 ml | 新鲜柠檬汁 |
| 15 ml | 香草糖浆 |
| 1 茶匙 | 覆盆子果酱 |

## 步骤 / Steps
1. 将所有液体材料加入摇酒壶。
2. 加满冰块，用力摇和 10-12 秒。
3. 双重过滤入预冷鸡尾酒杯，或过滤入装有新鲜冰块的杯中。
4. 完成装饰后立即饮用。

## 风味 / Flavor
甜菜土壤感、葡萄柚苦、柠檬酸和莓果果酱形成非常独特的层次。

## 情绪 / Mood
实验性、安静、适合喜欢非传统风味的人。

## 调酒师笔记 / Bartender Notes
- 甜菜汁要新鲜。
- 果酱必须充分摇散。
- 双重过滤。
