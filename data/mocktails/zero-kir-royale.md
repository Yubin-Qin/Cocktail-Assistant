# Zero Kir Royale / 无酒精皇家基尔

## 故事 / Story
Kir Royale 是黑加仑利口酒与香槟的经典组合。零酒精版本用优质黑加仑糖浆保留色泽和香气。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 15 ml | 黑加仑糖浆 |
|  | 0.0% 无酒精干型起泡酒 120 ml |
|  | 柠檬皮装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
黑加仑浓郁果香被干型气泡拉长，颜色深而口感轻。

## 情绪 / Mood
正式、精致、适合晚宴开场。

## 调酒师笔记 / Bartender Notes
- 黑加仑糖浆很强，只需少量。
- 起泡酒越干越平衡。
- 倒酒时缓慢避免溢泡。
