# Apple Cider Mule / 苹果西打骡子

## 故事 / Story
Apple Cider Mule 是 Moscow Mule 的季节性无酒精延伸，用苹果替代部分基酒存在感。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 90 ml | 无酒精苹果西打 |
| 15 ml | 新鲜青柠汁 |
| 90 ml | 姜汁啤酒 |
| 1 根 | 肉桂棒 |
|  | 苹果片装饰 |

## 步骤 / Steps
1. 在杯中加入非气泡材料。
2. 加满冰块并搅拌均匀。
3. 缓慢加入气泡材料。
4. 轻轻搅拌 1-2 次，完成装饰。

## 风味 / Flavor
苹果、姜、青柠和肉桂，清爽中带秋季香料。

## 情绪 / Mood
秋日、户外、舒适。

## 调酒师笔记 / Bartender Notes
- 苹果西打需确认无酒精。
- 姜汁啤酒要有辛辣。
- 肉桂只作香气。
