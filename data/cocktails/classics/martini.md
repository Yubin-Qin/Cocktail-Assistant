# Martini / 马天尼

## 故事 / Story
Martini 的源流常被追溯到 19 世纪末的 Martinez、Marguerite 等早期琴酒和味美思混合酒。进入 20 世纪后，干型琴酒和干味美思逐渐成为主流，马天尼也从偏甜的古典配方演变成现代人熟悉的冷峻形象。它的经典地位来自极少材料带来的巨大差异：酒款、比例、温度、稀释度和装饰都会让成品呈现完全不同的性格。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 琴酒 |
| 10-15 ml | 干味美思 |
| 1 dash | 橙味苦精 |
| 1 份 | 柠檬皮或橄榄 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 在调酒杯中加入琴酒、干味美思和苦精。
3. 加满冰块，用吧勺搅拌约 25-35 秒，直到杯壁明显结霜。
4. 过滤入预冷鸡尾酒杯。
5. 用柠檬皮在杯口挤出精油后投入，或以橄榄装饰。

## 风味 / Flavor
清冽、干爽、草本、杜松子香突出。柠檬皮版本更明亮，橄榄版本更咸鲜、更成熟。

## 情绪 / Mood
克制、锋利、极简，像把复杂情绪压成一句干净的判断。适合安静的夜晚、正式开场或需要一点冷静仪式感的时候。

## 调酒师笔记 / Bartender Notes
- 喜欢干爽可用 6:1 到 8:1 的琴酒和味美思比例；喜欢更圆润可用 4:1。
- 不建议完全省略味美思，它提供草本、葡萄酒酸度和结构。
- 搅拌要足够冷，但不要搅到水感明显。
