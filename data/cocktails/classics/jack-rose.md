# Jack Rose / 杰克玫瑰

## 故事 / Story
Jack Rose 在 20 世纪初美国流行，并出现在 Hemingway 等作家的文学世界中，是 Applejack 文化的代表鸡尾酒。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 苹果白兰地 |
| 22.5 ml | 新鲜柠檬汁 |
| 15 ml | 石榴糖浆 |
| 1 片 | 柠檬皮 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 将除装饰外的所有材料加入摇酒壶。
3. 加满冰块，用力摇和约 10-12 秒。
4. 双重过滤入预冷鸡尾酒杯。
5. 以柠檬皮装饰。

## 风味 / Flavor
苹果、石榴、柠檬和木桶，酸甜清楚。

## 情绪 / Mood
文学、禁酒时代前后、带一点硬朗浪漫。

## 调酒师笔记 / Bartender Notes
- 使用真正苹果白兰地，能带来果核和木桶深度。
- 石榴糖浆负责颜色和甜度，不宜过量。
