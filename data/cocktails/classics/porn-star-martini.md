# Porn Star Martini / 色情明星马天尼

## 故事 / Story
Porn Star Martini 由伦敦调酒师 Douglas Ankrah 在 2000 年代初创作，名称和独特的起泡酒 sidecar 让它迅速成为全球热门。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 50 ml | 香草伏特加 |
| 20 ml | 百香果利口酒 |
| 30 ml | 百香果泥 |
| 15 ml | 新鲜青柠汁 |
| 10 ml | 香草糖浆 |
| 50 ml，另杯 | 香槟或普罗塞克 |

## 步骤 / Steps
1. 将主酒材料加入摇酒壶，加冰用力摇和 10-12 秒。
2. 双重过滤入预冷鸡尾酒杯。
3. 另以小杯倒入冰镇起泡酒。
4. 以半颗百香果和另杯起泡酒装饰。

## 风味 / Flavor
百香果、香草、青柠和气泡酒，甜酸浓烈。

## 情绪 / Mood
外向、性感、夜店感强，是典型的高人气现代经典。

## 调酒师笔记 / Bartender Notes
- 起泡酒通常另杯提供，不要直接倒进主酒。
- 百香果酸度差异大，糖浆需按果泥调整。
