# Gin Fizz / 金菲士

## 故事 / Story
Fizz 家族在 19 世纪末已极其流行，Gin Fizz 是其中最基础也最持久的版本。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 琴酒 |
| 30 ml | 新鲜柠檬汁 |
| 15 ml | 单糖浆 |
| 60 ml | 苏打水 |
| 1 片 | 柠檬片 |

## 步骤 / Steps
1. 将除苏打水外的材料加冰摇和。
2. 过滤入冰镇高球杯。
3. 补入冰镇苏打水。
4. 轻轻搅拌一次。
5. 以柠檬片装饰。

## 风味 / Flavor
柠檬、杜松子、细腻气泡和轻甜。

## 情绪 / Mood
清爽、醒神、具有经典酒吧早餐般的明快感。

## 调酒师笔记 / Bartender Notes
- 酸酒基底先摇冷，再补苏打水，泡沫会更细。
- 与 Tom Collins 相似，但通常更短、更轻盈。
