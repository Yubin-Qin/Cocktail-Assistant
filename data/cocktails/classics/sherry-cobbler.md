# Sherry Cobbler / 雪莉考伯勒

## 故事 / Story
Sherry Cobbler 在 19 世纪风靡欧美，也推动了吸管和碎冰饮法的流行，是历史上最重要的长饮之一。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 90 ml | 阿蒙蒂亚多或奥罗罗索雪莉酒 |
| 10 ml | 单糖浆 |
| 2 片 | 橙片 |
|  | 碎冰 适量 |
| 1 份 | 时令莓果和薄荷 |

## 步骤 / Steps
1. 将雪莉酒、糖浆和橙片加入摇酒壶短摇。
2. 连同液体过滤入装满碎冰的高脚杯。
3. 补满碎冰并丰富装饰。
4. 以莓果和薄荷装饰。

## 风味 / Flavor
坚果、橙子、干果、冰凉稀释和新鲜水果。

## 情绪 / Mood
19 世纪、夏日、像坐在阳台上慢慢用吸管喝酒。

## 调酒师笔记 / Bartender Notes
- 大量碎冰和丰富水果装饰是 Cobbler 的核心体验。
- 雪莉酒本身复杂，糖只需少量。
