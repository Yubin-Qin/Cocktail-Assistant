# Blood and Sand / 血与沙

## 故事 / Story
Blood and Sand 以 1922 年同名电影命名，是少数以苏格兰威士忌为基酒的经典酸甜鸡尾酒。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 22.5 ml | 苏格兰威士忌 |
| 22.5 ml | 甜味美思 |
| 22.5 ml | 樱桃利口酒 |
| 22.5 ml | 橙汁 |
| 1 片 | 橙皮 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 将除装饰外的所有材料加入摇酒壶。
3. 加满冰块，用力摇和约 10-12 秒。
4. 双重过滤入预冷鸡尾酒杯。
5. 以橙皮装饰。

## 风味 / Flavor
橙子、樱桃、麦芽、红果和柔和烟熏。

## 情绪 / Mood
电影、戏剧、红金色调的旧时代浪漫。

## 调酒师笔记 / Bartender Notes
- 等量配方偏甜，可选更干的甜味美思和新鲜橙汁。
- 苏格兰威士忌不宜泥煤过重。
