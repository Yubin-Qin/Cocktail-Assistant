# Toronto / 多伦多

## 故事 / Story
Toronto 在 20 世纪早期的配方书中出现，以加拿大城市命名，是 Fernet 与威士忌结合的古典代表。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 黑麦威士忌 |
| 7.5 ml | Fernet-Branca |
| 7.5 ml | 单糖浆 |
| 2 dashes | 安哥斯图拉苦精 |
| 1 片 | 橙皮 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 把除装饰外的材料加入调酒杯并加满冰块。
3. 用吧勺搅拌约 25-35 秒。
4. 过滤入预冷鸡尾酒杯。
5. 以橙皮装饰。

## 风味 / Flavor
黑麦、薄荷、深色药草、橙皮和焦糖。

## 情绪 / Mood
寒冷、成熟、像冬夜城市里一件深色羊毛大衣。

## 调酒师笔记 / Bartender Notes
- Fernet 只需少量，更多会压住黑麦。
- 糖浆用于连接苦味和辛香，不应做成甜酒。
