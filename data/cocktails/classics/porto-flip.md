# Porto Flip / 波特翻转

## 故事 / Story
Flip 是历史很早的鸡尾酒家族，Porto Flip 以波特酒和蛋黄为核心，在 19 世纪已相当流行。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 15 ml | 白兰地 |
| 45 ml | 红波特酒 |
| 1 个 | 蛋黄 |
|  | 肉豆蔻粉 少许 |

## 步骤 / Steps
1. 将所有液体材料和蛋黄不加冰充分干摇。
2. 加入冰块，再用力摇和 12-15 秒。
3. 双重过滤入预冷小杯。
4. 以肉豆蔻粉装饰。

## 风味 / Flavor
红果、坚果、葡萄干、蛋黄和香料，浓郁厚实。

## 情绪 / Mood
冬夜、壁炉、古典餐后酒的满足感。

## 调酒师笔记 / Bartender Notes
- 蛋黄需要非常新鲜，充分摇和到质地丝滑。
- 先无冰摇匀，再加冰摇冷更容易乳化。
