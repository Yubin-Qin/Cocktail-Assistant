# Adonis / 阿多尼斯

## 故事 / Story
Adonis 诞生于 19 世纪末，名字来自当时百老汇长演音乐剧，是最早也最重要的低度雪莉鸡尾酒之一。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 菲诺或阿蒙蒂亚多雪莉酒 |
| 45 ml | 甜味美思 |
| 2 dashes | 橙味苦精 |
| 1 片 | 橙皮 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 把除装饰外的材料加入调酒杯并加满冰块。
3. 用吧勺搅拌约 25-35 秒。
4. 过滤入预冷鸡尾酒杯。
5. 以橙皮装饰。

## 风味 / Flavor
坚果、红果、草本、橙皮和葡萄酒酸度。

## 情绪 / Mood
安静、文艺、适合不想喝太烈的长谈夜晚。

## 调酒师笔记 / Bartender Notes
- 低酒精度但风味复杂，关键是新鲜开瓶的味美思和雪莉。
- 阿蒙蒂亚多更坚果，菲诺更干爽。
