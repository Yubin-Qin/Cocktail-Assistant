# Penicillin / 盘尼西林

## 故事 / Story
Penicillin 由 Sam Ross 在 2000 年代纽约创作，借用 Whiskey Sour 结构并加入苏格兰威士忌、姜和蜂蜜，成为最有影响力的现代经典之一。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 调和苏格兰威士忌 |
| 22.5 ml | 新鲜柠檬汁 |
| 22.5 ml | 蜂蜜姜糖浆 |
| 7.5 ml | 艾雷岛泥煤威士忌 |
| 1 份 | 姜片或柠檬皮 |

## 步骤 / Steps
1. 将调和威士忌、柠檬汁和蜂蜜姜糖浆加冰摇和。
2. 过滤入装有大冰块的古典杯。
3. 将泥煤威士忌缓慢漂浮在表面。
4. 以姜片或柠檬皮装饰。

## 风味 / Flavor
蜂蜜、姜、柠檬、麦芽和烟熏泥煤。

## 情绪 / Mood
温暖、治愈、像寒冷天气里一件厚外套。

## 调酒师笔记 / Bartender Notes
- 泥煤威士忌只需漂浮少量，提供烟熏香气。
- 蜂蜜姜糖浆要有真实姜辣，而不只是甜。
