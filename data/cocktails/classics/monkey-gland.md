# Monkey Gland / 猴腺

## 故事 / Story
Monkey Gland 由巴黎调酒师 Harry MacElhone 在 1920 年代推广，名字来自当时轰动一时的医学实验新闻。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 50 ml | 琴酒 |
| 30 ml | 橙汁 |
| 5 ml | 石榴糖浆 |
| 1 dash 或涮杯 | 苦艾酒 |
| 1 片 | 橙皮 |

## 步骤 / Steps
1. 用少量苦艾酒涮洗预冷鸡尾酒杯，倒掉多余酒液。
2. 将其余材料加冰摇和 10-12 秒。
3. 双重过滤入杯。
4. 以橙皮装饰。

## 风味 / Flavor
橙子、杜松子、石榴和一丝茴香。

## 情绪 / Mood
古怪、爵士时代、带一点科学怪谈式幽默。

## 调酒师笔记 / Bartender Notes
- 苦艾酒只需一丝，作用是增加茴香和药草背景。
- 橙汁应新鲜，石榴糖浆只负责颜色和少量甜味。
