# El Diablo / 恶魔

## 故事 / Story
El Diablo 出现在 20 世纪中期美国西海岸的龙舌兰鸡尾酒文献中，是早期经典 Tequila 长饮之一。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 龙舌兰 reposado |
| 15 ml | 黑加仑利口酒 |
| 15 ml | 新鲜青柠汁 |
| 90 ml | 姜汁啤酒 |
| 1 份 | 青柠角 |

## 步骤 / Steps
1. 在高球杯中加入主要酒液。
2. 加满冰块。
3. 补入汽水或长饮部分。
4. 轻轻搅拌 2-3 圈。
5. 以青柠角装饰。

## 风味 / Flavor
黑加仑、姜、青柠、烤龙舌兰和轻微辛辣。

## 情绪 / Mood
活泼、暗红、像派对里一杯带小恶作剧的长饮。

## 调酒师笔记 / Bartender Notes
- 黑加仑利口酒甜度高，只需少量。
- 姜汁啤酒的辛辣能撑起龙舌兰和莓果。
