# Army & Navy / 陆军与海军

## 故事 / Story
Army & Navy 是 20 世纪早期的美国经典，名称可能与军种比赛或俱乐部文化有关，具体起源仍有争议。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 琴酒 |
| 22.5 ml | 新鲜柠檬汁 |
| 15 ml | 杏仁糖浆 |
| 1 dash | 安哥斯图拉苦精 |
| 1 片 | 柠檬皮 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 将除装饰外的所有材料加入摇酒壶。
3. 加满冰块，用力摇和约 10-12 秒。
4. 双重过滤入预冷鸡尾酒杯。
5. 以柠檬皮装饰。

## 风味 / Flavor
柠檬、杜松子、杏仁和轻微香料。

## 情绪 / Mood
整洁、老派、像制服上的金属纽扣。

## 调酒师笔记 / Bartender Notes
- 杏仁糖浆要有真实杏仁香而不是糖精味。
- 苦精只需一滴增加香料深度。
