# Paloma / 帕洛玛

## 故事 / Story
Paloma 是墨西哥最受欢迎的龙舌兰长饮之一，具体发明史并不明确，但龙舌兰与葡萄柚汽水的组合已成为现代标准。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 50 ml | 龙舌兰 blanco |
| 15 ml | 新鲜青柠汁 |
| 100-120 ml | 葡萄柚汽水 |
|  | 盐 一小撮 |
| 1 片 | 葡萄柚片 |

## 步骤 / Steps
1. 在高球杯中加入主要酒液。
2. 加满冰块。
3. 补入汽水或长饮部分。
4. 轻轻搅拌 2-3 圈。
5. 以葡萄柚片装饰。

## 风味 / Flavor
葡萄柚、青柠、矿物、草本和轻微苦味。

## 情绪 / Mood
明亮、清爽、比 Margarita 更随性。

## 调酒师笔记 / Bartender Notes
- 少量盐会放大葡萄柚和龙舌兰的风味。
- 没有葡萄柚汽水时可用葡萄柚汁、糖浆和苏打水自调。
