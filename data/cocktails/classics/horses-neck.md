# Horse's Neck / 马颈

## 故事 / Story
Horse’s Neck 原本可能是不含酒精的姜汁汽水饮品，后来加入白兰地或波本，成为早期高球经典。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 40 ml | 干邑或波本 |
| 2 dashes | 安哥斯图拉苦精 |
| 120 ml | 姜汁汽水 |
| 1 条 | 整条长柠檬皮 |

## 步骤 / Steps
1. 在高球杯中加入主要酒液。
2. 加满冰块。
3. 补入汽水或长饮部分。
4. 轻轻搅拌 2-3 圈。
5. 以长柠檬皮装饰。

## 风味 / Flavor
姜、柠檬皮、木桶和轻微香料苦味。

## 情绪 / Mood
老派、轻松、带一点骑术俱乐部般的古典气质。

## 调酒师笔记 / Bartender Notes
- 长柠檬皮要沿杯壁盘绕，形似马颈，是视觉标志。
- 姜汁汽水要冰冷且不过甜。
