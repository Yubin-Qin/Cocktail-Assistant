# Chartreuse Swizzle / 夏翠丝斯威泽

## 故事 / Story
Chartreuse Swizzle 是 21 世纪极具影响力的现代经典，把历史悠久的法国草本利口酒放进加勒比 swizzle 结构中。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 绿夏翠丝 |
| 30 ml | 菠萝汁 |
| 22.5 ml | 新鲜青柠汁 |
| 15 ml | 法勒南糖浆 |
|  | 碎冰 适量 |
| 1 份 | 薄荷枝 |

## 步骤 / Steps
1. 将所有液体材料加入高球杯。
2. 加入约三分之二杯碎冰。
3. 用吧勺或 swizzle stick 快速搅动，直到杯壁结霜。
4. 补满碎冰。
5. 以薄荷枝装饰。

## 风味 / Flavor
浓烈草本、菠萝、青柠、香料和冰凉薄荷。

## 情绪 / Mood
异域、明亮、带一点修道院药草的神秘感。

## 调酒师笔记 / Bartender Notes
- 绿夏翠丝草本强烈，菠萝和碎冰能把它拉向热带风格。
- 用吧勺或 swizzle stick 快速搅动，杯壁应明显结霜。
