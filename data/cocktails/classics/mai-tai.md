# Mai Tai / 迈泰

## 故事 / Story
Mai Tai 通常与 Trader Vic 在 1940 年代的创作联系在一起，是 Tiki 文化最重要的经典之一；正统版本远比廉价果汁混合物更干练。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 30 ml | 陈年牙买加朗姆 |
| 30 ml | 农业朗姆或另一款陈年朗姆 |
| 15 ml | 橙味柑橘利口酒 |
| 15 ml | 杏仁糖浆 |
| 30 ml | 新鲜青柠汁 |
| 1 份 | 薄荷枝和青柠壳 |

## 步骤 / Steps
1. 在摇酒壶中加入除装饰外的所有材料。
2. 加满冰块，用力摇和约 10-12 秒。
3. 过滤入装有新鲜冰块的古典杯。
4. 以薄荷枝和青柠壳装饰。

## 风味 / Flavor
陈年朗姆、青柠、杏仁、橙皮和热带香料。

## 情绪 / Mood
复杂、异域、像一场精心布置的南太平洋幻想。

## 调酒师笔记 / Bartender Notes
- 核心是优质朗姆组合，不要用大量果汁掩盖。
- 杏仁糖浆提供坚果和质地，只需适量。
