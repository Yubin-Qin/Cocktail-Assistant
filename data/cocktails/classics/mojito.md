# Mojito / 莫吉托

## 故事 / Story
Mojito 与古巴哈瓦那密切相关，也常被关联到更早的朗姆、青柠、糖和薄荷饮品传统。它在 20 世纪因哈瓦那酒吧和旅行文化而广为流行，后来成为世界范围内最具辨识度的清爽长饮之一。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 50 ml | 白朗姆 |
| 25 ml | 新鲜青柠汁 |
| 20 ml，或细砂糖 2 茶匙 | 单糖浆 |
| 8-12 片 | 新鲜薄荷叶 |
| 60-90 ml | 苏打水 |
|  | 青柠角和薄荷枝装饰 |

## 步骤 / Steps
1. 高球杯中放入薄荷叶、青柠汁和糖浆。
2. 轻轻按压薄荷，让香气释放，不要捣碎成草泥。
3. 加入朗姆和碎冰，轻轻搅拌。
4. 补满苏打水，再加碎冰形成小山。
5. 用薄荷枝和青柠角装饰。

## 风味 / Flavor
清凉、薄荷、青柠、轻甜和朗姆的甘蔗香。气泡让整体更松弛、更适合长饮。

## 情绪 / Mood
清爽、松快、带有海风感。它不是沉思型的酒，而是把肩膀放下来的一杯。

## 调酒师笔记 / Bartender Notes
- 薄荷只需轻压，过度捣碎会释放苦涩叶绿味。
- 碎冰让酒快速降温，也让薄荷香更开放。
- 苏打水最后加，轻搅即可，保留气泡。
