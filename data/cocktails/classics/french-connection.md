# French Connection / 法国关系

## 故事 / Story
French Connection 在 20 世纪后期流行，名称借用了同名电影的文化影响，是典型的双材料餐后短饮。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 35 ml | 干邑 |
| 35 ml | 杏仁利口酒 |
| 1 块 | 大冰块 |

## 步骤 / Steps
1. 在古典杯中加入所有酒液。
2. 加入一块大冰块。
3. 轻轻搅拌约 15-20 秒。
4. 以橙皮，可选装饰。

## 风味 / Flavor
葡萄干、橡木、杏仁、香草和甜润尾韵。

## 情绪 / Mood
温暖、简单、像餐后沙发上的一段慢谈。

## 调酒师笔记 / Bartender Notes
- 等量是经典做法，也可提高干邑比例减轻甜度。
- 选有坚果和杏仁核感的利口酒，与干邑木桶香更协调。
