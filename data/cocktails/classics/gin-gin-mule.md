# Gin Gin Mule / 金金骡子

## 故事 / Story
Gin Gin Mule 由 Audrey Saunders 在纽约推广，被认为是 21 世纪初鸡尾酒复兴的重要现代经典。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 琴酒 |
| 22.5 ml | 新鲜青柠汁 |
| 15 ml | 单糖浆 |
| 8 片 | 薄荷叶 |
| 60-90 ml | 姜汁啤酒 |
| 1 份 | 薄荷枝 |

## 步骤 / Steps
1. 在摇酒壶中轻压薄荷与糖浆。
2. 加入琴酒和青柠汁，加冰短摇。
3. 过滤入装满冰块的高球杯。
4. 补入姜汁啤酒。
5. 以薄荷枝装饰。

## 风味 / Flavor
薄荷、姜、青柠、杜松子和气泡。

## 情绪 / Mood
清爽、聪明、像 Mojito 与 Moscow Mule 的完美交叉。

## 调酒师笔记 / Bartender Notes
- 薄荷轻压，姜汁啤酒最后补入。
- 琴酒选择清新柑橘型会更容易与姜和薄荷融合。
