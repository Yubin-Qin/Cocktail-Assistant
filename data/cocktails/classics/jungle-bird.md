# Jungle Bird / 丛林鸟

## 故事 / Story
Jungle Bird 源于 1970 年代吉隆坡酒店酒吧，后来被全球调酒界重新发现，成为最受欢迎的苦味 Tiki 经典之一。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 深色朗姆 |
| 22.5 ml | 金巴利 |
| 45 ml | 菠萝汁 |
| 15 ml | 新鲜青柠汁 |
| 15 ml | 单糖浆 |
| 1 份 | 菠萝叶 |

## 步骤 / Steps
1. 在摇酒壶中加入除装饰外的所有材料。
2. 加满冰块，用力摇和约 10-12 秒。
3. 过滤入装有新鲜冰块的古典杯。
4. 以菠萝叶装饰。

## 风味 / Flavor
菠萝、黑糖蜜、青柠和橙皮苦味，热带而成熟。

## 情绪 / Mood
异域、黄昏、像热带酒店里一杯有态度的欢迎酒。

## 调酒师笔记 / Bartender Notes
- 金巴利让热带甜味有了苦味骨架，不要省略。
- 菠萝汁要新鲜或高品质，否则会显得罐头感。
