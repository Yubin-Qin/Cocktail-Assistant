# Tuxedo / 燕尾服

## 故事 / Story
Tuxedo 得名于纽约 Tuxedo Park，与 19 世纪末美国俱乐部文化相关，是 Martini 家族的重要旁支。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 30 ml | 琴酒 |
| 30 ml | 干味美思 |
| 1 茶匙 | 黑樱桃利口酒 |
| 1 dash | 苦艾酒 |
| 2 dashes | 橙味苦精 |
| 1 份 | 柠檬皮和樱桃 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 把除装饰外的材料加入调酒杯并加满冰块。
3. 用吧勺搅拌约 25-35 秒。
4. 过滤入预冷鸡尾酒杯。
5. 以柠檬皮和樱桃装饰。

## 风味 / Flavor
干草本、杜松子、柑橘、樱桃核和茴香。

## 情绪 / Mood
正式、精致、像真正的黑领结晚宴。

## 调酒师笔记 / Bartender Notes
- 接近 Martini，但香气更复杂、比例更柔和。
- 苦艾酒和黑樱桃利口酒都只需点到为止。
