# Brooklyn / 布鲁克林

## 故事 / Story
Brooklyn 属于以纽约行政区命名的古典鸡尾酒家族，20 世纪初已有记录，是 Manhattan 的干型近亲。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 黑麦威士忌 |
| 22.5 ml | 干味美思 |
| 7.5 ml | Amer Picon 或橙味苦酒 |
| 7.5 ml | 黑樱桃利口酒 |
| 1 颗 | 酒渍樱桃 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 把除装饰外的材料加入调酒杯并加满冰块。
3. 用吧勺搅拌约 25-35 秒。
4. 过滤入预冷鸡尾酒杯。
5. 以酒渍樱桃装饰。

## 风味 / Flavor
黑麦、干草本、苦橙和樱桃核，干爽复杂。

## 情绪 / Mood
纽约、深夜、比 Manhattan 更冷门也更锋利。

## 调酒师笔记 / Bartender Notes
- Amer Picon 难寻，可用相近橙味苦酒替代。
- 黑麦应保持辛香，不能被利口酒压住。
