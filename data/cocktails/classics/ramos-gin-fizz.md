# Ramos Gin Fizz / 拉莫斯金菲士

## 故事 / Story
Ramos Gin Fizz 由 Henry C. Ramos 于 19 世纪末在新奥尔良创作，以复杂工序和极长摇和时间闻名。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 琴酒 |
| 15 ml | 新鲜柠檬汁 |
| 15 ml | 新鲜青柠汁 |
| 30 ml | 单糖浆 |
| 60 ml | 鲜奶油 |
| 20 ml | 蛋清 |
| 3 dashes | 橙花水 |
| 30-60 ml | 苏打水 |

## 步骤 / Steps
1. 将除苏打水外的材料不加冰充分干摇。
2. 加入冰块，继续长时间用力摇和。
3. 过滤入冰镇高球杯。
4. 缓慢补入苏打水，让泡沫升高。

## 风味 / Flavor
柑橘、奶油、橙花、杜松子和云朵般泡沫。

## 情绪 / Mood
奢华、柔软、像新奥尔良清晨的一杯甜梦。

## 调酒师笔记 / Bartender Notes
- 需要长时间摇和，让蛋白、奶油和柑橘完全乳化。
- 橙花水极强，只需几滴。
