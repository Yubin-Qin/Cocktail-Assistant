# Daiquiri / 戴基里

## 故事 / Story
Daiquiri 与古巴矿区和美国海军故事常被联系在一起。朗姆、青柠和糖是加勒比地区非常自然的组合，后来在哈瓦那酒吧文化中被精炼成经典酸酒。它也是检验调酒基本功的一杯：没有复杂材料可遮掩平衡问题。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 白朗姆 |
| 25 ml | 新鲜青柠汁 |
| 15-20 ml | 单糖浆 |

## 步骤 / Steps
1. 摇酒壶中加入白朗姆、青柠汁和糖浆。
2. 加满冰块，用力摇和 10-12 秒。
3. 双重过滤入预冷鸡尾酒杯。
4. 可用青柠轮或青柠皮装饰。

## 风味 / Flavor
清亮、酸甜、甘蔗、青柠皮和轻微热带果香。好的 Daiquiri 简洁得几乎透明，却很有张力。

## 情绪 / Mood
轻快、坦率、阳光感强。它适合开始一个晚上，也适合在炎热天气里把注意力拉回当下。

## 调酒师笔记 / Bartender Notes
- 青柠汁必须新鲜，瓶装汁会显得钝、苦、发闷。
- 糖浆量要按青柠酸度调整；成品应酸甜紧致而不是像柠檬水。
- 摇和后立刻饮用，温度回升会让结构松散。
