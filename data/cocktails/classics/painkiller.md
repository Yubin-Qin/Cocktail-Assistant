# Painkiller / 止痛药

## 故事 / Story
Painkiller 与英属维京群岛酒吧文化相关，后来因 Pusser’s Rum 的推广而全球知名。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 深色或海军朗姆 |
| 120 ml | 菠萝汁 |
| 30 ml | 橙汁 |
| 30 ml | 椰子奶油 |
|  | 肉豆蔻粉 少许 |

## 步骤 / Steps
1. 将除装饰外的材料加入摇酒壶。
2. 加冰摇和约 10-12 秒。
3. 过滤入装满新鲜冰块的高球杯或特色杯。
4. 撒上现磨肉豆蔻粉，并以菠萝片装饰。

## 风味 / Flavor
菠萝、椰子、橙子、深色朗姆和肉豆蔻。

## 情绪 / Mood
海岛、放空、像把时间切换到度假模式。

## 调酒师笔记 / Bartender Notes
- 朗姆强度要足够，否则会被椰子和果汁吞没。
- 肉豆蔻现磨能显著提升香气。
