# Three Dots and a Dash / 三点一划

## 故事 / Story
Three Dots and a Dash 由 Don the Beachcomber 在二战时期创作，名称是摩斯密码中的字母 V，象征 Victory。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 30 ml | 陈年农业朗姆 |
| 30 ml | 陈年朗姆 |
| 15 ml | 橙味柑橘利口酒 |
| 15 ml | 法勒南糖浆 |
| 15 ml | 蜂蜜糖浆 |
| 30 ml | 新鲜青柠汁 |
| 3 dashes | 安哥斯图拉苦精 |
|  | 碎冰 适量 |
|  | 3 颗樱桃和 1 条菠萝棒 |

## 步骤 / Steps
1. 将所有材料与少量碎冰快速搅打，或直接在杯中 swizzle。
2. 倒入特色杯并补满碎冰。
3. 让杯壁充分结霜。
4. 以3 颗樱桃和 1 条菠萝棒装饰。

## 风味 / Flavor
朗姆、蜂蜜、青柠、丁香、杏仁和热带香料。

## 情绪 / Mood
戏剧、胜利、充满 Tiki 仪式感。

## 调酒师笔记 / Bartender Notes
- 装饰中的三颗樱桃和一条长条正对应摩斯密码“V”。
- 香料材料很多，朗姆仍应是主体。
