# Hanky Panky / 花招

## 故事 / Story
Hanky Panky 由 Savoy Hotel 著名女调酒师 Ada Coleman 创作，据说一位客人喝后称赞它“真是个花招”，名字由此而来。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 琴酒 |
| 45 ml | 甜味美思 |
| 7.5 ml | Fernet-Branca |
| 1 片 | 橙皮 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 把除装饰外的材料加入调酒杯并加满冰块。
3. 用吧勺搅拌约 25-35 秒。
4. 过滤入预冷鸡尾酒杯。
5. 以橙皮装饰。

## 风味 / Flavor
深色草本、橙皮、薄荷、红果和杜松子。

## 情绪 / Mood
聪明、古怪、带一点老伦敦酒吧的机锋。

## 调酒师笔记 / Bartender Notes
- Fernet 只需少量，就能带来强烈药草和苦味。
- 使用有结构的琴酒和新鲜甜味美思。
