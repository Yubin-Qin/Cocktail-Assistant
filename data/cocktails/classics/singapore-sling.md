# Singapore Sling / 新加坡司令

## 故事 / Story
Singapore Sling 通常归功于新加坡 Raffles Hotel 的 Ngiam Tong Boon，20 世纪以来配方经历多次变化，但已成为城市象征。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 30 ml | 琴酒 |
| 15 ml | 樱桃利口酒 |
| 7.5 ml | 橙味利口酒 |
| 7.5 ml | 草本利口酒 |
| 120 ml | 菠萝汁 |
| 15 ml | 新鲜青柠汁 |
| 10 ml | 石榴糖浆 |
| 1 dash | 安哥斯图拉苦精 |
| 1 份 | 菠萝片和樱桃 |

## 步骤 / Steps
1. 将除装饰外的材料加入摇酒壶。
2. 加冰摇和约 10-12 秒。
3. 过滤入装满新鲜冰块的高球杯或特色杯。
4. 以菠萝片和樱桃装饰。

## 风味 / Flavor
菠萝、樱桃、柑橘、草本和香料，复杂热带。

## 情绪 / Mood
殖民酒店、旅行、华丽又有历史层次。

## 调酒师笔记 / Bartender Notes
- 材料多但每个都应有作用，避免做成单一甜果汁。
- 菠萝汁充分摇和会形成经典泡沫。
