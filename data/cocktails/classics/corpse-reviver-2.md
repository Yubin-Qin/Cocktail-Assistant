# Corpse Reviver #2 / 亡者复苏二号

## 故事 / Story
Corpse Reviver 是早期“醒酒”鸡尾酒家族的名称，#2 版本因 1930 年代配方书而著名，后来成为最受欢迎的等量酸酒之一。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 22.5 ml | 琴酒 |
| 22.5 ml | 橙味利口酒 |
| 22.5 ml | Lillet Blanc 或 Cocchi Americano |
| 22.5 ml | 新鲜柠檬汁 |
|  | 苦艾酒 涮杯 |
| 1 片 | 橙皮 |

## 步骤 / Steps
1. 用少量苦艾酒涮洗预冷鸡尾酒杯，倒掉多余酒液。
2. 将其余材料加冰摇和 10-12 秒。
3. 双重过滤入杯。
4. 以橙皮装饰。

## 风味 / Flavor
柠檬、橙皮、草本、白葡萄酒和一丝茴香。

## 情绪 / Mood
清醒、锋利、带一点“把夜晚重新启动”的戏剧性。

## 调酒师笔记 / Bartender Notes
- 苦艾酒只需涮杯，过多会完全主导。
- 等量结构要求四种材料品质都在线，尤其是新鲜柠檬。
