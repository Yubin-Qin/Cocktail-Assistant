# Bramble / 荆棘

## 故事 / Story
Bramble 由伦敦调酒师 Dick Bradsell 在 20 世纪 80 年代创作，灵感来自英国乡间采黑莓的记忆，现已成为现代经典。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 50 ml | 琴酒 |
| 25 ml | 新鲜柠檬汁 |
| 12.5 ml | 单糖浆 |
| 15 ml | 黑莓利口酒 |
| 1 份 | 黑莓和柠檬片 |

## 步骤 / Steps
1. 将琴酒、柠檬汁和糖浆加冰摇和。
2. 过滤入装满碎冰的古典杯。
3. 从顶部缓慢淋入黑莓利口酒。
4. 再补少量碎冰形成小山。
5. 以黑莓和柠檬片装饰。

## 风味 / Flavor
柠檬、杜松子、黑莓果酱和轻微酸涩，层次随饮用变化。

## 情绪 / Mood
英式、俏皮、像夏末灌木丛里的成熟浆果。

## 调酒师笔记 / Bartender Notes
- 先做好琴酒酸酒基底，再让黑莓利口酒从碎冰上缓慢下沉。
- 碎冰会持续稀释，基底不应调得过甜。
