# Gold Rush / 淘金热

## 故事 / Story
Gold Rush 诞生于 21 世纪初纽约 Milk & Honey 酒吧，是现代经典中最成功的三材料结构之一。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 波本威士忌 |
| 22.5 ml | 新鲜柠檬汁 |
| 22.5 ml | 蜂蜜糖浆 |
| 1 片 | 柠檬皮 |

## 步骤 / Steps
1. 在摇酒壶中加入除装饰外的所有材料。
2. 加满冰块，用力摇和约 10-12 秒。
3. 过滤入装有新鲜冰块的古典杯。
4. 以柠檬皮装饰。

## 风味 / Flavor
蜂蜜、柠檬、香草、焦糖和木桶。

## 情绪 / Mood
温暖、现代、像简单但质感很好的皮夹克。

## 调酒师笔记 / Bartender Notes
- 可视作不加姜的 Penicillin 近亲。
- 蜂蜜糖浆应充分溶解，冷藏后仍要保持流动。
