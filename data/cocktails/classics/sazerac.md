# Sazerac / 萨泽拉克

## 故事 / Story
Sazerac 与新奥尔良密不可分，其历史横跨干邑、黑麦威士忌、苦精和苦艾酒的演变，是美国最重要的古典鸡尾酒之一。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 黑麦威士忌 |
| 1 颗或单糖浆 7.5 ml | 方糖 |
| 3-4 dashes | Peychaud's 苦精 |
|  | 苦艾酒 涮杯 |
| 1 片 | 柠檬皮 |

## 步骤 / Steps
1. 用苦艾酒涮洗冰镇古典杯并倒掉多余酒液。
2. 在调酒杯中加入威士忌、糖和苦精。
3. 加冰搅拌 25-35 秒。
4. 过滤入不加冰的预冷杯中。
5. 以柠檬皮装饰。

## 风味 / Flavor
黑麦、茴香、红色苦精、柠檬油和香料。

## 情绪 / Mood
严肃、新奥尔良、像一场有历史重量的仪式。

## 调酒师笔记 / Bartender Notes
- 杯子只涮苦艾酒，不留大量液体。
- 柠檬皮通常只挤油后丢弃，保持酒体干净。
