# Dark 'N' Stormy / 黑暗风暴

## 故事 / Story
Dark ’N’ Stormy 与百慕大朗姆和姜汁啤酒文化紧密相关，因杯中深色朗姆像风暴云而得名。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 深色朗姆 |
| 100-120 ml | 姜汁啤酒 |
| 15 ml | 新鲜青柠汁 |
| 1 份 | 青柠角 |

## 步骤 / Steps
1. 在高球杯中加入姜汁啤酒和青柠汁。
2. 加满冰块。
3. 将深色朗姆沿吧勺背面缓慢漂浮在顶部。
4. 以青柠角装饰。

## 风味 / Flavor
黑糖蜜、姜辣、青柠和深色朗姆的烘烤感。

## 情绪 / Mood
海风、暴雨、冒险感强，是很有画面的长饮。

## 调酒师笔记 / Bartender Notes
- 先倒姜汁啤酒，再把深色朗姆缓慢漂浮在上方可形成“风暴云”视觉。
- 姜汁啤酒应有明显辛辣度。
