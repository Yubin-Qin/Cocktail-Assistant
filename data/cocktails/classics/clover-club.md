# Clover Club / 三叶草俱乐部

## 故事 / Story
Clover Club 得名于费城的同名绅士俱乐部，在禁酒令前已相当流行，后来随着古典鸡尾酒复兴重回主流。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 琴酒 |
| 22.5 ml | 新鲜柠檬汁 |
| 15 ml | 覆盆子糖浆 |
| 15-20 ml | 蛋清 |
| 2-3 颗 | 覆盆子 |

## 步骤 / Steps
1. 将所有材料加入摇酒壶，不加冰干摇约 10 秒。
2. 加入冰块，再用力摇和 10-12 秒。
3. 双重过滤入预冷鸡尾酒杯。
4. 以覆盆子装饰。

## 风味 / Flavor
覆盆子、柠檬、杜松子和柔滑蛋白泡沫。

## 情绪 / Mood
优雅、社交、略带 20 世纪初俱乐部文化的精致。

## 调酒师笔记 / Bartender Notes
- 先干摇再加冰摇，泡沫会更细腻。
- 覆盆子糖浆要保留酸度，不要做成果酱甜味。
