# Amaretto Sour / 杏仁酸

## 故事 / Story
Amaretto Sour 在 20 世纪后期广泛流行，现代调酒师通过加入高强度波本和新鲜柠檬重塑了它的平衡。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 杏仁利口酒 |
| 22.5 ml | 波本威士忌 |
| 30 ml | 新鲜柠檬汁 |
| 5 ml | 单糖浆 |
| 20 ml | 蛋清 |
| 2 dashes | 苦精 |

## 步骤 / Steps
1. 将所有材料不加冰干摇约 10 秒。
2. 加入冰块，再用力摇和 10-12 秒。
3. 过滤入装有新鲜冰块的古典杯。
4. 以柠檬皮和樱桃装饰。

## 风味 / Flavor
杏仁、核果、柠檬、香草和柔滑泡沫。

## 情绪 / Mood
亲切、甜而不幼稚，是很好的威士忌入门酒。

## 调酒师笔记 / Bartender Notes
- 加入波本能避免只剩甜杏仁味，是现代高质量版本的关键。
- 蛋清让酸度和甜度更融合。
