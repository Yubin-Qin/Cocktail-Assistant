# Paradise / 天堂

## 故事 / Story
Paradise 是 20 世纪早期的经典配方，出现在著名调酒书中，以简洁果香结构延续至今。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 35 ml | 琴酒 |
| 20 ml | 杏子白兰地利口酒 |
| 15 ml | 橙汁 |
| 1 片 | 橙皮 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 将除装饰外的所有材料加入摇酒壶。
3. 加满冰块，用力摇和约 10-12 秒。
4. 双重过滤入预冷鸡尾酒杯。
5. 以橙皮装饰。

## 风味 / Flavor
杏桃、橙子、杜松子和柔和核果甜香。

## 情绪 / Mood
柔和、阳光、带一点老酒店酒吧的浪漫。

## 调酒师笔记 / Bartender Notes
- 杏子利口酒甜度高，橙汁只需少量。
- 使用干型琴酒保持整体线条。
