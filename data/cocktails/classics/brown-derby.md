# Brown Derby / 棕色德比帽

## 故事 / Story
Brown Derby 通常与 1930 年代好莱坞同名餐厅文化联系在一起，是非常耐喝的波本蜂蜜酸酒。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 波本威士忌 |
| 30 ml | 新鲜葡萄柚汁 |
| 15 ml | 蜂蜜糖浆 |
| 1 片 | 葡萄柚皮 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 将除装饰外的所有材料加入摇酒壶。
3. 加满冰块，用力摇和约 10-12 秒。
4. 双重过滤入预冷鸡尾酒杯。
5. 以葡萄柚皮装饰。

## 风味 / Flavor
葡萄柚、蜂蜜、波本香草和轻微苦味。

## 情绪 / Mood
好莱坞黄金时代、阳光、成熟而亲切。

## 调酒师笔记 / Bartender Notes
- 蜂蜜糖浆与葡萄柚的苦酸要保持张力。
- 波本的香草和焦糖能很好连接蜂蜜。
