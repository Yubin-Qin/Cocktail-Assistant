# Remember the Maine / 记住缅因号

## 故事 / Story
Remember the Maine 的名字来自 1898 年缅因号事件及相关口号，后来由鸡尾酒作家 Charles H. Baker 记录并流传。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 黑麦威士忌 |
| 22.5 ml | 甜味美思 |
| 7.5 ml | 樱桃利口酒 |
|  | 苦艾酒 涮杯 |
| 1 颗 | 酒渍樱桃 |

## 步骤 / Steps
1. 用少量苦艾酒涮洗预冷鸡尾酒杯。
2. 将其余材料加入调酒杯，加冰搅拌 25-35 秒。
3. 过滤入杯。
4. 以酒渍樱桃装饰。

## 风味 / Flavor
黑麦辛香、红果、樱桃核和茴香草本。

## 情绪 / Mood
历史感、深色、像一段旧报纸标题。

## 调酒师笔记 / Bartender Notes
- 可视作带樱桃和苦艾酒的 Manhattan 亲戚。
- 苦艾酒只需薄薄一层，不要盖住黑麦。
