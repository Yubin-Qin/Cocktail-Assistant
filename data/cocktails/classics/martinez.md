# Martinez / 马丁内斯

## 故事 / Story
Martinez 是 19 世纪末重要的琴酒与味美思鸡尾酒，常被视为 Martini 演化链上的关键祖先之一。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 老汤姆琴酒 |
| 45 ml | 甜味美思 |
| 5 ml | 黑樱桃利口酒 |
| 2 dashes | 橙味苦精 |
| 1 片 | 柠檬皮 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 把除装饰外的材料加入调酒杯并加满冰块。
3. 用吧勺搅拌约 25-35 秒。
4. 过滤入预冷鸡尾酒杯。
5. 以柠檬皮装饰。

## 风味 / Flavor
红果、草本、杜松子、樱桃核和柑橘油。

## 情绪 / Mood
古典、温柔、像 Martini 尚未变得冷峻之前的样子。

## 调酒师笔记 / Bartender Notes
- 老汤姆琴酒能提供比伦敦干琴酒更圆润的过渡。
- 黑樱桃利口酒只作结构修饰。
