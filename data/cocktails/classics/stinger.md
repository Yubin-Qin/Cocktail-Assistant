# Stinger / 毒刺

## 故事 / Story
Stinger 在 20 世纪早期美国上流社交场合非常流行，曾是著名的夜晚餐后酒。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 50 ml | 干邑 |
| 20 ml | 白薄荷利口酒 |
| 1 片 | 薄荷叶 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 把除装饰外的材料加入调酒杯并加满冰块。
3. 用吧勺搅拌约 25-35 秒。
4. 过滤入预冷鸡尾酒杯。
5. 以薄荷叶，可选装饰。

## 风味 / Flavor
干邑、薄荷、葡萄干和清凉甜味。

## 情绪 / Mood
餐后、老纽约、像西装口袋里的一颗薄荷糖。

## 调酒师笔记 / Bartender Notes
- 白薄荷利口酒比绿色版本更适合保持颜色清澈。
- 可按口味降低利口酒比例，避免牙膏感。
