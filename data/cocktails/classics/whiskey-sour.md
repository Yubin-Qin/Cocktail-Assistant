# Whiskey Sour / 威士忌酸

## 故事 / Story
Sour 类鸡尾酒的公式很古老：烈酒、柑橘和甜味。Whiskey Sour 在 19 世纪已经成为美国酒吧的常见饮品，后来发展出加蛋清的 Boston Sour 版本。它是理解酸酒平衡的核心样本。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 波本或黑麦威士忌 |
| 25 ml | 新鲜柠檬汁 |
| 20 ml | 单糖浆 |
| 15-20 ml | 蛋清 |
| 1-2 dashes | 安哥斯图拉苦精 |

## 步骤 / Steps
1. 若使用蛋清，先将所有材料不加冰干摇 10 秒。
2. 加冰后再摇 10-12 秒。
3. 过滤入古典杯或鸡尾酒杯。
4. 滴苦精在泡沫表面，或用柠檬皮装饰。

## 风味 / Flavor
柠檬酸、谷物甜、木桶香、柔和泡沫和微苦香料。结构清楚，酸甜酒香各有位置。

## 情绪 / Mood
舒展、亲切、略带复古。它像一首旋律清楚的老歌，熟悉但不会无聊。

## 调酒师笔记 / Bartender Notes
- 蛋清带来柔滑泡沫，但不是必需；不用蛋清时可摇得更利落。
- 柠檬汁和糖浆要平衡，不能只有尖酸。
- 威士忌选择会改变个性：波本甜润，黑麦辛辣。
