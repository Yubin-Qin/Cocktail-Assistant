# Bamboo / 竹子

## 故事 / Story
Bamboo 通常与 19 世纪末横滨 Grand Hotel 的 Louis Eppinger 联系在一起，是最重要的低酒精经典之一。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 菲诺或阿蒙蒂亚多雪莉酒 |
| 45 ml | 干味美思 |
| 2 dashes | 橙味苦精 |
| 1 dash | 安哥斯图拉苦精 |
| 1 片 | 柠檬皮 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 把除装饰外的材料加入调酒杯并加满冰块。
3. 用吧勺搅拌约 25-35 秒。
4. 过滤入预冷鸡尾酒杯。
5. 以柠檬皮装饰。

## 风味 / Flavor
坚果、干草本、柠檬油和葡萄酒酸度。

## 情绪 / Mood
安静、细腻、适合慢慢进入夜晚。

## 调酒师笔记 / Bartender Notes
- 这是一杯低度酒，氧化变质的味美思会非常明显。
- 菲诺更清脆，阿蒙蒂亚多更坚果。
