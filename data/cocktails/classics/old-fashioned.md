# Old Fashioned / 古典鸡尾酒

## 故事 / Story
Old Fashioned 的名字本身就带有怀旧意味。19 世纪鸡尾酒常被定义为烈酒、糖、水和苦精的组合，后来配方越来越复杂，一些饮客开始要求“老式做法”。这杯酒于是成了鸡尾酒原型的象征。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 波本或黑麦威士忌 |
| 1 颗，或单糖浆 7.5 ml | 方糖 |
| 2-3 dashes | 安哥斯图拉苦精 |
| 1 片 | 橙皮 |
| 1 块 | 大冰块 |

## 步骤 / Steps
1. 在古典杯中放入方糖，滴上苦精。
2. 加少量水，将方糖压散并搅匀；若使用糖浆则直接加入。
3. 加入威士忌和大冰块。
4. 慢慢搅拌 20-30 秒。
5. 挤橙皮精油并擦杯口，橙皮投入杯中。

## 风味 / Flavor
木桶、焦糖、橙油、香料和微苦。第一口有糖和橙香的柔化，随后威士忌逐渐展开。

## 情绪 / Mood
朴素、可靠、带一点旧派魅力。它适合慢下来，也适合在一堆复杂选择里回到基本面。

## 调酒师笔记 / Bartender Notes
- 这杯酒的重点是威士忌本身，不要让糖过量。
- 大冰块能减慢稀释，让饮用过程有层次变化。
- 橙皮要现削，白色海绵层越少越好。
