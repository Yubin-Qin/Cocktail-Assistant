# Long Island Iced Tea / 长岛冰茶

## 故事 / Story
Long Island Iced Tea 的确切起源有争议，但 20 世纪后期在美国酒吧文化中迅速普及，成为最著名的多基酒长饮之一。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 15 ml | 伏特加 |
| 15 ml | 琴酒 |
| 15 ml | 白朗姆 |
| 15 ml | 龙舌兰 |
| 15 ml | 橙味利口酒 |
| 25 ml | 新鲜柠檬汁 |
| 15 ml | 单糖浆 |
| 30-60 ml | 可乐 |
| 1 片 | 柠檬片 |

## 步骤 / Steps
1. 将除可乐和装饰外的材料加入摇酒壶。
2. 加冰短摇约 8-10 秒。
3. 过滤入装满冰块的高球杯。
4. 补入可乐，轻轻搅拌。
5. 以柠檬片装饰。

## 风味 / Flavor
柑橘、可乐、轻微草本和复杂烈酒感，入口比酒精度更轻松。

## 情绪 / Mood
热闹、危险、适合派对但需要慢喝。

## 调酒师笔记 / Bartender Notes
- 名字里没有茶，颜色来自可乐。
- 多种基酒并不代表要倒得过量，精确量取非常重要。
