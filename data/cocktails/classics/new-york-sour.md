# New York Sour / 纽约酸

## 故事 / Story
New York Sour 是 19 世纪 Whiskey Sour 家族的著名变体，红酒漂浮层使它成为最具视觉辨识度的经典之一。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 黑麦或波本威士忌 |
| 25 ml | 新鲜柠檬汁 |
| 20 ml | 单糖浆 |
| 15 ml | 蛋清 |
| 15 ml | 干红葡萄酒 |
| 1 片 | 柠檬皮 |

## 步骤 / Steps
1. 将威士忌、柠檬汁、糖浆和可选蛋清按 Sour 方法摇和。
2. 过滤入装有新鲜冰块的古典杯。
3. 将红葡萄酒从吧勺背面缓慢漂浮在表面。
4. 以柠檬皮装饰。

## 风味 / Flavor
柠檬酸、威士忌木桶、红果和柔和单宁。

## 情绪 / Mood
戏剧性、都市、像夜色中一层深红天际线。

## 调酒师笔记 / Bartender Notes
- 红酒要从吧勺背面缓慢漂浮，形成清晰分层。
- 选择果香明显、单宁不过重的干红。
