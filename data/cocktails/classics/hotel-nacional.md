# Hotel Nacional / 国家酒店特调

## 故事 / Story
Hotel Nacional Special 与哈瓦那著名 Hotel Nacional de Cuba 的酒吧传统联系紧密，是古巴黄金时代的重要经典。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 陈年白朗姆 |
| 15 ml | 杏子利口酒 |
| 30 ml | 菠萝汁 |
| 15 ml | 新鲜青柠汁 |
| 5 ml | 单糖浆 |
| 1 片 | 青柠轮 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 将除装饰外的所有材料加入摇酒壶。
3. 加满冰块，用力摇和约 10-12 秒。
4. 双重过滤入预冷鸡尾酒杯。
5. 以青柠轮装饰。

## 风味 / Flavor
菠萝、杏桃、青柠和陈年甘蔗香。

## 情绪 / Mood
哈瓦那、酒店、像 1930 年代旅行海报。

## 调酒师笔记 / Bartender Notes
- 杏子利口酒和菠萝都偏甜，青柠必须清晰。
- 使用有陈年感但不太重的朗姆。
