# Gin Basil Smash / 罗勒金酒砸

## 故事 / Story
Gin Basil Smash 由德国汉堡调酒师 Jörg Meyer 在 2008 年创作，迅速成为全球最成功的现代经典之一。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 琴酒 |
| 25 ml | 新鲜柠檬汁 |
| 20 ml | 单糖浆 |
| 10-15 片 | 新鲜罗勒叶 |
| 1 份 | 罗勒枝 |

## 步骤 / Steps
1. 将新鲜香草与糖浆放入摇酒壶，轻轻按压。
2. 加入其余材料和冰块，用力摇和。
3. 双重过滤入装有新鲜冰块的古典杯。
4. 以罗勒枝装饰。

## 风味 / Flavor
新鲜罗勒、柠檬、杜松子和草本绿色感。

## 情绪 / Mood
现代、清新、像春夏植物园里的冷空气。

## 调酒师笔记 / Bartender Notes
- 罗勒可以充分压香，但不要连硬茎一起过度捣碎。
- 双重过滤可减少碎叶，同时保留鲜绿色。
