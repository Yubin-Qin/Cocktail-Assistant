# Brandy Crusta / 白兰地杯缘

## 故事 / Story
Brandy Crusta 可追溯到 19 世纪的新奥尔良，被认为对 Sidecar 等后来的柑橘酸酒产生了重要影响。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 52.5 ml | 干邑 |
| 7.5 ml | 黑樱桃利口酒 |
| 7.5 ml | 橙味利口酒 |
| 7.5 ml | 新鲜柠檬汁 |
| 5 ml | 单糖浆 |
| 2 dashes | 安哥斯图拉苦精 |
| 1 份 | 糖边和长柠檬皮 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 将除装饰外的所有材料加入摇酒壶。
3. 加满冰块，用力摇和约 10-12 秒。
4. 双重过滤入预冷鸡尾酒杯。
5. 以糖边和长柠檬皮装饰。

## 风味 / Flavor
干邑果香、柑橘、香料与糖边，古典而华丽。

## 情绪 / Mood
戏剧性、复古、像 19 世纪酒吧的一件精致器物。

## 调酒师笔记 / Bartender Notes
- 杯口糖壳和长柠檬皮是标志，不只是装饰。
- 甜味元素很多，糖浆只需少量。
