# Revolver / 左轮手枪

## 故事 / Story
Revolver 由旧金山调酒师 Jon Santer 在 2000 年代创作，成为咖啡与威士忌结合的现代经典。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 波本威士忌 |
| 15 ml | 咖啡利口酒 |
| 2 dashes | 橙味苦精 |
| 1 片 | 橙皮 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 把除装饰外的材料加入调酒杯并加满冰块。
3. 用吧勺搅拌约 25-35 秒。
4. 过滤入预冷鸡尾酒杯。
5. 以火焰橙皮或普通橙皮装饰。

## 风味 / Flavor
咖啡、橙皮、波本香草和木桶。

## 情绪 / Mood
夜色、电影、像一杯更现代的 Old Fashioned。

## 调酒师笔记 / Bartender Notes
- 选择干爽咖啡利口酒，避免像甜咖啡。
- 橙皮可点燃喷油增加戏剧性，但家庭操作时普通挤油更安全。
