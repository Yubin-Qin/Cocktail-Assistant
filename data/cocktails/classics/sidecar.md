# Sidecar / 边车

## 故事 / Story
Sidecar 常被认为兴起于一战后欧洲的酒吧，名字据说与乘坐摩托车边车的客人有关。它属于 Daisy/Sour 家族，使用干邑让这杯酒比朗姆或琴酒酸酒更温暖、更有绅士感。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 50 ml | 干邑或优质白兰地 |
| 25 ml | 橙味利口酒 |
| 25 ml | 新鲜柠檬汁 |
|  | 糖边 |
|  | 柠檬皮装饰 |

## 步骤 / Steps
1. 可用柠檬擦杯口并轻沾细砂糖。
2. 摇酒壶中加入干邑、橙味利口酒和柠檬汁。
3. 加冰摇和 10-12 秒。
4. 过滤入预冷鸡尾酒杯。
5. 用柠檬皮装饰。

## 风味 / Flavor
葡萄蒸馏酒的成熟果香、橙皮甜香、柠檬酸度和轻微橡木。精致、明亮又有厚度。

## 情绪 / Mood
优雅、怀旧、带有一点巴黎夜色的想象。适合需要精致但不想太沉重的场景。

## 调酒师笔记 / Bartender Notes
- 传统配方可能更干，现代口味可微调为 2:1:1。
- 糖边会明显增加入口甜感，做半圈更灵活。
- 干邑不必极昂贵，但需要有果香和木桶骨架。
