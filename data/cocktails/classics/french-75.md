# French 75 / 法式 75

## 故事 / Story
French 75 得名于一战时期的法国 75 毫米野战炮，暗示这杯酒的力度。它把琴酒酸酒和香槟结合起来，既有鸡尾酒结构，也有起泡酒的节庆感。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 30 ml | 琴酒 |
| 15 ml | 新鲜柠檬汁 |
| 15 ml | 单糖浆 |
| 60-90 ml | 香槟或干型起泡酒 |
|  | 柠檬皮装饰 |

## 步骤 / Steps
1. 摇酒壶中加入琴酒、柠檬汁和糖浆。
2. 加冰摇和 8-10 秒。
3. 过滤入香槟杯。
4. 缓慢补入冰镇起泡酒。
5. 用柠檬皮装饰。

## 风味 / Flavor
柠檬、杜松子、细腻气泡、白花和烘焙香。轻盈外表下有不低的酒精强度。

## 情绪 / Mood
庆祝感、优雅、带一点危险的轻快。它像一场开场白漂亮但后劲十足的晚宴。

## 调酒师笔记 / Bartender Notes
- 起泡酒要足够干，否则整体会偏甜。
- 先摇酸酒基底，再补起泡酒，避免气泡被摇散。
- 杯子和起泡酒都要充分冰镇。
