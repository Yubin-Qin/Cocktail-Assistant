# Bloody Mary / 血腥玛丽

## 故事 / Story
Bloody Mary 的起源存在多种说法，但 20 世纪巴黎和纽约酒吧都与其传播有关；它后来发展成最具个性化空间的经典之一。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 伏特加 |
| 90 ml | 番茄汁 |
| 15 ml | 新鲜柠檬汁 |
| 2-3 dashes | 伍斯特酱 |
| 2-4 dashes | 辣椒酱 |
|  | 盐和黑胡椒 适量 |
| 1 份 | 芹菜杆或柠檬角 |

## 步骤 / Steps
1. 将除装饰外的所有材料加入一个大杯。
2. 加冰后在两个杯具之间来回滚动混合 3-4 次。
3. 倒入装满新鲜冰块的高球杯。
4. 以芹菜杆或柠檬角装饰。

## 风味 / Flavor
番茄鲜味、酸、咸、辣和香料感，像一杯可饮用的冷汤。

## 情绪 / Mood
清醒、恢复、很有早午餐气质，适合慢慢喝。

## 调酒师笔记 / Bartender Notes
- 番茄汁不适合暴力摇和，采用滚动混合可减少泡沫。
- 咸、酸、辣要逐步调，伍斯特酱和辣椒酱品牌差异很大。
