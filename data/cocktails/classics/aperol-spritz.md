# Aperol Spritz / 阿佩罗橙光

## 故事 / Story
Spritz 源自意大利东北部的开胃酒传统，Aperol 版本在 20 世纪后期成为全球最流行的低度鸡尾酒之一。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | Aperol |
| 90 ml | 普罗塞克 |
| 30 ml | 苏打水 |
| 1 片 | 橙片 |

## 步骤 / Steps
1. 在大葡萄酒杯中加满冰块。
2. 先倒入起泡酒，再加入 Aperol。
3. 补入苏打水并轻轻搅拌。
4. 以橙片装饰。

## 风味 / Flavor
苦橙、气泡、轻草本和柔和甜味。

## 情绪 / Mood
黄昏、社交、意大利广场般的松弛感。

## 调酒师笔记 / Bartender Notes
- 经典记忆法是 3 份起泡酒、2 份 Aperol、1 份苏打水。
- 使用大杯和大量冰，保持整个饮用过程低温。
