# Naked and Famous / 赤裸与名声

## 故事 / Story
Naked and Famous 由调酒师 Joaquín Simó 于 2011 年创作，常被看作 Last Word 与 Paper Plane 两种等量结构的交叉灵感。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 22.5 ml | 梅斯卡尔 |
| 22.5 ml | 黄色夏翠丝 |
| 22.5 ml | Aperol |
| 22.5 ml | 新鲜青柠汁 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 将除装饰外的所有材料加入摇酒壶。
3. 加满冰块，用力摇和约 10-12 秒。
4. 双重过滤入预冷鸡尾酒杯。

## 风味 / Flavor
烟熏、青柠、橙皮、草本和柔和苦甜。

## 情绪 / Mood
现代、聪明、带一点纽约酒吧的冷幽默。

## 调酒师笔记 / Bartender Notes
- 四种材料严格等量，梅斯卡尔不宜选烟熏过重的款式。
- 青柠必须足够新鲜，才能撑住两种利口酒。
