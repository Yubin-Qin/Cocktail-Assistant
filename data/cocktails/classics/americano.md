# Americano / 美式鸡尾酒

## 故事 / Story
Americano 源自意大利开胃酒传统，常被视为 Milano-Torino 的加苏打版本，也是 Negroni 的直接前身。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 30 ml | 金巴利 |
| 30 ml | 甜味美思 |
| 60-90 ml | 苏打水 |
| 1 份 | 橙片或橙皮 |

## 步骤 / Steps
1. 在高球杯中加入主要酒液。
2. 加满冰块。
3. 补入汽水或长饮部分。
4. 轻轻搅拌 2-3 圈。
5. 以橙片或橙皮装饰。

## 风味 / Flavor
橙皮、草本、红果和柔和苦甜，酒精感低于 Negroni。

## 情绪 / Mood
松弛、开胃、适合黄昏，像一段不急着进入正题的谈话。

## 调酒师笔记 / Bartender Notes
- 先把金巴利和味美思充分冰镇，再补苏打水，苦味会更圆润。
- 不要搅拌过度，保留气泡和轻盈感。
