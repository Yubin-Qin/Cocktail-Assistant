# Alexander / 亚历山大

## 故事 / Story
Alexander 在 20 世纪初已经见于鸡尾酒文献，早期常以琴酒为基酒；后来使用干邑的 Brandy Alexander 更为流行，逐渐成为奶油类经典的代表。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 30 ml | 干邑或琴酒 |
| 30 ml | 深色可可利口酒 |
| 30 ml | 鲜奶油 |
|  | 肉豆蔻粉 少许 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 将除装饰外的所有材料加入摇酒壶。
3. 加满冰块，用力摇和约 10-12 秒。
4. 双重过滤入预冷鸡尾酒杯。
5. 以肉豆蔻粉装饰。

## 风味 / Flavor
可可、奶油、葡萄蒸馏酒或杜松子香，甜润而丝滑，像液体甜点。

## 情绪 / Mood
柔软、复古、带一点晚宴尾声的从容，适合作为餐后酒。

## 调酒师笔记 / Bartender Notes
- 三种主体等量是经典结构，奶油必须新鲜且充分冰镇。
- 摇和要有力，让酒体细腻融合，但不要让奶油变成厚重泡沫。
