# Hemingway Special / 海明威特调

## 故事 / Story
这杯酒与 Ernest Hemingway 在哈瓦那 El Floridita 的饮酒习惯相关，由 Daiquiri 演变而来，因此也常被称为 Hemingway Daiquiri。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 白朗姆 |
| 40 ml | 新鲜葡萄柚汁 |
| 15 ml | 新鲜青柠汁 |
| 15 ml | 黑樱桃利口酒 |
| 1 片 | 青柠轮 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 将除装饰外的所有材料加入摇酒壶。
3. 加满冰块，用力摇和约 10-12 秒。
4. 双重过滤入预冷鸡尾酒杯。
5. 以青柠轮装饰。

## 风味 / Flavor
葡萄柚、青柠、甘蔗和樱桃核，干爽而锐利。

## 情绪 / Mood
清醒、文学、带一点哈瓦那午后的孤独感。

## 调酒师笔记 / Bartender Notes
- 不加额外糖浆，整体比普通 Daiquiri 更干。
- 葡萄柚的轻苦与黑樱桃核香是关键。
