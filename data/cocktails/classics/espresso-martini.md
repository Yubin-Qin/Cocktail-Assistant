# Espresso Martini / 浓缩马天尼

## 故事 / Story
Espresso Martini 由伦敦调酒师 Dick Bradsell 在 20 世纪 80 年代创作的故事广为流传。虽然名字里有 Martini，它并不属于传统马天尼结构，而是借用鸡尾酒杯和都市感命名，后来成为现代经典。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 伏特加 |
| 25 ml | 咖啡利口酒 |
| 30 ml | 新鲜浓缩咖啡 |
| 5-10 ml | 单糖浆 |
| 3 颗装饰 | 咖啡豆 |

## 步骤 / Steps
1. 提前萃取浓缩咖啡，稍微降温。
2. 摇酒壶中加入伏特加、咖啡利口酒、浓缩咖啡和糖浆。
3. 加冰用力摇和 12-15 秒，摇出细腻泡沫。
4. 双重过滤入预冷鸡尾酒杯。
5. 放上 3 颗咖啡豆。

## 风味 / Flavor
咖啡烘焙、可可、焦糖、伏特加的干净支撑和轻微苦甜。入口像冷萃和甜点之间的交界。

## 情绪 / Mood
夜晚、精神、带一点都市兴奋感。适合晚餐后但还不想结束的时刻。

## 调酒师笔记 / Bartender Notes
- 咖啡要新鲜，陈旧咖啡会带焦苦和空洞感。
- 摇和必须有力，泡沫是这杯酒的标志。
- 咖啡利口酒已经有甜度，糖浆只用于微调。
