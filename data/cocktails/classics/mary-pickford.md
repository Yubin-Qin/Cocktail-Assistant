# Mary Pickford / 玛丽·碧克馥

## 故事 / Story
Mary Pickford 以默片时代著名女演员命名，通常被认为诞生于 1920 年代哈瓦那，是古巴鸡尾酒黄金时代的代表之一。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 白朗姆 |
| 45 ml | 菠萝汁 |
| 7.5 ml | 黑樱桃利口酒 |
| 5 ml | 石榴糖浆 |
| 1 颗 | 酒渍樱桃 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 将除装饰外的所有材料加入摇酒壶。
3. 加满冰块，用力摇和约 10-12 秒。
4. 双重过滤入预冷鸡尾酒杯。
5. 以酒渍樱桃装饰。

## 风味 / Flavor
菠萝、石榴、樱桃和甘蔗，甜美但有果酸。

## 情绪 / Mood
电影黄金时代、明亮、带一点明星光环。

## 调酒师笔记 / Bartender Notes
- 石榴糖浆和黑樱桃利口酒都很甜，只需少量。
- 菠萝汁充分摇和会带来漂亮泡沫。
