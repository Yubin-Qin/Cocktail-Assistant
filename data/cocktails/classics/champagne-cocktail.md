# Champagne Cocktail / 香槟鸡尾酒

## 故事 / Story
Champagne Cocktail 在 19 世纪已经广为人知，是最古老的起泡酒鸡尾酒之一，也展现了“糖、苦精、酒”的早期鸡尾酒原型。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 1 颗 | 方糖 |
| 2-3 dashes | 安哥斯图拉苦精 |
| 10 ml | 干邑 |
| 90-120 ml | 香槟 |
| 1 片 | 柠檬皮 |

## 步骤 / Steps
1. 将方糖放入冰镇香槟杯并滴上苦精。
2. 如使用干邑，加入杯中。
3. 缓慢补满冰镇香槟。
4. 以柠檬皮装饰。

## 风味 / Flavor
起泡酒、香料、柑橘和轻微焦糖感，简单但仪式感强。

## 情绪 / Mood
庆祝、复古、像老电影里的正式举杯。

## 调酒师笔记 / Bartender Notes
- 方糖吸满苦精后会缓慢释放味道和气泡视觉效果。
- 香槟必须很冷，倒酒要慢。
