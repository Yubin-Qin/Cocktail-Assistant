# Casino / 卡西诺

## 故事 / Story
Casino 是 20 世纪初期的琴酒经典，在古典配方书中已有记录，体现了烈酒、少量酸甜和苦精的老式平衡。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 40 ml | 老汤姆琴酒或伦敦干琴酒 |
| 10 ml | 黑樱桃利口酒 |
| 10 ml | 新鲜柠檬汁 |
| 2 dashes | 橙味苦精 |
| 1 颗 | 酒渍樱桃 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 将除装饰外的所有材料加入摇酒壶。
3. 加满冰块，用力摇和约 10-12 秒。
4. 双重过滤入预冷鸡尾酒杯。
5. 以酒渍樱桃装饰。

## 风味 / Flavor
杜松子、柠檬、樱桃核和橙皮香，短而集中。

## 情绪 / Mood
精致、旧派、略带纸牌桌般的紧张感。

## 调酒师笔记 / Bartender Notes
- 老汤姆琴酒会更圆润，伦敦干型则更利落。
- 黑樱桃利口酒只需少量，否则会压住琴酒。
