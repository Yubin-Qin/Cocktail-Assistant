# Cosmopolitan / 大都会

## 故事 / Story
现代 Cosmopolitan 在 1980-90 年代逐步定型，并因纽约酒吧文化和影视传播成为全球最知名的现代经典之一。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 40 ml | 柑橘伏特加 |
| 15 ml | 橙味利口酒 |
| 30 ml | 蔓越莓汁 |
| 15 ml | 新鲜青柠汁 |
| 1 片 | 橙皮 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 将除装饰外的所有材料加入摇酒壶。
3. 加满冰块，用力摇和约 10-12 秒。
4. 双重过滤入预冷鸡尾酒杯。
5. 以橙皮装饰。

## 风味 / Flavor
蔓越莓、青柠、橙皮和干净伏特加，酸甜明快。

## 情绪 / Mood
都市、时髦、外向，带有 1990 年代纽约夜生活的标志感。

## 调酒师笔记 / Bartender Notes
- 蔓越莓汁主要提供果酸和颜色，不应把酒做成甜果汁。
- 橙皮精油能明显提升香气层次。
