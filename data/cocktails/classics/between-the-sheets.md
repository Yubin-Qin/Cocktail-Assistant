# Between the Sheets / 床笫之间

## 故事 / Story
Between the Sheets 常被追溯到 20 世纪上半叶的欧洲酒吧文化，其结构与 Sidecar 同源，但加入朗姆后更复杂、更有力度。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 30 ml | 白朗姆 |
| 30 ml | 干邑 |
| 30 ml | 橙味利口酒 |
| 20 ml | 新鲜柠檬汁 |
| 1 片 | 柠檬皮 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 将除装饰外的所有材料加入摇酒壶。
3. 加满冰块，用力摇和约 10-12 秒。
4. 双重过滤入预冷鸡尾酒杯。
5. 以柠檬皮装饰。

## 风味 / Flavor
葡萄、甘蔗、橙皮和柠檬酸度并存，香气华丽，酒精感明显。

## 情绪 / Mood
放纵、夜色、带一点旧派暧昧感。

## 调酒师笔记 / Bartender Notes
- 可视作 Sidecar 加入白朗姆后的更强劲变体。
- 酸度要足够，否则三种酒的甜香会显得黏重。
