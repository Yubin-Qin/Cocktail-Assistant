# Saturn / 土星

## 故事 / Story
Saturn 由 J. “Popo” Galsini 在 1967 年创作并赢得调酒比赛，后来成为最著名的琴酒 Tiki 酒之一。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 琴酒 |
| 22.5 ml | 新鲜柠檬汁 |
| 15 ml | 百香果糖浆 |
| 7.5 ml | 杏仁糖浆 |
| 7.5 ml | 法勒南糖浆 |
| 1 份 | 柠檬皮 |

## 步骤 / Steps
1. 在摇酒壶中加入除装饰外的所有材料。
2. 加满冰块，用力摇和约 10-12 秒。
3. 过滤入装有新鲜冰块的古典杯。
4. 以柠檬皮装饰。

## 风味 / Flavor
百香果、杏仁、香料、柠檬和杜松子。

## 情绪 / Mood
复古未来、热带、像 1960 年代想象中的太空旅行。

## 调酒师笔记 / Bartender Notes
- 它是少见的琴酒 Tiki 经典，琴酒不能被热带糖浆完全遮住。
- 三种甜味材料都应精确量取。
