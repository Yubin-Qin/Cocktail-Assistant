# Lemon Drop Martini / 柠檬滴马天尼

## 故事 / Story
Lemon Drop 在 1970 年代美国西海岸兴起，名字来自同名柠檬糖果，后来成为全球酒吧常见的酸甜伏特加经典。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 伏特加 |
| 20 ml | 橙味利口酒 |
| 25 ml | 新鲜柠檬汁 |
| 10 ml | 单糖浆 |
| 1 份 | 糖边和柠檬皮 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 将除装饰外的所有材料加入摇酒壶。
3. 加满冰块，用力摇和约 10-12 秒。
4. 双重过滤入预冷鸡尾酒杯。
5. 以糖边和柠檬皮装饰。

## 风味 / Flavor
柠檬糖、橙皮、干净伏特加，酸甜直接。

## 情绪 / Mood
明快、聚会、略带 1970 年代糖果色。

## 调酒师笔记 / Bartender Notes
- 半圈糖边比整圈更灵活。
- 柠檬汁和糖浆应做成明亮酸甜，而不是糖水。
