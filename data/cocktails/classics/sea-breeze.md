# Sea Breeze / 海风

## 故事 / Story
Sea Breeze 的现代版本在 20 世纪后期伏特加长饮潮流中普及，与 Cape Codder、Bay Breeze 属于近亲家族。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 40 ml | 伏特加 |
| 90 ml | 蔓越莓汁 |
| 30 ml | 葡萄柚汁 |
| 1 份 | 青柠角 |

## 步骤 / Steps
1. 在高球杯中加入主要酒液。
2. 加满冰块。
3. 补入汽水或长饮部分。
4. 轻轻搅拌 2-3 圈。
5. 以青柠角装饰。

## 风味 / Flavor
蔓越莓、葡萄柚、轻苦和清爽果酸。

## 情绪 / Mood
海边、轻松、颜色明亮，适合无压力聚会。

## 调酒师笔记 / Bartender Notes
- 葡萄柚提供苦酸，让蔓越莓不过分甜。
- 用高品质果汁并充分冰镇。
