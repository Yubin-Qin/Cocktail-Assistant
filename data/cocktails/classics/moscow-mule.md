# Moscow Mule / 莫斯科骡子

## 故事 / Story
Moscow Mule 兴起于 20 世纪中期的美国，常被认为与伏特加品牌、姜汁啤酒和铜杯营销有关。它帮助伏特加在美国市场走红，也证明包装、器皿和饮用体验可以共同塑造一杯酒的命运。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 50 ml | 伏特加 |
| 15 ml | 新鲜青柠汁 |
| 120 ml | 姜汁啤酒 |
|  | 青柠角装饰 |

## 步骤 / Steps
1. 在铜杯或高球杯中加入伏特加和青柠汁。
2. 加满冰块。
3. 倒入姜汁啤酒，轻轻搅拌。
4. 用青柠角装饰。

## 风味 / Flavor
姜的辛辣、青柠酸、伏特加的干净酒体和气泡刺激。清爽中带一点热辣。

## 情绪 / Mood
轻松、社交、直接。它不需要复杂理解，第一口就把气氛打开。

## 调酒师笔记 / Bartender Notes
- 姜汁啤酒应有明显辛辣感，而不只是甜汽水。
- 铜杯能强化冰冷触感，但也可用普通高球杯。
- 不要剧烈搅拌，避免气泡流失。
