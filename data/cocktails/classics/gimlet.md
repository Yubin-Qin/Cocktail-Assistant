# Gimlet / 吉姆雷特

## 故事 / Story
Gimlet 常与英国海军和防坏血病的青柠补给故事相关。虽然实际历史比传说复杂，但琴酒和青柠 cordial 的组合确实来自保存柑橘与烈酒文化的交汇。后来它成为极简酸酒的代表。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 琴酒 |
| 20 ml | 新鲜青柠汁 |
| 15 ml | 单糖浆 |
| 5-10 ml 增加传统风格 | 可加入少量青柠 cordial |

## 步骤 / Steps
1. 摇酒壶中加入所有材料。
2. 加冰摇和 10 秒。
3. 过滤入预冷鸡尾酒杯。
4. 用青柠皮或青柠轮装饰。

## 风味 / Flavor
青柠清香、杜松子、轻甜和干爽酸度。比 Daiquiri 更草本，比 Martini 更活泼。

## 情绪 / Mood
干净、敏捷、带一点航海式的实用主义。它没有多余动作，像一句短而准确的回复。

## 调酒师笔记 / Bartender Notes
- 传统 Gimlet 与 Rose's Lime Cordial 关系密切，但现代做法常用鲜榨青柠加糖浆。
- 想要更古典，可用一部分 cordial 带来糖渍青柠风味。
- 琴酒不宜过度花香，否则会压过青柠的干净线条。
