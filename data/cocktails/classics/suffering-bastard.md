# Suffering Bastard / 受苦的混蛋

## 故事 / Story
Suffering Bastard 通常与二战时期开罗 Shepheard’s Hotel 的调酒师 Joe Scialom 联系在一起，据说为宿醉士兵而作。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 30 ml | 干邑 |
| 30 ml | 琴酒 |
| 15 ml | 新鲜青柠汁 |
| 2 dashes | 安哥斯图拉苦精 |
| 90 ml | 姜汁啤酒 |
| 1 份 | 薄荷枝 |

## 步骤 / Steps
1. 在高球杯中加入主要酒液。
2. 加满冰块。
3. 补入汽水或长饮部分。
4. 轻轻搅拌 2-3 圈。
5. 以薄荷枝装饰。

## 风味 / Flavor
姜辣、青柠、葡萄、杜松子和香料。

## 情绪 / Mood
幽默、恢复、像炎热天气里的一记冷醒。

## 调酒师笔记 / Bartender Notes
- 姜汁啤酒应辛辣，不要用过甜姜味汽水。
- 干邑和琴酒的双基酒结构需要足够青柠提亮。
