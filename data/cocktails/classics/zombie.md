# Zombie / 僵尸

## 故事 / Story
Zombie 由 Don the Beachcomber 在 1930 年代创作，是 Tiki 文化最具传奇性的配方之一，也以强烈酒精度闻名。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 30 ml | 陈年牙买加朗姆 |
| 30 ml | 波多黎各朗姆 |
| 30 ml | 高酒精度朗姆 |
| 20 ml | 新鲜青柠汁 |
| 15 ml | 葡萄柚汁 |
| 15 ml | 肉桂糖浆 |
| 10 ml | 法勒南糖浆 |
| 5 ml | 石榴糖浆 |
| 1 dash | 安哥斯图拉苦精 |
| 1 dash | 苦艾酒 |
|  | 碎冰 适量 |

## 步骤 / Steps
1. 将所有材料与少量碎冰快速搅打，或直接在杯中 swizzle。
2. 倒入特色杯并补满碎冰。
3. 让杯壁充分结霜。
4. 以薄荷和热带水果装饰。

## 风味 / Flavor
深色朗姆、青柠、葡萄柚、肉桂、丁香和热带果香。

## 情绪 / Mood
狂野、戏剧、像 Tiki 世界里的终极风暴。

## 调酒师笔记 / Bartender Notes
- 朗姆种类和香料很多，必须精确量取。
- 酒精度极高，经典酒吧常限制每位客人的杯数。
