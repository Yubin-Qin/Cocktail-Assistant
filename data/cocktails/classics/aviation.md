# Aviation / 航空

## 故事 / Story
Aviation 诞生于 20 世纪初，因紫罗兰利口酒带来的天空色调而得名；它曾因原料难寻而淡出，后来随着古典鸡尾酒复兴重新流行。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 琴酒 |
| 15 ml | 黑樱桃利口酒 |
| 15 ml | 新鲜柠檬汁 |
| 5 ml | 紫罗兰利口酒 |
| 1 颗 | 酒渍樱桃 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 将除装饰外的所有材料加入摇酒壶。
3. 加满冰块，用力摇和约 10-12 秒。
4. 双重过滤入预冷鸡尾酒杯。
5. 以酒渍樱桃装饰。

## 风味 / Flavor
花香、柠檬、樱桃核和杜松子，颜色可呈淡淡天空蓝紫。

## 情绪 / Mood
浪漫、轻盈、有一点旧时代旅行想象，适合夜晚开场。

## 调酒师笔记 / Bartender Notes
- 紫罗兰利口酒只需少量，过多会像香水并压住柠檬。
- 黑樱桃利口酒应提供核果感，而不是单纯甜味。
