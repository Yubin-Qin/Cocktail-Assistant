# Algonquin / 阿尔冈昆

## 故事 / Story
Algonquin 以纽约著名的 Algonquin Hotel 命名，与 20 世纪文学和剧场圈文化相连。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 黑麦威士忌 |
| 22.5 ml | 干味美思 |
| 22.5 ml | 菠萝汁 |
| 1 份 | 菠萝叶 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 将除装饰外的所有材料加入摇酒壶。
3. 加满冰块，用力摇和约 10-12 秒。
4. 双重过滤入预冷鸡尾酒杯。
5. 以菠萝叶，可选装饰。

## 风味 / Flavor
黑麦、菠萝、干草本和轻微热带甜香。

## 情绪 / Mood
文学、纽约、像老酒店酒吧里的编辑聚会。

## 调酒师笔记 / Bartender Notes
- 菠萝汁只占小比例，目的是果香和轻微质地。
- 使用辛辣黑麦能防止成品过于柔软。
