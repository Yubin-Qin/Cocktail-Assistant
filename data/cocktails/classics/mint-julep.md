# Mint Julep / 薄荷朱莉普

## 故事 / Story
Julep 家族历史悠久，Mint Julep 后来与美国南方和 Kentucky Derby 紧密联系，成为波本文化标志。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 波本威士忌 |
| 10 ml | 单糖浆 |
| 8-12 片 | 新鲜薄荷叶 |
|  | 碎冰 适量 |
| 1 大束 | 薄荷枝 |

## 步骤 / Steps
1. 在银杯或古典杯中轻压薄荷与糖浆。
2. 加入波本和大量碎冰。
3. 快速搅拌，直到杯壁结霜。
4. 补满碎冰。
5. 以薄荷枝装饰。

## 风味 / Flavor
薄荷、香草、焦糖、木桶和冰冷稀释。

## 情绪 / Mood
南方、夏日、从容，像树荫下慢慢过去的下午。

## 调酒师笔记 / Bartender Notes
- 薄荷只需轻压释放香气，避免捣碎出苦味。
- 杯外结霜和高高的碎冰是体验的一部分。
