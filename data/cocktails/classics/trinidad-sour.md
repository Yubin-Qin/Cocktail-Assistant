# Trinidad Sour / 特立尼达酸

## 故事 / Story
Trinidad Sour 由 Giuseppe González 在 2000 年代创作，以整整一份苦精作为主基酒，成为现代调酒史上最有影响力的反常规配方之一。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 安哥斯图拉苦精 |
| 15 ml | 黑麦威士忌 |
| 30 ml | 杏仁糖浆 |
| 22.5 ml | 新鲜柠檬汁 |
| 1 片 | 柠檬皮 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 将除装饰外的所有材料加入摇酒壶。
3. 加满冰块，用力摇和约 10-12 秒。
4. 双重过滤入预冷鸡尾酒杯。
5. 以柠檬皮装饰。

## 风味 / Flavor
丁香、肉桂、杏仁、柠檬和黑麦，浓烈但平衡。

## 情绪 / Mood
大胆、实验性、喝过一次就很难忘。

## 调酒师笔记 / Bartender Notes
- 苦精在这里是基酒，不是几滴调味，使用量非常反常规。
- 杏仁糖浆和柠檬必须足够支撑高浓度香料。
