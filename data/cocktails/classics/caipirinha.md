# Caipirinha / 凯匹林纳

## 故事 / Story
Caipirinha 是巴西最具代表性的鸡尾酒之一，以本土卡莎萨、青柠和糖构成极简结构。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 卡莎萨甘蔗酒 |
| 1 颗，切块 | 青柠 |
| 2 茶匙 | 细砂糖 |
|  | 碎冰 适量 |

## 步骤 / Steps
1. 将青柠块和糖放入古典杯。
2. 轻轻按压，释放果汁和外皮精油。
3. 加入基酒和碎冰。
4. 充分搅拌后再补碎冰。
5. 以青柠块装饰。

## 风味 / Flavor
青柠皮油、甘蔗、青草、酸甜和粗粝生命力。

## 情绪 / Mood
热烈、随性、带有巴西街头和海滩气息。

## 调酒师笔记 / Bartender Notes
- 只需压出青柠汁和皮油，不要把白色内皮捣烂。
- 卡莎萨的青草和甘蔗风味是核心，不建议用白朗姆完全替代。
