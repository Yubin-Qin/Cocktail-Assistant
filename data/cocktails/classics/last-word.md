# Last Word / 最后一言

## 故事 / Story
Last Word 源自 20 世纪早期底特律，沉寂多年后在 21 世纪初被西雅图调酒师重新推广，并引发大量等量结构变体。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 22.5 ml | 琴酒 |
| 22.5 ml | 绿夏翠丝 |
| 22.5 ml | 黑樱桃利口酒 |
| 22.5 ml | 新鲜青柠汁 |
| 1 颗 | 酒渍樱桃 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 将除装饰外的所有材料加入摇酒壶。
3. 加满冰块，用力摇和约 10-12 秒。
4. 双重过滤入预冷鸡尾酒杯。
5. 以酒渍樱桃装饰。

## 风味 / Flavor
草本、青柠、樱桃核、杜松子，复杂却极其紧凑。

## 情绪 / Mood
机智、神秘、像辩论结束时一句无法反驳的话。

## 调酒师笔记 / Bartender Notes
- 严格等量是它的魅力，任何一种材料过强都会失衡。
- 青柠汁必须新鲜，能切开两种利口酒的甜度。
