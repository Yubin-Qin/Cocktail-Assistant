# Hurricane / 飓风

## 故事 / Story
Hurricane 与新奥尔良 Pat O’Brien’s 酒吧密切相关，二战时期因朗姆供应充足而发展，标志性弧形杯也由此闻名。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 深色朗姆 |
| 45 ml | 白朗姆 |
| 30 ml | 百香果汁或糖浆 |
| 30 ml | 橙汁 |
| 20 ml | 新鲜青柠汁 |
| 10 ml | 石榴糖浆 |
| 1 份 | 橙片和樱桃 |

## 步骤 / Steps
1. 将除装饰外的材料加入摇酒壶。
2. 加冰摇和约 10-12 秒。
3. 过滤入装满新鲜冰块的高球杯或特色杯。
4. 以橙片和樱桃装饰。

## 风味 / Flavor
百香果、橙子、朗姆、青柠和红色热带果香。

## 情绪 / Mood
狂欢、新奥尔良、像热带风暴前的霓虹灯。

## 调酒师笔记 / Bartender Notes
- 朗姆必须有存在感，不要让果汁比例无限膨胀。
- 百香果的酸甜决定主体风格。
