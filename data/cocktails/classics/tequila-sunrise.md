# Tequila Sunrise / 龙舌兰日出

## 故事 / Story
现代 Tequila Sunrise 在 1970 年代美国加州走红，并因音乐文化和巡演故事成为时代符号。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 龙舌兰 blanco |
| 90 ml | 橙汁 |
| 15 ml | 石榴糖浆 |
| 1 份 | 橙片和樱桃 |

## 步骤 / Steps
1. 在高球杯中加入龙舌兰和橙汁。
2. 加满冰块并轻轻搅匀。
3. 沿杯壁缓慢倒入石榴糖浆，让其自然下沉。
4. 以橙片和樱桃装饰。

## 风味 / Flavor
橙子、龙舌兰草本、石榴甜味，视觉先于复杂度。

## 情绪 / Mood
公路、加州、1970 年代摇滚感。

## 调酒师笔记 / Bartender Notes
- 石榴糖浆最后沿杯壁缓慢倒入，不搅拌，形成日出渐层。
- 使用新鲜橙汁，颜色和酸度都会更自然。
