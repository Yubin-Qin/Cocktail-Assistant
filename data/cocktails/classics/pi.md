# Pi / ña Colada 椰林飘香

## 故事 / Story
Piña Colada 与 20 世纪中期波多黎各酒吧文化紧密相关，后来成为全球最知名的热带鸡尾酒之一。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 50 ml | 白朗姆 |
| 30 ml | 椰浆或椰子奶油 |
| 50 ml | 菠萝汁 |
| 1 杯 | 碎冰 |
| 1 份 | 菠萝片和樱桃 |

## 步骤 / Steps
1. 将除装饰外的材料和碎冰加入搅拌机。
2. 高速搅打至顺滑冰沙质地。
3. 倒入冰镇飓风杯或大杯。
4. 以菠萝片和樱桃装饰。

## 风味 / Flavor
菠萝、椰子、朗姆和奶油般热带质地。

## 情绪 / Mood
度假、逃离、毫不掩饰的海滩快乐。

## 调酒师笔记 / Bartender Notes
- 椰子奶油甜度很高，品牌不同需调整比例。
- 搅打到顺滑但仍有冰沙质感，不要化成水。
