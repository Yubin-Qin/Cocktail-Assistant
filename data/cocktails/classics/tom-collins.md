# Tom Collins / 汤姆柯林斯

## 故事 / Story
Tom Collins 在 19 世纪因同名恶作剧和酒吧文化走红，后来成为 Collins 家族的标准模板。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 老汤姆琴酒或伦敦干琴酒 |
| 30 ml | 新鲜柠檬汁 |
| 15 ml | 单糖浆 |
| 60-90 ml | 苏打水 |
| 1 份 | 柠檬片和樱桃 |

## 步骤 / Steps
1. 将基酒、柑橘汁和糖浆加冰短摇。
2. 过滤入装满冰块的高球杯。
3. 补入苏打水并轻轻搅拌。
4. 以柠檬片和樱桃装饰。

## 风味 / Flavor
柠檬、杜松子、轻甜和长气泡。

## 情绪 / Mood
阳光、经典、是一杯几乎不会出错的夏日长饮。

## 调酒师笔记 / Bartender Notes
- 老汤姆琴酒更符合早期风格，伦敦干型更清爽。
- 高球杯中需要足够冰块，避免快速变温。
