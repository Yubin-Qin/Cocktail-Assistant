# Bee's Knees / 蜜蜂膝盖

## 故事 / Story
Bee’s Knees 通常与美国禁酒时代联系在一起；“the bee’s knees”当时也有“非常棒”的俚语意味。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 琴酒 |
| 22.5 ml | 新鲜柠檬汁 |
| 22.5 ml | 蜂蜜糖浆 |
| 1 片 | 柠檬皮 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 将除装饰外的所有材料加入摇酒壶。
3. 加满冰块，用力摇和约 10-12 秒。
4. 双重过滤入预冷鸡尾酒杯。
5. 以柠檬皮装饰。

## 风味 / Flavor
柠檬明亮、蜂蜜圆润、杜松子和花香清晰。

## 情绪 / Mood
温暖、讨喜、带一点禁酒时代的俏皮，是很好的入门酸酒。

## 调酒师笔记 / Bartender Notes
- 蜂蜜糖浆可用蜂蜜与温水约 2:1 调匀，更容易溶解。
- 选择柑橘或花香明显的琴酒，会与蜂蜜更协调。
