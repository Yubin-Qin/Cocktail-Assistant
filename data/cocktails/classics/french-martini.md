# French Martini / 法式马天尼

## 故事 / Story
French Martini 在 1980-90 年代的现代鸡尾酒浪潮中走红，凭借覆盆子与菠萝的讨喜组合成为持久流行款。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 伏特加 |
| 15 ml | 覆盆子利口酒 |
| 45 ml | 菠萝汁 |
| 1 颗 | 覆盆子 |

## 步骤 / Steps
1. 将鸡尾酒杯预冷。
2. 将除装饰外的所有材料加入摇酒壶。
3. 加满冰块，用力摇和约 10-12 秒。
4. 双重过滤入预冷鸡尾酒杯。
5. 以覆盆子装饰。

## 风味 / Flavor
覆盆子、菠萝、热带果香和柔和甜酸。

## 情绪 / Mood
甜美、社交、带一点 1990 年代酒吧的闪亮感。

## 调酒师笔记 / Bartender Notes
- 菠萝汁需要有力摇和才能形成漂亮泡沫。
- 它并非传统 Martini，只是借用杯型和都市命名。
