# Oaxaca Old Fashioned / 瓦哈卡古典

## 故事 / Story
Oaxaca Old Fashioned 由 Phil Ward 在 2000 年代纽约创作，对梅斯卡尔在现代鸡尾酒中的普及影响很大。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 龙舌兰 reposado |
| 15 ml | 梅斯卡尔 |
| 5 ml | 龙舌兰糖浆 |
| 2 dashes | 安哥斯图拉苦精 |
| 1 片 | 橙皮 |

## 步骤 / Steps
1. 将除装饰外的材料加入调酒杯并加满冰块。
2. 搅拌约 25-35 秒，直到充分冰镇。
3. 过滤入装有大冰块的古典杯。
4. 以橙皮装饰。

## 风味 / Flavor
烤龙舌兰、烟熏、橙皮、香料和焦糖。

## 情绪 / Mood
现代、深色、像 Old Fashioned 走进墨西哥高原。

## 调酒师笔记 / Bartender Notes
- 梅斯卡尔只需少量烟熏点缀。
- 龙舌兰糖浆甜度高，5 ml 往往已足够。
