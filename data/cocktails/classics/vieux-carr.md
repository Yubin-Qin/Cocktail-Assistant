# Vieux Carr / é 老广场

## 故事 / Story
Vieux Carré 由新奥尔良 Hotel Monteleone 的 Walter Bergeron 在 1930 年代创作，名字来自法语“老广场”，即法国区。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 30 ml | 黑麦威士忌 |
| 30 ml | 干邑 |
| 30 ml | 甜味美思 |
| 1 茶匙 | Bénédictine 草本利口酒 |
| 2 dashes | Peychaud's 苦精 |
| 2 dashes | 安哥斯图拉苦精 |
| 1 片 | 柠檬皮 |

## 步骤 / Steps
1. 将除装饰外的材料加入调酒杯并加满冰块。
2. 搅拌约 25-35 秒，直到充分冰镇。
3. 过滤入装有大冰块的古典杯。
4. 以柠檬皮装饰。

## 风味 / Flavor
黑麦、葡萄、红果、草本、香料和木桶，层次极深。

## 情绪 / Mood
新奥尔良、爵士、像老城区夜色一样复杂。

## 调酒师笔记 / Bartender Notes
- 双基酒和多种草本材料需要充分搅拌稀释。
- Bénédictine 只需少量即可提供深度。
