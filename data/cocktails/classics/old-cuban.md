# Old Cuban / 老古巴

## 故事 / Story
Old Cuban 由纽约调酒师 Audrey Saunders 在 2000 年代初创作，很快成为现代经典。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 陈年朗姆 |
| 22.5 ml | 新鲜青柠汁 |
| 15 ml | 单糖浆 |
| 2 dashes | 安哥斯图拉苦精 |
| 6-8 片 | 薄荷叶 |
| 60 ml | 香槟或干型起泡酒 |
| 1 份 | 薄荷叶 |

## 步骤 / Steps
1. 将除起泡酒外的材料加冰摇和。
2. 双重过滤入预冷鸡尾酒杯。
3. 缓慢补入冰镇香槟或干型起泡酒。
4. 以薄荷叶装饰。

## 风味 / Flavor
陈年朗姆、薄荷、青柠、香料和细腻气泡。

## 情绪 / Mood
成熟、庆祝、像 Mojito 穿上了晚礼服。

## 调酒师笔记 / Bartender Notes
- 薄荷只轻拍或轻压，避免叶绿苦味。
- 先摇好朗姆酸酒基底，再补冰冷起泡酒。
