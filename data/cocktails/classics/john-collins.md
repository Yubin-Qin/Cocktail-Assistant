# John Collins / 约翰柯林斯

## 故事 / Story
Collins 家族源自 19 世纪英国的长饮传统，John Collins 与 Tom Collins 的名称和基酒历史上多有交叉。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 琴酒 |
| 30 ml | 新鲜柠檬汁 |
| 15 ml | 单糖浆 |
| 60-90 ml | 苏打水 |
| 1 份 | 柠檬片和樱桃 |

## 步骤 / Steps
1. 将基酒、柑橘汁和糖浆加冰短摇。
2. 过滤入装满冰块的高球杯。
3. 补入苏打水并轻轻搅拌。
4. 以柠檬片和樱桃装饰。

## 风味 / Flavor
柠檬、杜松子、轻甜和清爽气泡。

## 情绪 / Mood
明快、传统、适合从下午一直喝到傍晚。

## 调酒师笔记 / Bartender Notes
- 先把酸酒基底混匀，再补苏打水。
- 若用老汤姆琴酒，常被称为 Tom Collins。
