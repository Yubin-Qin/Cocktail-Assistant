# Pisco Sour / 皮斯科酸

## 故事 / Story
Pisco Sour 在秘鲁和智利都有深厚文化关联，现代配方通常与 20 世纪初利马酒吧文化联系在一起。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 60 ml | 皮斯科 |
| 30 ml | 新鲜青柠或柠檬汁 |
| 20 ml | 单糖浆 |
| 20 ml | 蛋清 |
| 3 dashes | 安哥斯图拉苦精 |

## 步骤 / Steps
1. 将所有材料不加冰干摇约 10 秒。
2. 加入冰块，再用力摇和 10-12 秒。
3. 过滤入装有新鲜冰块的古典杯。
4. 将苦精滴在泡沫表面。

## 风味 / Flavor
葡萄蒸馏酒、柑橘酸、柔滑泡沫和香料苦精。

## 情绪 / Mood
明亮、优雅、带安第斯地区的鲜明个性。

## 调酒师笔记 / Bartender Notes
- 先干摇再加冰摇，泡沫更细密。
- 皮斯科类型会影响花香和果香，可按喜好选择。
