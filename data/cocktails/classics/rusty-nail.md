# Rusty Nail / 锈钉

## 故事 / Story
Rusty Nail 在 20 世纪中期广为流行，凭苏格兰威士忌与 Drambuie 的简单组合成为经典双材料鸡尾酒。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | 苏格兰威士忌 |
| 25 ml | 蜂蜜草本威士忌利口酒 |
| 1 块 | 大冰块 |
| 1 片 | 柠檬皮 |

## 步骤 / Steps
1. 在古典杯中加入所有酒液。
2. 加入一块大冰块。
3. 轻轻搅拌约 15-20 秒。
4. 以柠檬皮装饰。

## 风味 / Flavor
蜂蜜、麦芽、香料、草本和温暖木桶。

## 情绪 / Mood
旧派、沉稳、像深夜酒吧最后一杯。

## 调酒师笔记 / Bartender Notes
- 调和威士忌更圆润，烟熏型单一麦芽会更强烈。
- 利口酒甜度高，可按口味降低到 15-20 ml。
