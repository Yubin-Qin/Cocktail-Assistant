# Cuba Libre / 自由古巴

## 故事 / Story
Cuba Libre 与 20 世纪初古巴和美国文化交流有关，名称意为“自由古巴”，后来成为世界范围内最普及的长饮之一。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 50 ml | 白朗姆或金朗姆 |
| 10 ml | 新鲜青柠汁 |
| 120 ml | 可乐 |
| 1 份 | 青柠角 |

## 步骤 / Steps
1. 在高球杯中加入主要酒液。
2. 加满冰块。
3. 补入汽水或长饮部分。
4. 轻轻搅拌 2-3 圈。
5. 以青柠角装饰。

## 风味 / Flavor
焦糖、香草、青柠和甘蔗，气泡活泼。

## 情绪 / Mood
轻松、直接、适合聚会和炎热天气。

## 调酒师笔记 / Bartender Notes
- 青柠汁不是可有可无，它把酒从“朗姆可乐”变成完整鸡尾酒。
- 可乐和杯具要很冷，避免快速融冰。
