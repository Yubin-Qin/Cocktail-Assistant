# Margarita / 玛格丽特

## 故事 / Story
Margarita 的具体发明者有多种说法，但大多与 20 世纪上半叶的墨西哥和美国边境酒吧有关。它可以看作 Daisy 类鸡尾酒的龙舌兰版本：烈酒、柑橘、甜味元素构成核心，“Margarita”在西班牙语中也有雏菊之意。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 50 ml | 龙舌兰 blanco |
| 25 ml | 橙味利口酒 |
| 25 ml | 新鲜青柠汁 |
|  | 盐边 |
| 1 份 | 青柠角 |

## 步骤 / Steps
1. 用青柠擦杯口外侧，轻沾盐形成半圈盐边。
2. 摇酒壶中加入龙舌兰、橙味利口酒和青柠汁。
3. 加冰用力摇和 10-12 秒。
4. 过滤入预冷鸡尾酒杯，或倒入加冰古典杯。
5. 用青柠角装饰。

## 风味 / Flavor
青柠酸度、龙舌兰的草本和矿物感、橙皮甜香与盐的咸鲜。口感锋利但有热情。

## 情绪 / Mood
明快、外向、带一点派对感。它让人想到热空气、盐、笑声和不想太早结束的傍晚。

## 调酒师笔记 / Bartender Notes
- 盐边做半圈更好，饮客可选择有盐或无盐入口。
- 使用 100% agave 龙舌兰，风味会更干净。
- 橙味利口酒影响甜度；若偏甜，可减少利口酒或增加少量青柠。
