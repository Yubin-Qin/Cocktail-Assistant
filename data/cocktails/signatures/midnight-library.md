# Midnight Library / 午夜图书馆

## 故事 / Story
For the kind of night spent alone with a book and one good drink. Rye whisky and a coffee liqueur build the body; Fernet-Branca's menthol-dark amaro and a whisper of Allspice dram add the dusty, leathery depth of old bindings. A few drops of saline lift the cocoa and spice. It is bitter, brooding, and quietly caffeinated — the way the best small hours feel.

献给那种独自一人、一本书、一杯好酒的夜晚。黑麦威士忌与咖啡利口酒搭起骨架；Fernet-Branca 的薄荷黑草本与一丝 Allspice dram 叠出旧书皮般的沉郁深度；几滴盐水把可可与香料托起来。它苦、深沉、悄悄带一点咖啡因——正像最好的凌晨时分该有的样子。

**设计思路**：以「黑草本 + 咖啡 + 木质」复刻「翻动旧书页」的嗅觉记忆，苦味是主旋律，咸与甜只是配角。

## 配方 / Ingredients
| 用量 Amount | 材料 Ingredient |
| --- | --- |
| 45 ml | Rye whisky 黑麦威士忌 |
| 15 ml | Coffee liqueur 咖啡利口酒 |
| 8 ml | Fernet-Branca 费内特苦酒 |
| 5 ml | Allspice dram 多香果利口酒 |
| 2 dash | Orange bitters 橙味苦精 |
| 3 滴 | Saline solution (20% 盐水) 盐水 |

## 步骤 / Steps
1. 在调酒杯中加入大块冰。
2. 依次倒入黑麦、咖啡利口酒、Fernet-Branca、Allspice dram、橙味苦精与盐水。
3. 搅拌约 25–30 秒，至充分冰镇、稀释适中。
4. 过滤倒入冰镇过的浅碟杯。
5. 挤压柠檬皮精油于酒面，放上皮，再刨几片黑巧克力屑于表面。

## 调酒师笔记 / Bartender Notes
- Fernet-Branca 用量要克制——它的薄荷草本极具侵略性，多了会让整杯只剩牙膏感。
- 盐水能显著提亮咖啡与可可香气，是「让苦味变贵」的小技巧。
- 搅拌偏久一点，稀释稍多，口感更柔顺——这是一杯慢饮，不追求烈。
- 黑麦可换波本，会更甜润；但黑麦的辛辣与 Fernet 更契合「书卷气」。

## 变体 / Variations
- **Silent Spring**：加 5 ml 接骨木花利口酒，苦中带春。
- **Library Sour**：摇和版，加 20 ml 鲜柠檬汁与蛋清，绵密但更轻盈。
